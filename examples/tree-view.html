Tree View

A tree view has parent and child nodes that can expand and collapse.

See Expand/Collapse Menu for a non-tree alternative.

Keyboard: Only one active node can receive Tab focus. Tabbing again moves the user past the tree view.

  • Right arrow: expands closed node, or goes to first child of open node
  • Left arrow: closes open node, or goes to closed node's parent
  • Up arrow: goes to previous visible node
  • Down arrow: goes to next visible node
  • Home: goes to root node
  • End: goes to last visible node

Screen Reader:

  • Announces tree role
  • Announces name of active node and expanded/collapsed state
  • Announces level of active item in the tree (top level may be level 0 or level 1)
  • Announces number of items in the list at that level
  • Announces expanded/collapsed state changes
