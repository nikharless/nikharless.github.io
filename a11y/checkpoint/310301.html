Overview

Unusual words may include:

  • Idioms
  • Jargon
  • Rare, archaic, or obsolete word usage
  • Words with multiple definitions where only one applies in context
  • New author-created definitions
  • Unfamiliar words from other languages

Basic Requirements and Best Practices

= Requirement
= Best Practice

  • Definitions must be provided for unusual words.
User Story: As a user with a cognitive disability, I want words used in unusual ways to be defined so that I can understand the content.
Success: If you were a user with a cognitive disability, when you encountered words used in unusual ways, you would have definitions provided to help you understand the content.

Testing Logic

Does the text contain any words or phrases used in an unusual or restricted way, such as idioms and jargon?

  • Yes; Are definitions provided for these words or phrases?
    • Yes; PASS
    • No; See 3.1.3.1.1 Definitions are not provided for unusual words.
  • No; PASS

Web Test Procedure

  1. Read the text of the page to identify any words or phrases used in an unusual or restricted way.
  2. Verify that definitions are provided.

Failures

  • 3.1.3.1.1 Definitions are not provided for unusual words. (Medium)
