Overview

Basic Requirements and Best Practices

= Requirement
= Best Practice

  • The default page language must be set.
User Story: As a screen reader user, I want the default page language set so that I my screen reader knows how to pronounce the page content.
Success: If you were a screen reader user, when you read the page, the screen reader would pronounce the content correctly.

Testing Logic

Is the default page language set using the lang attribute on the <html> tag?

  • Yes; Is the value of the lang attribute valid?
    • Yes; Is the value of the lang attribute set to the correct language?
      • Yes; PASS
      • No; See {{ site.data.3101.3101[31010103].name }}.
    • No; See {{ site.data.3101.3101[31010102].name }}.
  • No; See {{ site.data.3101.3101[31010101].name }}.

Web Test Procedure

Desktop

axe DevTools

  1. Run axe DevTools to flag an issue if the lang attribute or its value is invalid.
  2. Verify that the lang attribute value matches the actual page language.

ANDI

  1. Run ANDI.
  2. Select structures.
  3. Select page language from the more details menu.
  4. Verify that the <html> element has a lang attribute.
  5. Verify that the lang attribute value is valid and matches the actual page language.

Manual

  1. Look at the source code of the page and verify that the <html> element has a lang attribute set to a language code.
  2. Verify that the lang attribute value matches the actual page language.

Mobile

  1. Open the page on a desktop computer in Chrome.
  2. Open the Developer tools:
    • Right-click the control and select Inspect, OR
    • From the Customize and control Google Chrome menu, select More tools > Developer tools, OR
    • Press Ctrl+Shift+I
  3. Activate Toggle device toolbar (Ctrl+Shift+M) to view the page as a mobile device.
  4. Select a device from the Dimensions drop-down.
  5. Follow the procedure for Desktop Web.

PDF Test Procedure

Adobe Acrobat Pro

  1. Go to Document Properties:
    • Select Menu > Document properties....
    • Windows: Control+D
    • MacOS: Command+D
  2. In the Document Properties dialog, select the Advanced tab.
  3. Under Reading Options, verify that a language is set in the Language drop-down.
  4. Verify that the language matches the actual language of the document.
