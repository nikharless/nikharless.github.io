Overview

It's not necessary that the reading order matches the visual order unless it affects the meaning. There may be more than one logical reading order, and only one needs to be provided.

Examples:

  • Passages of text, tables, and ordered lists have meaningful sequences that cannot be rearranged.
  • Items in unordered lists do not have a meaningful sequence.
  • Sidebars and main content areas usually do not have to be in a particular order.
  • Headings and associated content have a meaningful sequence.

Basic Requirements and Best Practices

= Requirement
= Best Practice

  • The reading order of page content must be logical and intuitive.
  • All meaningful content must be readable by assistive technology.
  • Content meant to be hidden from all users must not be readable by assistive technology.
  • Reading order should match the visual order, but it is not a failure unless the difference changes meaning.
User Story: As a blind screen reader user, I want the content to be read in a meaningful sequence so that I can understand the content.
Success: If you could not see the web page, when you use a screen reader, you would hear the content in an order that makes logical sense.

Web Test Procedure

Desktop Screen Reader

  1. Starting at the top of the page, read through the page content with the screen reader.
    • NVDA and JAWS: Use Down Arrow and Up Arrow
    • VoiceOver: Use VO+Down Arrow and Up Arrow
  2. Verify that the reading order of the static content follows the intended meaning of the content.
  3. Verify that all visible text can be read.
  4. Verify that content that is meant to be hidden from all users is hidden from the screen reader.

Mobile Screen Reader

VoiceOver or TalkBack

  1. Starting at the top of the page, read through the page content with the screen reader by swiping left to right.
  2. Verify that the reading order of the static content follows the intended meaning of the content.
  3. Verify that all visible text can be read.
  4. Verify that content that is meant to be hidden from all users is hidden from the screen reader.
