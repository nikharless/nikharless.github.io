Basic Requirements and Best Practices

= Requirement
= Best Practice

  • Instructions provided for understanding and operating content do not rely solely on visual characteristics of components such as shape, color, size, visual location, or orientation.
User Story: As a blind screen reader user, I want instructions to not require vision so that I can follow the them.
Success: If you could not see the web page, when you use a screen reader to read instructions, you would still understand how to follow them.

Testing Logic

Do any instructions on the page refer to visual cues such as shape, color, size, visual location, or orientation?

  • Yes; Do the instructions also include references which do not rely on visual cues?
    • Yes; PASS
    • No; See F1.3.3.a.a Instructions require the ability to see.
  • No; PASS

Web Test Procedure

Desktop

  1. Locate any instructions related to understanding or operating content.
  2. Determine if any of the instructions reference visual characteristics such as shape, color, size, visual location, or orientation.
  3. If so, verify that they also reference another non-visual characteristic, such as a text label, that can be accessed using assistive technology and by people who cannot perceive color.

Failures

  • F1.3.3.a.a Instructions require the ability to see. (Medium)

Testing Tools

  • Screen reader
