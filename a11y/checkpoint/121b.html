Overview

Video-only content includes the following:

  • Silent movie
  • Video with only music

Basic Requirements and Best Practices

= Requirement
= Best Practice

  • Prerecorded video-only content must have a text or audio alternative that includes all of the information presented in the video.
  • The text or audio description should be presented next to the video and be easy to find.
User Story: As a blind user, I want to listen to an audio description along with the video or read a transcript with my screen reader so that I can get all of the same information that is conveyed visually.
Success: If you could not see the video, when you listen to the audio description or read the transcript, you would get all of the information conveyed in the video content.

Testing Logic

Does the page contain any video-only content?

  • Yes; Is the video purely decorative?
    • Yes; PASS
    • No; Does the video-only content have an audio description?
      • Yes; Does the audio description include all the important actions, visuals, and text in the video?
        • Yes; PASS
        • No; See F1.2.1.b.b Existing description or text transcript is inadequate.
      • No; Does the video-only content have an accompanying text transcript?
        • Yes; Does the transcript include all the important actions, visuals, and text in the video?
          • Yes; PASS
          • No; See F1.2.1.b.b Existing description or text transcript is inadequate.
        • No; Is the video-only content itself a media alternative for text?
          • Yes; Is it clearly labeled as such?
            • Yes; PASS
            • No; Add text making it clear that the video is an alternative for the text content.
          • No; See F1.2.1.b.a Audio description or text transcript is missing.
  • No; PASS

Web Test Procedure

  1. Locate each video-only object.
  2. Verify that there is either an audio description or a text transcript that provides all the information presented visually.

Mobile App Test Procedure

  • Follow the procedure for Web.

Failures

  • F1.2.1.b.a Audio description or text transcript is missing. (Medium)
  • F1.2.1.b.b Existing description or text transcript is inadequate. (Medium)
