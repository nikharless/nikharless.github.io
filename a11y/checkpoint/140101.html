Overview

Some ways to convey information without relying on color:

  • Text
  • A symbol
  • Font weight
  • Pattern
  • Contrast

The alternate visual means are in addition to accessible programmatic alternatives for assistive technology.

Basic Requirements and Best Practices

= Requirement
= Best Practice

  • Color must not be used as the only visual means of conveying information.
User Story: As a color blind user, I want to have an alternative non-color-related means to get information so that I get the same information that non-color blind users get.
Success: If you could not see colors, when you view the page, you would get all the same information as non-color blind users.

Testing Logic

Is color used as the only means to convey information?

  • Yes; See 1.4.1.1.1 Color is used as the only means of conveying information.
  • No; Is color used as the only means to indicate an action?
    • Yes; See 1.4.1.1.2 Color is used as the only means of indicating an action.
    • No; Is color used as the only means to prompt a response?
      • Yes; See 1.4.1.1.3 Color is used as the only means of prompting a response.
      • No; Is color used as the only means to distinguish a visual element?
        • Yes; See 1.4.1.1.4 Color is used as the only means of distinguishing a visual element.
        • No; PASS

Web Test Procedure

  1. Examine the page for any elements where color is used to convey information, indicate an action, prompt a response, or distinguish a visual element.
  2. Verify that the information conveyed by color is also conveyed visually without relying on color.

Mobile App Test Procedure

  1. Follow the procedure for Web.

PDF Test Procedure

  1. Follow the procedure for Web.

Failures

  • 1.4.1.1.1 Color is used as the only means of conveying information. (Medium)
  • 1.4.1.1.2 Color is used as the only means of indicating an action. (Medium)
  • 1.4.1.1.3 Color is used as the only means of prompting a response. (Medium)
  • 1.4.1.1.4 Color is used as the only means of distinguishing a visual element. (Medium)
