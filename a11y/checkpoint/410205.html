Basic Requirements and Best Practices

= Requirement
= Best Practice

  • The role of custom user interface components must be programmatically determinable by assistive technologies.
  • HTML markup must meet W3C specifications and facilitate accessibility support with assistive technologies.
  • ARIA should be used to indicate role for tabs, modals, or other dynamic widgets.
User Story: As a screen reader user, I want the role of user interface components to be programmatically determinable so that I can get this information via my screen reader.
Success: If you were a screen reader user, when accessing user interface components, you would be able to determine the role of the components.
