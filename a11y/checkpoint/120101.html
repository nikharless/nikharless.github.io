Overview

Audio-only content includes the following:

  • Audio podcasts
  • mp3 files

Example

The text transcript for this audio podcast can be read while the audio is playing.

This American Life podcast episode transcript

Basic Requirements and Best Practices

= Requirement
= Best Practice

  • Prerecorded audio-only content must have a text transcript that includes all of the information presented in the audio.
  • The text transcript should be presented next to the audio and be easy to find.
User Story: As a deaf user, I want to read a transcript of the audio content so that I can get all of the same information.
Success: If you could not hear the audio, when you read the transcript, you would get all of the information conveyed in the audio content.

Testing Logic

Does the page contain any audio-only content?

  • Yes; Is the audio-only content accompanied by a full transcript?
    • Yes; Does the dialogue in the transcript match the dialogue in the audio-only content?
      • Yes; If the audio includes more than one speaker, does the transcript identify who is speaking each time it changes?
        • Yes; Does the transcript include important sound effects?
          • Yes; PASS
          • No; See 1.2.1.1.4 Existing transcript does not include important sound effects.
        • No; See 1.2.1.1.3 Existing text transcript does not identify speakers.
      • No; See 1.2.1.1.2 Existing text transcript does not include all dialogue.
    • No; Is the audio-only content itself a media alternative for text?
      • Yes; Is it clearly labeled as such?
        • Yes; PASS
        • No; Add text making it clear that the audio is an alternative for the text content.
      • No; See 1.2.1.1.1 Text transcript is missing.
  • No; PASS

Web Test Procedure

  1. Locate each audio-only object.
  2. Verify that there is a text transcript available.
  3. Verify that the transcript includes all dialogue and sound effects, and that it identifies the speakers.

Mobile App Test Procedure

  • Follow the procedure for Web.

Failures

  • 1.2.1.1.1 Text transcript is missing. (High)
  • 1.2.1.1.2 Existing text transcript does not include all dialogue. (Medium)
  • 1.2.1.1.3 Existing text transcript does not identify speakers. (Low)
  • 1.2.1.1.4 Existing transcript does not include important sound effects. (Medium)
  • 1.2.1.1.5 Existing transcript is inadequate. (Medium)
