Overview

Audio-only content includes the following:

  • Audio podcasts
  • mp3 files

Example

The text transcript for this audio podcast can be read while the audio is playing.

This American Life podcast episode transcript

Basic Requirements and Best Practices

= Requirement
= Best Practice

  • Prerecorded audio-only content must have a text transcript.
  • Text transcript must include all dialogue and narration.
  • Text transcript must identify speakers.
  • Text transcript must include important sounds.
  • Text transcript must be an actual text alternative.
  • The text transcript should be presented next to the audio and be easy to find.
User Story: As a deaf user, I want to read a transcript of the audio content so that I can get all of the same information.
Success: If you could not hear the audio, when you read the transcript, you would get all of the information conveyed in the audio content.

Web Test Procedure

  1. Locate each audio-only object.
  2. Verify that there is a text transcript available.
  3. Verify that the transcript meets all the requirements listed above.
