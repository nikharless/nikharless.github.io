Overview

Basic Requirements and Best Practices

= Requirement
= Best Practice

  • Line spacing must be adjustable to at least 1.5.
  • Paragraph spacing must be adjustable to at least 1.5 times the line spacing.
User Story: As a user with a visual or cognitive disability, I want text to be spaced so that I can more easily read the content.
Success: If you had difficulty reading tightly spaced text, when you encountered blocks of text with line spacing of less than 1.5, you would be able to adjust the line spacing.

Testing Logic

Does the page have blocks of text with line spacing less than 1.5?

  • Yes; Is there a mechanism on the page to adjust the line spacing?
    • Yes; Is the paragraph spacing at least 1.5 times the line spacing?
      • Yes; PASS
      • No; Is there a mechanism on the page to adjust the paragraph spacing?
        • Yes; PASS
        • No; See {{ site.data.1408.1408[14080402].name }}
    • No; See {{ site.data.1408.1408[14080401].name }}
  • No; PASS

Web Test Procedure

Desktop

  1. Check if there are blocks of text with line spacing of less than 1.5.
  2. If so, verify that there is a mechanism available to adjust the line spacing to at least 1.5.
  3. Check if there are blocks of text with paragraph spacing of less than 1.5 times the line spacing.
  4. If so, verify that there is a mechanism available to adjust the paragraph spacing to at least 1.5 times the line spacing.
