Overview

Labels and instructions should include enough information for the user to know what is expected, including data format.

Basic Requirements and Best Practices

= Requirement
= Best Practice

  • Form or form field must have visual essential instructions.
  • Important instructions should be accessible to assistive technology. (For the requirement, see {{ site.data.1301.1301[1301].name }}.)
User Story: As a user with a cognitive disability, I want form inputs and instructions to have enough information so that I know what is expected.
Success: If you were a user with a cognitive disability, when you encountered form inputs, labels and instructions would provide enough information for me to know what is expected.

Testing Logic

Are there form inputs on the page?

  • Yes; Does the form or form field require any special instructions (e.g., format instructions, order of completion, special conditions, etc.)?
    • Yes; Are the instructions provided visually, either in the field labels or elsewhere in the form?
      • Yes; PASS
      • No; See {{ site.data.3302.3302[33020201].name }}.
    • No; This checkpoint is not applicable.
  • No; This checkpoint is not applicable.

Web Test Procedure

  1. Locate any forms.
  2. Determine whether additional instruction is needed to know how to enter data.
  3. If so, verify that enough instruction is provided either in the labels or separate instructions.
