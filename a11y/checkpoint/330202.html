Overview

Labels and instructions should include enough information for the user to know what is expected, including data format.

Basic Requirements and Best Practices

= Requirement
= Best Practice

  • Expected data format must be specified if it isn't clear.
  • Important instructions must be accessible to assistive technology.
User Story: As a user with a cognitive disability, I want form inputs and instructions to have enough information so that I know what is expected.
Success: If you were a user with a cognitive disability, when you encountered form inputs, labels and instructions would provide enough information for me to know what is expected.

Testing Logic

Are there form inputs on the page?

  • Yes; Does each form input have a label that accurately describes its purpose?
    • Yes; Does the label include enough information to know what data is expected?
      • Yes; PASS
      • No; Are there separate instructions which include enough information to know what data is expected?
        • Yes; Are the instructions accurate?
          • Yes; Are the instructions communicated to assistive technology?
            • Yes; PASS
            • No; See F3.3.2.b.d Form input instructions are not communicated to assistive technology.
          • No; See F3.3.2.b.c Form input instructions are inaccurate.
        • No; See F3.3.2.b.b Label does not indicate the form input's expected data format.
    • No; See F3.3.2.b.a Label does not accurately describe form input purpose.
  • No; Not applicable

Web Test Procedure

  1. Locate any form inputs in the content.
  2. Verify that the label for each input accurately describes its purpose.
  3. Determine whether additional instruction is needed to know how to enter data.
  4. If so, verify that enough instruction is provided.
  5. Turn on a screen reader and verify that the instructions are communicated to assistive technology.

Mobile App Test Procedure

Follow the procedure for Web.

PDF Test Procedure

Follow the procedure for Web.

Failures

  • F3.3.2.b.a Label does not accurately describe form input purpose.
  • F3.3.2.b.b Label does not indicate the form input's expected data format.
  • F3.3.2.b.c Form input instructions are inaccurate.
  • F3.3.2.b.d Form input instructions are not communicated to assistive technology.
