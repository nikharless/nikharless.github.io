Overview

Basic Requirements and Best Practices

= Requirement
= Best Practice

  • Text must not be justified.
User Story: As a user with a cognitive disability, I want text to be evenly spaced so that I can more easily read the content.
Success: If you had difficulty reading unevenly spaced text, when you encountered blocks of text, you would be able to easily read it.

Testing Logic

Does the page have blocks of text that are justified?

  • Yes; Is there a mechanism on the page to remove the justification?
    • Yes; PASS
    • No; See F1.4.8.c.a Text is justified.
  • No; PASS

Web Test Procedure

Desktop

  1. Check if there are blocks of text which are aligned on both the left and right sides.
  2. If so, verify that there is a mechanism available to remove the justification.

Failures

  • F1.4.8.c.a Text is justified. (Medium)
