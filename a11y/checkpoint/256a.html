Overview

Basic Requirements and Best Practices

= Requirement
= Best Practice

  • Users must be able to switch between different modes of input.
User Story: As a user with a disability, I want to be able to switch between methods of input so that I can interact with the content in ways that are easiest for me.
Success: If you were a user with a disability, when you switched methods of input, you could interact with the content.

Testing Logic

Is the user restricted to certain methods of input?

  • Yes; Is the restriction essential to the content or required to ensure the security of the content?
    • Yes; PASS
    • No; Is the restriction required to respect user settings?
      • Yes; PASS
      • No; See
  • No; PASS

Web Test Procedure

Desktop

  1. Verify that you can interact with all web content using various methods of input, such as:
    • Keyboard
    • Mouse
    • Switch
    • Voice control
    • Touchscreen

Mobile

  1. Verify that you can interact with all web content using various methods of input, such as:
    • Touchscreen
    • Bluetooth keyboard and mouse
    • Switch
    • Voice control

Failures
