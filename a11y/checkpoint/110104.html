Overview

Decorative Example

The image of 3-D shapes is purely decorative and must be hidden from assistive technology.

Redundant Example

Alan Bates and Joanna Pettet on Broadway in "Poor Richard"

This image has adjacent text describing its content. Repeating this in the alt text would be redundant, so alt="" is appropriate here.

Basic Requirements and Best Practices

= Requirement
= Best Practice

  • Decorative or redundant images must not have text alternatives.
  • Decorative or redundant images should be hidden from screen readers.
  • Text alternatives for linked images should not duplicate adjacent link text for the same destination. Combine the image and text into a single link.
  • Text alternatives for images inside of links or buttons should not duplicate the text content of the link or button. The alt text should be null.
User Story: As a blind screen reader user, I don't want to hear a description of the decorative image so that I am not overwhelmed with irrelevant screen reader chatter.
Success: If you could not see the image, when you read it with a screen reader, you would not know it existed.

Some ways to code images to be ignored by assistive technologies include:

  • Adding alt="" to an <img> element
  • Using role="none", role="presentation", or aria-hidden="true" on the image element
  • Using aria-hidden="true" on an <svg> element
  • Using CSS background images

Web Test Procedure

Desktop

  1. Run axe DevTools or WAVE to flag missing alt attributes.
  2. Locate each decorative image on the page using one or more of the following methods:
    • Visually
    • Windows / NVDA: Use the G key to cycle through images.
    • MacOS / VoiceOver: Use VO+Command+G to cycle through images.
  3. Verify that each decorative or redundant image is ignored by assistive technologies.

Mobile

  1. For iOS devices, turn off image recognition features.
  2. Navigate through the page with the screen reader.
  3. Verify that decorative images are not focusable and not announced by the screen reader.

Mobile App Test Procedure

  • Follow the procedure for Web Mobile.

PDF Test Procedure

  1. Locate each decorative or redundant image in the PDF.
  2. Verify that each decorative or redundant image is tagged as <Artifact>.

PDF Accessibility Checker

  1. Run PAC on the PDF.
  2. Select the WCAG tab and check if 1.3 Adaptable is marked as a failure.
  3. If it is, activate the Results in Detail button.
  4. Check if Alternative text for "Figure" structure elements is marked as a failure.

Adobe Acrobat Pro

  1. Select Accessibility > Accessibility Check in the Task pane to open the Accessibility Checker Options dialog.
  2. Select "Alternate Text and Headings" from the Category drop-down, and make sure "Figures require alternate text" is checked.
