Overview

Basic Requirements and Best Practices

= Requirement
= Best Practice

  • When text requires more than a 9th grade reading level to understand, supplemental content must be provided.
User Story: As a user with a reading disability, I want a version of difficult text that is easier to read so that I can understand the content.
Success: If you were a user with a reading disability, when you encountered difficult text, you would have additional or alternate content that is easier to understand.

Testing Logic

Does the content, excluding any proper names and titles, contain any difficult text which requires more than a 9th grade reading level?

  • Yes; Is there supplemental content such as explanatory graphics, audio, or an alternate version of the text that is easier to read?
    • Yes; PASS
    • No; See F3.1.5.a.a Difficult text does not have supplemental content.
  • No; PASS

Web Test Procedure

  1. Remove proper nouns and titles from the text.
  2. Evaluate the text for reading level.
  3. If the reading level is higher than 9th grade, verify that there is supplemental content to explain the text or an alternate version that is easier to read.

Failures

  • F3.1.5.a.a Difficult text does not have supplemental content. (Low)
