Overview

This criterion only applies to web content, not flashing caused by user agents. Examples include:

  • Video
  • Animations
  • Blinking elements

Basic Requirements and Best Practices

= Requirement
= Best Practice

  • Content must not flash more than 3 times per second.
  • Content should not flash at all.
User Story: As a user with a seizure disorder, I want to view content that does not flash so that it does not trigger seizures.
Success: If you had a seizure disorder, when viewing flashing content, you would not have seizures.

Testing Logic

Is there any flashing content on the page?

  • Yes; Does the content flash more than 3 times per second?
    • Yes; See {{ site.data.2302.2302[23020101].name }}.
    • No; PASS
  • No; PASS

Web Test Procedure

Desktop and Mobile

  1. Locate any flashing content.
  2. Count the number of flashes per second.
  3. Verify that there are no more than 3 flashes per second.
