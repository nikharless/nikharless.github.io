Overview

Basic Requirements and Best Practices

= Requirement
= Best Practice

  • When the meaning of words depends on the pronunciation, the intended pronunciation must be provided.
User Story: As a user with a cognitive disability, I want pronunciations of ambiguous words so that I can understand the content.
Success: If you were a user with a cognitive disability, when you encountered a word where the meaning depends on the pronunciation, you would have access to a mechanism providing the intended pronunciation.

Testing Logic

Does the content contain any words where the meaning depends on the pronunciation?

  • Yes; Can the pronunciation be determined from the context?
    • Yes; PASS
    • No; Is the pronunciation provided?
      • Yes; PASS
      • No; See {{ site.data.3106.3106[31060101].name }}.
  • No; PASS

Web Test Procedure

  1. Evaluate the text for words where the meaning depends on the pronunciation.
  2. If the pronunciation cannot be determined from the context, verify that the pronunciation is provided.
