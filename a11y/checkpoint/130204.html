Overview

Basic Requirements and Best Practices

= Requirement
= Best Practice

  • The reading order of modal content must be logical and intuitive.
  • When a modal is open, it must not be possible for assistive technology to read outside of it.
User Story: As a blind screen reader user, I want the content to be read in a meaningful sequence so that I can understand the content.
Success: If you could not see the web page, when you use a screen reader, you would hear the content in an order that makes logical sense.

Testing Logic

Are there modals on the page?

  • Yes; When the modal is open, can assistive technology read content outside the modal?
    • Yes; See {{ site.data.1302.1302[13020401].name }}
    • No; PASS
  • No; This checkpoint is not applicable.

Web Test Procedure

Screen Reader

  1. Starting at the top of the content being tested, navigate to and activate any controls that open modals.
  2. Verify that the focus goes to a logical location within the modal when it is opened and that the content that is read makes sense.
  3. Verify that no modal content is skipped.
  4. Verify that the screen reader does not read outside any modals while they are open.
