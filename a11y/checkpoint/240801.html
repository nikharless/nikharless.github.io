Overview

Methods of providing location context include:

  • Site map
  • Breadcrumbs
  • Links to parent web pages
  • Indicators of current location within navigation areas
  • Hierarchy information in page titles

Basic Requirements and Best Practices

= Requirement
= Best Practice

  • Information about the user's location within a set of web pages must be available.
User Story: As a user with a cognitive disability, I want have information about my location within a set of web pages so that I can orient myself.
Success: If you were a user with a cognitive disability, when viewing a web page, you would have information about your location within the set of web pages.

Testing Logic

Is there information available to indicate the user's current location within the site?

  • Yes; PASS
  • No; See 2.4.8.1.1 There is no information about the user's location within the site.

Web Test Procedure

  1. Check if the web page provides any context for where the page is located within its set of web pages.

Failures

  • 2.4.8.1.1 There is no information about the user's location within the site. (Low)
