Overview

This checkpoint applies to blocks of text rather than an entire page. It is possible that a page might require horizontal scrolling while individual blocks of text do not. The intention is that a user wouldn't have to scroll left to right to read a single block of text.

Basic Requirements and Best Practices

= Requirement
= Best Practice

  • When text is resized up to 200%, it must not require horizontal scrolling.
User Story: As a user with a visual or cognitive disability, I want to be able to increase the size of text without having to scroll horizontally so that I don't become disoriented.
Success: If you had low vision, when you resized the text up to 200%, you would be able to read the full line of text without having to scroll horizontally.

Testing Logic

Can the text be resized without assistive technology up to 200%?

  • Yes; When resized to 200%, is horizontal scrolling required to read a full line of text on a full-screen window?
    • Yes; Is the text an item not meant for reading, such as a long url?
      • Yes; PASS
      • No; See {{ site.data.1408.1408[14080501].name }}
    • No; PASS
  • No; See {{ site.data.1404.1404[1404].name }}.

Web Test Procedure

Desktop

  1. Resize the browser window to full screen.
  2. Zoom the text to 200%.
  3. Verify that horizontal scrolling is not required to read a full line of text.

Mobile

This checkpoint does not apply to mobile.
