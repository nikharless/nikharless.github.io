Overview

Basic Requirements and Best Practices

= Requirement
= Best Practice

  • The expanded form or meaning of abbreviations must be provided.
User Story: As a user with a cognitive disability, I want abbreviations to be defined so that I can understand the content.
Success: If you were a user with a cognitive disability, when you encountered abbreviations, you would have definitions provided to help you understand the content.

Testing Logic

Does the text contain any abbreviations or acronyms?

  • Yes; Are definitions provided for them?
    • Yes; PASS
    • No; See 3.1.4.1.1 Definitions are not provided for abbreviations.
  • No; PASS

Web Test Procedure

  1. Read the text of the page to identify any abbreviations.
  2. Verify that definitions are provided.

Failures

  • 3.1.4.1.1 Definitions are not provided for abbreviations. (Medium)
