Overview

Basic Requirements and Best Practices

= Requirement
= Best Practice

  • Keyboard focus must not get stuck on any component.
  • If moving focus from an element requires non-standard keystrokes, instructions must be provided.
User Story: As a keyboard user, I want to be able to navigate freely through the page with the keyboard so that I can use the page.
Success: If you used only a keyboard, when you navigated through a page, your keyboard focus would not get stuck anywhere.

Testing Logic

Is it possible to move focus off of each page element using only the keyboard?

  • Yes; Is it possible to move focus off of each page element with the screen reader turned on?
    • Yes; Does any element require non-standard keystrokes to move the focus off of it?
      • Yes; Are instructions provided explaining the non-standard keystrokes?

        • No; See {{ site.data.2102.2102[21020103].name }}.
      • No; PASS
    • No; See {{ site.data.2102.2102[21020102].name }}.
  • No; See {{ site.data.2102.2102[21020101].name }}.

Web Test Procedure

Desktop Keyboard

  1. Using only the keyboard (no mouse or screen reader), navigate through the page.
    • Press the Tab key to navigate the focus through menu items, form inputs, links, help items, and other interactive UI elements.
    • Press Shift+Tab to navigate the focus in the reverse direction through all interactive UI elements.
    • In grouped inputs, menus, or list item elements, press the up, down, right, or left arrow keys to move between items inside the element.
    • To escape a modal without a close button, try the Esc key.
    • To escape a menu, try Esc, Enter, or Space.
    • If escaping an element requires non-standard keystrokes, verify that instructions are provided.

Desktop Screen Reader

  1. Turn on the screen reader.
  2. Follow the process for Desktop Keyboard.

Mobile

Screen Reader (iOS and Android)

  1. Turn on the screen reader.
  2. Swipe to each interactive element on the page.
  3. Verify that you are able to navigate away from each interactive element using standard gestures.

Bluetooth Keyboard (Android)

  1. Pair a Bluetooth keyboard with your device.
  2. With TalkBack off, follow Desktop Keyboard procedure.
