Overview

Basic Requirements and Best Practices

= Requirement
= Best Practice

  • Section headings must be used to organize the content.
  • Section headings should be programmatically determinable. (See 1.3.1.e Headings for requirement.)
User Story: As a user with a cognitive disability, I want sections of content to have headings so that I can and understand how the content is organized.
Success: If you were a user with a cognitive disability, when you read or navigate through a page, you can understand how the document is organized.

Testing Logic

Is the content of the page organized into sections?

  • Yes; Does each section have a heading?
    • Yes; PASS
    • No; See F2.4.10.a.a Sections of content do not have headings.
  • No; PASS

Web Test Procedure

  1. Determine whether the page content is divided into sections.
  2. If so, verify that each section has a heading or title.

PDF Testing

  1. Determine whether the document is divided into sections.
  2. If so, verify that each section has a heading or title.

Failures

  • F2.4.10.a.a Sections of content do not have headings. (Medium)
