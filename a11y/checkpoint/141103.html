Overview

Exceptions include:

  • Decorative graphics
  • Graphics with accompanying text that conveys the same information
  • Graphics where the colors are essential (such as logos)

Examples

Single-color Icon

A single-color icon must have 3:1 contrast with the background.

Multi-color Icon

The hand in this icon has more than one color. However, the shape of the hand is understandable without contrast between those colors. The contrast between the hand and the background is sufficient, so the icon passes.

Pie Chart

This pie chart has some wedges which have a contrast ratio of less than 3:1 with each other. However, a white line is visible between them which does have 3:1 contrast with the wedges on either side, so this passes.

Text Character Used as a Graphic Symbol

Text characters such as x, <, and > are often used as graphic symbols. As such, the non-text contrast rule of 3:1 applies.

Basic Requirements and Best Practices

= Requirement
= Best Practice

  • Parts of graphics required to understand the content must have a contrast ratio of 3:1 against adjacent colors.
  • The color contrast of non-text visual information in icons should have a 4.5:1 contrast.
User Story: As a user with low vision, I want graphical objects to have sufficient contrast so that I can tell what they are.
Success: If you had low vision, when you encountered graphical objects such as icons or graphs, you would be able to distinguish them and understand their purpose.

Testing Logic

Does the page include graphics which convey meaningful information?

  • Yes; Do parts of the graphic that are required to understand the content have a contrast ratio of at least 3:1 against adjacent colors?
    • Yes; PASS
    • No; Does the graphic have more than one color?
      • Yes; Is the graphic a chart or graph?
        • Yes; See 1.4.11.3.3 Adjacent parts of a graph do not have 3:1 contrast.
        • No; Is the graphic a text character used as a symbol?
          • Yes; See 1.4.11.3.4 Text characters used as graphic symbols do not have 3:1 contrast.
          • No; See 1.4.11.3.2 Part of graphic does not have 3:1 contrast against adjacent colors.
      • No; See 1.4.11.3.1 Icon does not have 3:1 contrast with its background.
  • No; PASS

Web Test Procedure

  1. Identify each graphic on the page that includes information required for understanding the content.
  2. Mentally break down each graphic into its smallest meaningful pieces. Each meaningful piece is a graphical object.
  3. For each graphical object, verify that the contrast against adjacent colors is at least 3:1.
  4. If there are multiple colors and/or a gradient, choose the least contrasting area to test.
  5. If the area with the least contrast is less than 3:1, assume that area is invisible.
  6. Verify that the meaning of the graphical object is understandable without the invisible area.

Mobile App Test Procedure

Follow the web test procedure.

PDF Test Procedure

Follow the web test procedure.

Failures

  • 1.4.11.3.1 Icon does not have 3:1 contrast with its background. (Medium)
  • 1.4.11.3.2 Part of graphic does not have 3:1 contrast against adjacent colors. (Medium)
  • 1.4.11.3.3 Adjacent parts of a graph do not have 3:1 contrast. (Medium)
  • 1.4.11.3.4 Text characters used as graphic symbols do not have 3:1 contrast. (Medium)
