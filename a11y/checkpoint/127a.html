Examples

See examples from 3PlayMedia under Extended Audio Description.

Basic Requirements and Best Practices

= Requirement
= Best Practice

  • Prerecorded synchronized media which does not have sufficient audio pauses to fit all the important visual information must have an extended audio description of the visual content.
User Story: As a blind user, I want to listen to an audio description along with the video so that I can get all of the same information that is conveyed in the visual portion of the video.
Success: If you could not see the video, when you listen to the audio description you would get all of the information conveyed in the video portion.

Testing Logic

Does the page contain synchronized media (e.g., video with dialogue)?

  • Yes; Does the video content in the synchronized media have an audio description?
    • Yes; Does the audio description include all the important actions, visuals, and text in the video?
      • Yes; PASS
      • No; Does the video have sufficient pauses to include all the important actions, visuals, and text in the video?
        • Yes; See F1.2.5.a.b Existing audio description is incorrect.
        • No; See F1.2.7.a.b Existing audio description is inadequate.
    • No; Is the video itself a media alternative for text?
      • Yes; Is it clearly labeled as such?
        • Yes; PASS
        • No; Add text making it clear that the video is an alternative for the text content.
      • No; Does the video have sufficient pauses to include all the important actions, visuals, and text in the video?
        • Yes; See F1.2.5.a.a Synchronized media does not have an audio alternative for the video portion.
        • No; See F1.2.7.a.a Synchronized media does not have an extended audio description for the video portion.
  • No; PASS

Web Test Procedure

  1. Locate each video element on the page.
  2. Determine whether the video content contains any information not provided through the audio content.
  3. If it does, verify that it has an audio description that provides the visual information.

Mobile App Test Procedure

  • Follow the procedure for Web.

Failures

  • F1.2.7.a.a Synchronized media does not have an extended audio description for the video portion. (Medium)
  • F1.2.7.a.b Existing audio description is inadequate. (Medium)
