Overview

The spacing between letters, words, lines of text, and paragraphs can be adjusted and still be readable and functional.

Exceptions include:

  • Images of text
  • PDF
  • Captions embedded in a video
  • Text within a canvas

Examples

Basic Requirements and Best Practices

= Requirement
= Best Practice

  • The spacing between letters, words, lines of text, and paragraphs can be adjusted.
  • Text remains readable and functional when adjusted to:
    • Line height: 1.5 times the font size
    • Paragraph spacing: 2 times the font size
    • Letter spacing: .12 times the font size
    • Word spacing: .16 times the font size
User Story: As a user with vision or reading impairments, I want to be able to adjust text spacing so that I can more easily read it.
Success: If you had a reading impairment, when you encountered blocks of text, you would be able to adjust the spacing to suit your needs without obscuring the text or functionality.

Web Test Procedure

Desktop

  1. Browser 1: Open the page in a browser with the viewport set to 1366px width by 768px height and zoom set to 100%.
  2. Browser 2: Open the page in a second browser with the viewport set to 1366px width by 768 height and zoom set to 100%.
  3. Activate Dylan Barrell’s 1.4.12 Text Spacing Bookmarklet.
  4. Visually confirm that text spacing responds to the bookmarklet. The expected change is an increase in:
    • Line height (1.5 times the font size)
    • Spacing following paragraph (at least 2 times the font size)
    • Letter spacing (at least .12 times the font size)
    • Word spacing (at least .16 times the font size)
  5. Compare the content and functionality available in the two browsers. Make sure there is no loss of content or functionality on the page where the bookmarklet has been applied.
