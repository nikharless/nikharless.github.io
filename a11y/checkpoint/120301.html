Overview

Example

This video has the audio description included in the main audio track. If you view the video on YouTube, it also has a text transcript.

Basic Requirements and Best Practices

= Requirement
= Best Practice

  • Prerecorded synchronized media must have a text or audio description of the visual content.
  • The text or audio description should be presented next to the video and be easy to find.
User Story: As a blind user, I want to listen to an audio description along with the video or read a transcript with my screen reader so that I can get all of the same information that is conveyed in the visual portion of the video.
Success: If you could not see the video, when you listen to the audio description or read the transcript, you would get all of the information conveyed in the video portion.

Testing Logic

    • Yes; Does the audio description include all the important actions, visuals, and text in the video?

      • No; Is the video accompanied by a text transcript?
        • Yes; Does the transcript include all the important actions, visuals, and text in the video?
          • Yes; PASS

        • No; See {{ site.data.1203.1203[12030101].name }}
  • No; PASS

Web Test Procedure

  1. Locate each video element on the page.
  2. Determine whether the video content contains any information not provided through the audio content.
  3. If it does, verify that it has a text or audio description that provides the visual information in another form.
