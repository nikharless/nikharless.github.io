Overview

Basic Requirements and Best Practices

= Requirement
= Best Practice

  • The width of a block of text must be able to be adjusted so that no more than 80 characters are displayed without the need for horizontal scrolling.
User Story: As a user with a visual or cognitive disability, I want to be able to adjust the length of lines of text so that I can more easily read the content.
Success: If you had difficulty reading long lines of text, when you encountered blocks of text that you found difficult to read, you would be able to adjust the the width of the lines to suit your specific needs.

Testing Logic

Does the page have blocks of text that are more than 80 characters by default?

  • Yes; Can the user adjust the width of the text block to no more than 80 characters by resizing?
    • Yes; PASS
    • No; See {{ site.data.1408.1408[14080201].name }}
  • No; PASS

Web Test Procedure

Desktop

  1. Locate any blocks of text which are wider than 80 characters.
  2. Resize the browser window so that only 80 characters of text is displayed per line.
  3. Verify that there is no horizontal scrolling necessary to view the full line of text.
