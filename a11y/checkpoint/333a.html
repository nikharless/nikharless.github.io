Overview

Basic Requirements and Best Practices

= Requirement
= Best Practice

  • Error feedback must provide suggestions for error correction, if possible.
User Story: As a user with a cognitive disability, I want error messages to provide suggestions for how to correct the error so that I know how to correct it.
Success: If you were a user with a cognitive disability, when an error was automatically detected, suggestions would be provided to help you correct the error.

Testing Logic

Are there interactions which trigger errors on the page?

  • Yes; When an error occurs, does the error message provide suggestions for how to correct the error?
    • Yes; Is the previously entered data re-displayed?
      • Yes; PASS
      • No; Would re-displaying the data jeopardize the security or purpose of the content?
        • Yes; PASS
        • No; See F3.3.3.a.d Previously entered data is not redisplayed.
    • No; Would providing suggestions jeopardize the security or purpose of the content?
      • Yes; PASS
      • No; Does the input in error require a specific data format?
        • Yes; See F3.3.3.a.a Error message does not indicate expected data format.
        • No; Does the input in error require an entry from a specific range of values?
          • Yes; See F3.3.3.a.b Error message does not indicate expected range of data values.
          • No; See F3.3.3.a.c Error message does not provide suggestions for correcting the error.
  • No; Not applicable

Web Test Procedure

Desktop

  1. Locate any forms in the content.
  2. Deliberately trigger errors by entering user input that falls outside the required format or values, and submit the form.
  3. If an input error is automatically detected, verify that the error message gives a suggestion about how to fix it.

Mobile

Using VoiceOver or TalkBack, follow the Desktop Web procedure.

Mobile App Test Procedure

Follow the procedure for Mobile Web.

PDF Test Procedure

Follow the procedure for Desktop Web.

Failures

  • F3.3.3.a.a Error message does not indicate expected data format.
  • F3.3.3.a.b Error message does not indicate expected range of data values.
  • F3.3.3.a.c Error message does not provide suggestions for correcting the error.
  • F3.3.3.a.d Previously entered data is not redisplayed.
