Basic Requirements and Best Practices

= Requirement
= Best Practice

  • At least two ways must be available to find web pages on the site.
User Story: As a user with a disability, I want multiple ways to locate a web page on a site so that I can use the method that is easiest for me.
Success: If you were a user with a disability, when trying to locate a page on a website, you can use a method that is easier for you.

Testing Logic

Is there more than one way to locate the web page within a set of web pages?

  • Yes; PASS
  • No; Is the page a step in or the result of a process?
