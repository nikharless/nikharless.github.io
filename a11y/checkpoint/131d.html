Basic Requirements and Best Practices

= Requirement
= Best Practice

  • Form controls which require a group label to be fully understood must be programmatically associated with the group label.
User Story: As a blind user, I want grouped form controls to be programmatically associated with the group label so that I know the context of the individual form controls.
Success: If you could not see a group of form controls, when you read each control with a screen reader, you understand the context of the control.

Testing Logic

Are there groups of related input controls where the individual labels for each input control do not provide enough context?

  • Yes; Are the input controls programmatically grouped?
    • Yes; Does the group of controls have a visible label?
      • Yes; Are the individual controls programmatically associated with the group label?
        • Yes; PASS
        • No; Are the controls checkboxes?
          • Yes; See F1.3.1.d.a Individual checkboxes are not programmatically associated with group label.
          • No; Are the controls radio buttons?
            • Yes; See F1.3.1.d.b Individual radio buttons are not programmatically associated with group label.
            • No; See F1.3.1.d.c Form control does not have a programmatic name.
      • No;
    • No;
  • No; PASS

Web Test Procedure

  1. Locate any form controls that have a group label.
  2. Determine if the purpose of the individual controls can be determined without the group label.
  3. If not, verify that the fields are grouped and associated with the group label

Screen Reader Testing

  1. Locate any form controls that have a group label.
  2. Determine if the purpose of the individual controls can be determined without the group label.
  3. If not, navigate to the control and verify that the group label is read along with the individual control label.

PDF Test Procedure

  1. Locate any form controls that have a group label.
  2. Determine if the purpose of the individual controls can be determined without the group label.
  3. If not, navigate to the control with a screen reader and verify that the group label is read along with the individual control label.

Failures

  • F1.3.1.d.a Individual checkboxes are not programmatically associated with group label (Medium)
  • F1.3.1.d.b Individual radio buttons are not programmatically associated with group label (Medium)
  • F1.3.1.d.c Form control does not have a programmatic name (Medium)
