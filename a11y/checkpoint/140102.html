Overview

Some means to distinguish links from regular text:

  • Underline
  • Bold
  • Italic
  • Chevron
  • Arrow
  • Outline
  • Highlight
  • Placement in an obvious navigation area

Basic Requirements and Best Practices

= Requirement
= Best Practice

  • Color alone must not be used to distinguish links from regular text.
    • If links have a contrast ratio of at least 3:1 with regular text, this counts as an additional visual distinction.
  • Links should not be indistinguishable from surrounding text. However, this scenario would not fail this success criterion, because no users are able to distinguish the links.
User Story: As a color blind user, I want to have an alternative non-color-related means to distinguish links so that I get the same information that non-color blind users get.
Success: If you could not see colors, when you view the page, you would be able to distinguish the links from surrounding text.

Testing Logic

Is font color alone being used to distinguish links from regular text content?

  • Yes; Is there at least a 3:1 contrast ratio between link text and regular text?
    • Yes; Is there any additional meaning to the link color? (For example, a wiki with different color links for existing and non-existing pages)
      • Yes; Is there another means besides color to convey this meaning?
        • Yes; PASS
        • No; See F1.4.1.b.b Color alone is used to convey additional meaning between links.
      • No; PASS
    • No; See F1.4.1.b.a Color alone is used to distinguish links from text.
  • No; PASS

Web Test Procedure

  1. Locate any links on the page.
  2. Verify that there is a means other than color to distinguish them from regular text.
  3. Determine if there is additional meaning that is conveyed by link color.
  4. If so, verify that there is also a means other than color to convey this additional meaning.

Mobile App Test Procedure

In the context of native mobile applications, links are elements that do tasks such as opening a browser or another app such as a phone, email, or contact app.

  1. Follow the procedure for Web.

PDF Test Procedure

  1. Follow the procedure for Web.

Failures

  • F1.4.1.b.a Color alone is used to distinguish links from text. (Medium)
  • F1.4.1.b.b Color alone is used to convey additional meaning between links. (Medium)
