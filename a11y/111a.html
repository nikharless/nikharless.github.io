All linked images must have a descriptive text alternative that serves the same purpose and presents the same information as the linked image.

Success: If you replaced the active image with the text alternative, you would still understand the destination, purpose, or function of the control.

Text alternatives for active images include the following:

  • alt attribute value
  • title attribute value
  • aria-label property
  • aria-labelledby property

Active images include the following:

  • Images inside a link
  • Images inside a button
  • <area> elements in an image group
  • <input type="image"> elements
