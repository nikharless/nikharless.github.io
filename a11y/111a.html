{{ page.title }}

All linked images must have a descriptive text alternative that serves the same purpose and presents the same information as the linked image.

Success: If you replaced the active image with the text alternative, you would still understand the destination, purpose, or function of the control.

Text alternatives for active images include the following:

  • alt attribute value
  • title attribute value
  • aria-label property
  • aria-labelledby property

Active images include the following:

  • Images inside a link
  • Images inside a button
  • <area> elements in an image group
  • <input type="image"> elements

Basic Requirements and Best Practices

  • If a control is labeled with only an image, the image must have alternative text which describes the function or destination of the control.
  • If a control is labeled with both an image and text:
    • If the image provides information not conveyed by the associated text, the image's alternative text along with the visible text must together describe the function or destination of the control.
    • If the image does not provide any information not conveyed by the associated text, check against 1.1.1.d Alternative Text (Decorative Images).
  • If the image contains meaningful text, the text alternative should contain that exact text.
  • Text alternatives for linked images should not duplicate adjacent link text for the same destination. Combine the image and text into a single link and check the image against 1.1.1.d Alternative Text (Decorative Images).
  • Text alternatives for active images should not include words such as "link" or "button".
  • Trademarks, copyright symbols, etc. on an image should be included in the text alternative.
  • Text alternatives should be as succinct as possible.
