WCAG | 1.1.1 Non-text Content

Each audio and video element must have a text alternative which summarizes its purpose.

Success: You are able to understand what the audio and video content is before you play it.

Basic Requirements and Best Practices

= Requirement
= Best Practice

  • Each audio and video element must have a text alternative which summarizes its purpose.
  • If there is text in the audio/video preview, that exact text should be included in the text alternative.

Testing Logic

Is there audio or video content on the page?

  1. Yes; Does the audio or video content have a short text alternative?
    1. Yes; Does the text alternative sufficiently summarize the content of the media?
      1. Yes; PASS
      2. No; Change the text alternative to sufficiently summarize the content of the media.
    2. No; Add alternative text summarizing the content of the media.
  2. No; PASS

Does the text alternative for the audio-only content refer to the accompanying transcript?

If a transcript does not exist, see 1.2.1 Audio-only and Video-only (Prerecorded).

  1. Yes; PASS
  2. No; Include a reference to the accompanying transcript in the text alternative for the audio content.

Web Test Procedure

Desktop

  1. Locate each audio or video element using one or more of the following methods:
    • Visually
    • Windows / NVDA: Use the O key to cycle through objects.
    • MacOS / VoiceOver: Use VO+Command+O to cycle through objects.
  2. Verify that it has a text alternative that summarizes its content.

Mobile

  1. For iOS devices, turn off image recognition features.
  2. Navigate to each audio or video element.
  3. Verify that it has a text alternative that summarizes its content.

Mobile App Test Procedure

  • Follow the procedure for Web Mobile.

Failures

  • F1.1.1.h.a Audio object does not have a text alternative. (Medium)
  • F1.1.1.h.b Video object does not have a text alternative. (Medium)
  • F1.1.1.h.c Existing text alternative does not adequately summarize the content of the audio. (Medium)
  • F1.1.1.h.d Existing text alternative does not adequately summarize the content of the video. (Medium)
