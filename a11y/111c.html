WCAG | 1.1.1 Non-text Content

Basic Requirements and Best Practices

= Requirement
= Best Practice

  • Complex images that cannot be sufficiently described with a short text alternative must also have a long description that fully conveys all the information provided by the image.
  • Complex images must have a short descriptive text alternative that summarizes the information provided by the image.
  • The short text alternative should reference the location of the long description.
  • Short text alternatives for complex images should not include words such as "image" or "graphic". Words like "graph" or "chart" may be used.
  • Short text alternatives should be as succinct as possible.
Success: If you could not see the image, you would still get all the information intended to be conveyed by the image.

Text alternatives for complex images may be provided in context or on a separate page, such as by:

  • Providing a long description in the form of adjacent text, preferably with a reference in the short text alternative
  • Providing graph information in table format in addition to the complex image
  • Providing a link to a long description, placed immediately adjacent to the complex image
  • Providing a long description that is hidden within a collapsible region
  • Providing a long description in a dialog that is opened by user choice
  • Providing a link to a long description on another page via the longdesc attribute (Warning: This is the least preferred option, as it is only available to screen reader users.)

Testing Logic

Are there complex images on the page that cannot be sufficiently described with a short text alternative?

  • Yes; Does each complex image have a standard short text alternative?
    • Yes; Does each complex image also have a long text alternative?
      • Yes; Is the long text alternative provided in context or via an adjacent link to a separate page?
        • Yes; Does the long text alternative sufficiently describe all visual complexities and relationships conveyed visually?
          • Yes; PASS
          • No; Rewrite the long text alternative to include all details, complexities, and relationships conveyed visually.
        • No; Provide the long text description in context or via an adjacent link to a separate page.
      • No; Add a descriptive long text alternative that sufficiently describes all visual complexities and relationships conveyed visually. It may be provided in context or on a separate page.
    • No; Add a text alternative to the active image.
  • No; PASS

Web Test Procedure

Desktop

  1. Run axe DevTools or WAVE to flag missing text alternatives.
  2. Locate each complex image on the page.
  3. Verify that each complex image has a short text alternative which summarizes the information conveyed by the image.
  4. Verify that a long description has also been provided.

Mobile

  1. For iOS devices, turn off image recognition features.
  2. Navigate to each complex image.
  3. Verify that each complex image has a short text alternative which summarizes the information conveyed by the image.
  4. Verify that a long description has also been provided.

Mobile App Test Procedure

  • Follow the procedure for Web Mobile.

PDF Test Procedure

Find missing text alternatives.

PDF Accessibility Checker

  1. Look for failures under Logical Structure > Alternative Descriptions > Alternative text for "Figure" structure elements.

Adobe Acrobat Pro

  1. Select Accessibility > Accessibility Check in the Task pane to open the Accessibility Checker Options dialog.
  2. Select "Alternate Text and Headings" from the Category drop-down, and make sure "Figures require alternate text" is checked.

Verify the text alternatives are sufficient.

  1. Locate each complex image in the PDF.
  2. Verify that each complex image has a short text alternative which summarizes the information conveyed by the image.
  3. Verify that a long description has also been provided.

Failures

  • F1.1.1.c.a Complex image is missing a short text alternative. (Medium)
  • F1.1.1.c.b Existing short text alternative does not adequately summarize the purpose of the complex image. (Medium)
  • F1.1.1.c.c Complex image is missing a long text description. (High)
  • F1.1.1.c.d Existing long text alternative does not convey all the information in the complex image. (Medium)
