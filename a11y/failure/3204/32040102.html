Expected Result

Links that have the same target within a set of web pages are identified consistently.

Actual Result

Links that have the same target within a set of web pages are not identified consistently.

Solution

Ensure links that have the same target within a set of web pages have consistent text and accessible names each time they appear.

Passing Example

Page 1

Lorem ipsum dolor sit amet, consectetur adipiscing elit. Mauris ac malesuada nisl, at efficitur metus. Aenean volutpat, dolor non ornare luctus, leo augue venenatis justo, a dignissim dui urna in orci. Nulla sed orci non sem luctus consequat.
