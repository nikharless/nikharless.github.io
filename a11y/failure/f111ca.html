Expected Result

Complex images have a short descriptive text alternative that summarizes the information provided by the image.

Actual Result

The complex image does not have a short text alternative.

Solution

Add a short text alternative which summarizes the information provided by the image using one of the following methods:
  • Add an alt attribute to the <img> tag, OR
  • Add an aria-label attribute to the image tag, OR
  • Add an aria-labelledby attribute to the image tag, set to the id value of the text to use as a label, OR
  • Add a title attribute to the <img> tag

Example

Complex Image with Link to Long Description

<figure role="group">
    <img src="constellations.png" alt="Historical chart of constellations.">
    <figcaption>
        <a href="long-description.html">Follow this link for a detailed description of the chart.</a>
    </figcaption>
</figure>

Why It's Important

Blind and visually impaired users need to have a short text alternative for complex images so that they can understand the purpose of the image and know whether they want to explore the long description.
