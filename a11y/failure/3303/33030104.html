Expected Result

Previously entered data is redisplayed in input fields so the user can see what needs to be corrected.

Actual Result

Previously entered data is not redisplayed in input fields.

Solution

Redisplay previously entered data in the input fields.

Examples

HTML

The username is too short. It must be at least 10 characters and include an uppercase letter.

Not Applicable

  • No error detection:
    The form does not check for errors.
  • Existence of errors:
    File under {{ site.data.3301.3301[3301].name }}.
  • Programmatic association:
    File under {{ site.data.1301.1301[1301].name }}.
  • Assistive technology notification:
    File under {{ site.data.4103.4103[4103].name }}.

Disabilities Affected

  • Blind/vision impaired: Users are aware of the nature of the error and how to crrect it.
  • Mobility impaired: Users can reduce the number of times they need to change an input value.
  • Cognitive Impairment: Users are able to understand how to successfully complete a form.
