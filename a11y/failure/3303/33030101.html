Expected Result

Error messages indicate the expected data format.

Actual Result

Error message does not indicate the expected data format.

Solution

Include the expected data format in the error message.

PDF

Indicate the required format for the field.

  1. Select All tools > Prepare a form.
  2. Activate a form field to open the Properties dialog.
  3. In the Format tab, select the appropriate option from the Select format category drop-down.
  4. Select the appropriate options for the selection.

Examples

HTML

Please enter your email address in the format: yourname@example.com

PDF

Why It's Important

People can address errors faster and with reduced effort.
