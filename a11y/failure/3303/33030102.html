Expected Result

Error messages indicate the expected range of data values.

Actual Result

Error message does not indicate the expected range of data values.

Solution

Include the expected range of data values in the error message.

PDF

Indicate the required range of values for the field.

  1. Select All tools > Prepare a form.
  2. Activate a form field to open the Properties dialog.
  3. In the Format tab, select Special from the Select format category drop-down.
  4. Under Special Options, select Arbitrary Mask.
  5. In the field below the selection, enter the correct mask. See "Format text and dropdown fields" in Customize PDF form fields for examples of what to enter for the mask.

Examples

HTML

Please lengthen this text to 10 characters or more (you are currently using 3 characters).

PDF

Why It's Important

People can address errors faster and with reduced effort.
