Expected Result

The mechanism for pausing, stopping, or controlling the volume passes all applicable Success Criteria.

Actual Result

The mechanism for pausing, stopping, or controlling the volume does not pass all applicable Success Criteria.

Solution

Provide a mechanism that itself passes all applicable Success Criteria.

For example, if a button is used, the button must meet all requirements for buttons.

Failing Examples

  • A pause button is provided that cannot be operated with a keyboard.
  • A volume control is provided that does not provide feedback via assistive technology.
  • A custom control is provided that does not announce a role.
  • A toggle button is provided that does not correctly announce its state.

Passing Example

Not Applicable

  • No mechanism to control auto-playing content:
    File under {{ site.data.1402.1402[14020101].name }}.

Disabilities Affected

If the mechanism itself does not pass all applicable Success Criteria, it does not sufficiently perform its intended function.
