Expected Result

Information about the user's location within the set of web pages is available.

Actual Result

There is no information providing context for the user's location within the set of web pages.

Solution

Provide location context using one of the following (or some other method):

  • Site map
  • Breadcrumbs
  • Links to parent web pages
  • Indicators of current location within navigation areas
  • Hierarchy information in page titles

Examples

Breadcrumbs

  1. {{ page.root }}
  2. {{ page.sctitle }}
  3. {{ page.title }}

Current Page Highlighted in Navigation

  • Home
  • News
  • Contact
  • About

Why It's Important

Location indicators reduce confusion for people with cognitive disabilities.
