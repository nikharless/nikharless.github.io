Expected Result

When both column and row headers are present, the header cells are marked up with the scope attribute.

Actual Result

Both column and row headers are present, but the header cells do not have a scope attribute.

Solution

  • Add scope="col" to column headers.
  • Add scope="row" to row headers.

Example

Table with Column and Row Headers

Name Time Attempts
John 18 min 3
Jane 20 min 1 
<table>
    <tr>
        <th scope="col">Name</th>
        <th scope="col">Time</th>
        <th scope="col">Attempts</th>
    </tr>
    <tr>
        <th scope="row">John</th>
        <td>18 min</td>
        <td>3</td>
    </tr>
    <tr>
        <th scope="row">Jane</th>
        <td>20 min</td>
        <td>1</td>
    </tr>
</table>

Why It's Important

Table relationships are visible at a glance for sighted users. Assistive technology users need to get this relationship information from the semantic structure and to be able to easily navigate the data.
