Expected Result

When a web page updates with dynamic content using the same url, the title of the web page also updates to reflect the current content.

Actual Result

The web page updates with dynamic content, but the title does not change to reflect the current content.

Solution

When the page content changes, ensure that the title is also changed.

Examples

<html>
    <head>
        <title>
            Search Results: Accessibility
        </title>
    </head>
    ...
</html>

Disabilities Affected

Users can easily identify whether the information on the web page is relevant to their needs.
  • Blind/visually impaired: Screen reader users can differentiate content when multiple web pages are open.
  • Motion impaired: Users who rely on audio when navigating between web pages can differentiate content.
  • Cognitively impaired: Users can identify content by its title.
