Expected Result

When a web page updates with dynamic content using the same url, the title of the web page also updates to reflect the current content.

Actual Result

The web page updates with dynamic content, but the title does not change to reflect the current content.

Solution

When the page content changes, ensure that the title is also changed.

Examples

<html>
  <head>
    <title>
      Search Results: Accessibility
    </title>
  </head>
  ...
&lgt;/html>

Why It's Important

Screen reader users, as well as people with cognitive, memory, or reading disabilities benefit from the ability to identify content by its title.
