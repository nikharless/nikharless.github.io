Expected Result

The <head> element of the web page has a <title> element which contains a descriptive title for the web page.

Actual Result

The <head> element does not contain a <title> element.

Solution

Add a descriptive title within the <title> element.

Examples

<html>
    <head>
        <title>
            Understanding Success Criterion 2.4.2: Page Titled
        </title>
    </head>
    ...
</html>

Why It's Important

Users can easily identify whether the information on the web page is relevant to their needs.
  • Blind/visually impaired: Screen reader users can differentiate content when multiple web pages are open.
  • Motion impaired: Users who rely on audio when navigating between web pages can differentiate content.
  • Cognitively impaired: Users can identify content by its title.
