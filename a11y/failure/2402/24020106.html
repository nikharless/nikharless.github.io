Expected Result

PDF file has a title describing the content.

Actual Result

PDF file does not have a title describing the content.

Solution

Add a descriptive title describing the PDF file's content.

Adobe Acrobat Pro

  1. Select Menu > Document properties....
  2. In the Document Properties dialog, select the Description tab.
  3. Enter a descriptive title in the Title field.
  4. Activate the OK button.

Examples

Why It's Important

Screen reader users, as well as people with cognitive, memory, or reading disabilities benefit from the ability to identify content by its title.
