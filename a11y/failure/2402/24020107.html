Expected Result

PDF file has a title describing the content.

Actual Result

PDF file has a title describing the content, but it is not displayed in the title bar.

Solution

Set the PDF preferences to display the title in the title bar.

Adobe Acrobat Pro

  1. Select Menu > Document properties....
  2. In the Document Properties dialog, select the Initial View tab.
  3. Under Window Options, select Document Title from the Show menu.
  4. Activate the OK button.

Examples

Why It's Important

Screen reader users, as well as people with cognitive, memory, or reading disabilities benefit from the ability to identify content by its title.
