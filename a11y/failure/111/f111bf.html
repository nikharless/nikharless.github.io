Expected Result

The text alternative for the meaningful image includes all meaningful text.

Actual Result

The meaningful image has a text alternative, but it does not include all meaningful text in the image.

Solution

Change the text alternative to include all the meaningful text in the image.

Examples

Describing this image as "Scrabble tiles on a table" leaves out the meaningful text that they spell out.

Photo

<img src="carpe-diem.jpg" alt="Scrabble tiles on a table that spell out Carpe Diem">

Why It's Important

Blind and visually impaired users need to have a text alternative that describes all the information conveyed by the image in context.
