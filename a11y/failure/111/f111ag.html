Expected Result

The text alternative for the functional image describes the purpose of the control.

Actual Result

The functional image has a text alternative, but it does not describe the purpose of the control.

Solution

Change the text alternative to describe the purpose of the image.

Example

Search Button

A text alternative such as "magnifying glass" would be inappropriate for this control, because it doesn't describe the function.

<input type="image" src="submit.png" alt="Search">

Why It's Important

Blind and visually impaired users need to have a text alternative that describes the function of the control so that they know what it does. The literal description of the image is irrelevant.
