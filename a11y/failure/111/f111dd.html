Expected Result

Redundant images must be hidden from screen readers.

Actual Result

The redundant image is read by screen readers.

Solution

Hide the image using one of the following methods:
  • Add alt="" to the <img> tag.
  • Add aria-hidden="true" to the image element.
  • Implement the image as a CSS background image.

Examples

Combined Image and Text Link

Flag images are used in a list of country links simply to illustrate each one. Since the name of the country is already provided in text, the image is redundant. The link and image should be combined within a single link and the image hidden from screen readers.


Norway 
<a href="norway.html">
    <img src="norway-flag.png" alt="">
    Norway
</a>

Why It's Important

Blind and visually impaired users want decorative or redundant images hidden so they do not get overwhelmed with unimportant screen reader chatter.
