Expected Result

A cognitive function test is not required for authentication unless an alternative or assistive mechanism is provided or the test involves object recognition or personal non-text content.

Actual Result

A cognitive function test is required for authentication, but an alternative or assistive mechanism is not provided and the test does not involve object recognition or personal non-text content.

Solution

Provide one of the following:

  • An alternative authentication method that is not a cognitive function test, OR
  • A mechanism to assist the user in completing the cognitive function test.

Examples

  • Allowing for auto-filling of username and password fields
  • Allowing for copy and paste into fields
  • Allowing for auto-fill or copy and paste for two-factor authentication codes

Why It's Important

Some people with cognitive disabilities cannot solve puzzles, memorize a username and password, or retype one-time passcodes.
