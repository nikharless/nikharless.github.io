WCAG | 1.1.1 Non-text Content | 1.1.1.a Functional Images (Text Alternative)

Expected Result

 If a control is labeled with only an image, the image must have a text alternative which describes the function or destination of the control.

Actual Result

 The control is labeled with only an image, but the image does not have Alternate Text.

Solution

 Add Alternate Text to the image either in the source document or directly in the PDF.

Adobe PDF

  1. Select Accessibility > Reading Order
  2. Right-click on the image and select Edit Alternate Text....
  3. In the dialog, for Alternate Text, enter text which describes the function or destination of the control.

Example

Image as Link
Why It's Important Blind and visually impaired users need to have a text alternative to functional images so that they can be interpreted by assistive technology such as screen readers and braille devices. Without a text alternative, these users would not know what the link or control does.
