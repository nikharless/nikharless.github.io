Expected Result

All functionality is available through the keyboard interface.

Actual Result

The modal cannot be dismissed using only the keyboard without submitting data.

Solution

  • Ensure any close buttons in the modal can be activated with the keyboard.
  • Enable the use of the Esc key to close the modal.

Examples

Scripted Link

Using onclick allows activation with both the mouse and the keyboard.

Example Modal

<a href="#" onclick="return doStuff();">Click this link</a>

Why It's Important

Some users with vision disabilities or hand tremors are unable to use a mouse well or at all.
