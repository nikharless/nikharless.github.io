Expected Result

All functionality is available through the keyboard interface.

Actual Result

An action cannot be performed using only a keyboard, because the shortcut conflicts with an existing user agent or assistive technology shortcut, and there is no equivalent means of achieving the action.

Solution

Change the shortcut to one that does not conflict with existing user agent or assistive technology shortcuts.

Examples

Press Shift + Alt + s to open the search.

Why It's Important

Assistive technology provides users with shortcuts for browsing the web with the keyboard. These shortcuts will override any conflicting shortcuts on a web page.
