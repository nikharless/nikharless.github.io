WCAG | 1.2.1 Audio-only and Video-only (Prerecorded) | 1.2.1.b Text or Audio Description (Prerecorded Video-only - No Dialogue)

Expected Result Prerecorded video-only content must have a text or audio alternative that includes all of the information presented in the video.
Actual Result The prerecorded video-only content does not have an audio description or text transcript.
Solution Add an audio description or text transcript that includes all of the information presented in the video using one of the following methods:
  • Provide an audio file of a description of the video, OR
  • Provide a separate video with an audio description included as the audio track, OR
  • Provide a text transcript describing the video content adjacent to the video, OR
  • Provide a text transcript via a link adjacent to the video
