Expected Result

Regular text, including images of regular text, has a contrast ratio of at least 7:1 with the background.

Actual Result

Regular text does not have a contrast ratio of at least 7:1 with the background.

Solution

Change the text or background color so that the contrast ratio between the text and the background is at least 7:1.

Failing Example

This text fails with a contrast ratio of 3.01:1 with the background.

Passing Examples

This text has passing contrast with the background at 7:1.

This text has very good contrast with the background at 11.44:1.

Not Applicable

  • Large text:
    File under {{ site.data.1403.1403[14030201].name }}.
  • Disabled form controls
  • Purely decorative text
  • Visited link text
  • Logo text
  • Text within an image that contains significant other visual content

Disabilities Affected

Vision impaired and color blind users need sufficient contrast in order to be able to read text.

  • G174: Providing a control with a sufficient contrast ratio that allows users to switch to a presentation that uses sufficient contrast
