Expected Result

Regular-sized placeholder text has a contrast ratio of at least 7:1 with its background.

Actual Result

Regular-sized placeholder text does not have a contrast ratio of at least 7:1 with its background.

Solution

Change the text or background so that the contrast ratio between the placeholder text and the background is at least 7:1.

Failing Example

This placeholder text fails the minimum contrast requirement with a contrast ratio of 2.84:1.

input::placeholder {
    color: #999999;
}

Passing Example

This placeholder text is styled to use the color #595959, which has a contrast ratio of 7:1 with the white background.

input::placeholder {
    color: #595959;
}

Not Applicable

  • Large text:
    File under {{ site.data.1406.1406[14060204].name }}.

Disabilities Affected

  • Visual Impairment: Users often have difficulty reading low contrast text.
  • Color Blindness: Some color deficiencies lower the contrast in the user's vision.
