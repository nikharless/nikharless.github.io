Expected Result

Prerecorded synchronized media which does not have sufficient audio pauses to fit all the important visual information must have an extended audio description of the visual content.

Actual Result

The prerecorded synchronized media content does not have an audio description for the video portion, but it doesn't contain sufficient pauses in the audio to insert descriptions.

Solution

Provide an extended audio description that includes all key visual elements necessary to understand the video content. Pause the video where necessary to fit all the information.

Examples

Extended Audio Description

Why It's Important

  • Blind/vision impaired: People who are blind or vision impaired need to be able to access the same information provided in the video track.
  • Information processing: People who have the ability to see use alternatives to more easily process the visual information.
