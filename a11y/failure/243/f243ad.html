Expected Result

Focus moves to modal dialog when it is opened.

Actual Result

Focus does not move to modal dialog when it is opened.

Solution

Place the focus inside the modal when it is opened. Generally, focus should be placed on the first control inside the modal.

Examples

When this modal opens, focus goes to the "Name" field.

{% include examples/modal.html %}

Why It's Important

Navigating a site or application with only the keyboard will make sense.
