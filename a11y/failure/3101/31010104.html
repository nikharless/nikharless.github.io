Expected Result

The PDF file has a document language defined.

Actual Result

The PDF file does not have a document language defined.

Solution

Adobe Acrobat Pro

  1. Go to Document Properties:
    • Select Menu > Document properties....
    • Windows: Control+D
    • MacOS: Command+D
  2. In the Document Properties dialog, select the Advanced tab.
  3. Under Reading Options, select the appropriate language from the Language drop-down.

Examples

Why It's Important

Assistive technology users get information in the correct language.
