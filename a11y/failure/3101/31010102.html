Expected Result

The web page has a lang attribute that sets the default page language.

Actual Result

The web page has a lang attribute, but the language code is invalid.

Solution

Set the lang value to a valid language code.

Examples

<html lang="es">

Why It's Important

Assistive technology users get information in the correct language.
