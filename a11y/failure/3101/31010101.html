Expected Result

The web page has a lang attribute that sets the default page language.

Actual Result

The web page does not have a lang attribute on the <html> tag.

Solution

Add a lang attribute to the <html> tag and set it to the appropriate language code.

Examples

<html lang="en">

Why It's Important

  • Blind/visually impaired: Screen reader users need the page read to them with the correct pronunciation.
  • Deaf/hard of hearing: Users need the captions to be in the correct language.
  • Cognitively impaired: Users of text-to-speech software need content to be read with the correct pronunciation.
