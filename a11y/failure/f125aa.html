Expected Result

Prerecorded synchronized media must have an audio description of the visual content.

Actual Result

The prerecorded synchronized media content does not have an audio alternative for the video portion.

Solution

Provide an audio description that includes all key visual elements necessary to understand the video content.

Examples

Audio Description Added to Spaces in Audio

Audio Description Integrated into Dialogue

Why It's Important

Visually impaired users, or any users who have difficulty understanding moving content, need an alternate means of receiving the information.
