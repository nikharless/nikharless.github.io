Expected Result

The purpose of the link can be determined from the image alternative text along with its immediate context without having to leave the link.

Actual Result

The purpose of the link cannot be determined without leaving the link.

Solution

  • Add alternative text to the image, describing the link purpose (alt, title, aria-label, or aria-labelledby), OR
  • Add alternative text to the link describing the link purpose (aria-label or aria-labelledby), OR
  • If the image has accompanying visible text, combine the image and the text into a single link and use alt="" on the image.

Examples

Image Alternative Text

<a href="http://github.com">
  <img 
    src="github-mark.png" 
    alt="GitHub home" />
</a>

Link Alternative Text

<a href="#" aria-label="Add to Cart">
  <img src="shopping-cart.png">
</a>

Image and Text in Single Link

Download the User Guide 

<a href="user-guide.pdf">
  Download the User Guide 
  <img 
    src="pdf-icon.png" 
    alt="PDF" />
</a>

Why It's Important

Users with visual and cognitive disabilities can navigate more easily.
