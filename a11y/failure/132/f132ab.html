Expected Result

The screen reader reads the content in a logical order.

Actual Result

The screen reader does not read the content in a logical order.

Solution

Ensure that elements are ordered logically in the source code, so that when they are interpreted by assistive technology they still appear in the correct order.

Example

Why It's Important

Assistive technologies rely on the source code or other programmatically determined order to render the content in the correct sequence.
