Expected Result

A single pointer alternative is available for any functionality which uses a dragging movement.

Actual Result

No single pointer alternative is available for functionality that uses a dragging movement.

Solution

Provide a single pointer alternative for functionality that uses a dragging movement.

Examples

  • A list can be reordered by dragging and dropping the items. Alternatively, activating an item reveals arrow buttons which move the item within the list.
  • A list can be reordered by dragging and dropping the items. Alternatively, text fields next to the list items allow the user to enter numbers to change the order.
  • A Kanban board allows dragging and dropping of cards between columns representing stages of work. Alternatively, cards can be selected and selected cards moved using a drop-down list of columns.

Drag and Drop List

This list can be sorted by dragging and dropping the items. Alternatively, you can tab so an item, press Space to select it, and use the arrow keys to move it.

Why It's Important

Some users cannot use a mouse to drag items.
