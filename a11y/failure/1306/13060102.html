Expected Result

The purpose of user interface components can be programmatically determined.

Actual Result

The purpose of a user interface component cannot be programmatically determined.

Solution

Add the purpose of the user interface component using the following methods:

  • Use native HTML elements (<button>, <input>/<label>, <select>) where possible.
  • Add the purpose of the user interface component using an aria-label, aria-labelledby, or aria-describedby attribute.

Failing Example

Button

The purpose of this button is to open a search bar. This example has no means of communicating the purpose to assistive technology.

<button>🔍</button>

Search field

This example has a visible label that describes its purpose, but it is not programmatically associated with the input.

Search

<b>Search</b>
<input type="text" placeholder="Search...">

Passing Example

Button

The aria-label attribute provides the purpose.

<button aria-label="search">🔍</button>

Search field

<label for="search-field"><b>Search</b></label>
<input id="search-field" type="text" placeholder="Search...">
