Expected Result

The purpose of links can be programmatically determined.

Actual Result

The purpose of a link cannot be programmatically determined

Solution

Add the purpose of the link using the following methods:

  • Use the native HTML <a> element where possible.
  • Add the purpose of the link using an aria-label, aria-labelledby, or aria-describedby attribute.

Failing Example

Navigation link

The purpose of this link is to go to the home page. This example has no means of communicating the purpose to assistive technology.

🏠

<a href="/">🏠</a>

Passing Example

Navigation link

🏠

<a href="/" aria-label="home">🏠</a>
