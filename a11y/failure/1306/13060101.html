Expected Result

The purpose of informational icons can be programmatically determined.

Actual Result

The purpose of an informational icon cannot be programmatically determined.

Solution

  • <img>: Add the purpose of the icon to the alt attribute.
  • <svg>: Add the purpose of the icon to the <title> element within the <svg>.
  • Other image types: Add the purpose of the icon using an aria-label, aria-labelledby, or aria-describedby attribute.

Failing Example

Caution icon

The purpose of this non-interactive icon is to indicate a warning. This example has no means of communicating the purpose to assistive technology.

<i class="fa-solid fa-triangle-exclamation"></i>

Passing Example

Caution icon

This example uses role="img" to identify the element as an image and aria-label to communicate the purpose.

<i class="fa-solid fa-triangle-exclamation" aria-label="caution" role="img"></i>

Not Applicable

  • Decorative icons

Disabilities Affected

  • Cognitive Impairment: Users with language, memory, or executive function issues may implement personalization, such as substituting icons with familiar symbols.
