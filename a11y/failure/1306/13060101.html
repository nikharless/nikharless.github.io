Expected Result

The purpose of user interface components can be programmatically determined.

Actual Result

The purpose of a user interface component cannot be programmatically determined.

Solution

  • Use semantic native HTML elements where possible.
  • Use the ARIA role, aria-label, aria-labelledby, and aria-describedby to specify the purpose.

Failing Example

Because this checkpoint is largely geared toward future functionality, real-world failures that aren't covered by other criteria are rare. Test Level A and Level AA success criteria first and file any failures accordingly.
