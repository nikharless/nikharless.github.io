Expected Result

Each control's accessible name contains the text of its visible label.

Actual Result

The control does not have an accessible name.

Solution

Add an accessible name which includes the visible label text.

Examples

Exact Text

<button>This is the label</button>

Visible Label at the Beginning

<button aria-label="Read more about accessibility">Read More</button>

Why It's Important

Voice control users use the visible labels in their commands.
