WCAG | 1.1.1 Non-text Content | 1.1.1.d Decorative Images (Text Alternative)

Expected Result Decorative images must not have a text alternative
Actual Result The decorative image has a text alternative.
Solution Remove the text alternative from the image using one of the following methods:
  • Remove any text from the alt value in the <img> tag.
  • Remove any title attribute from the <img> tag.
  • Remove any aria-label attribute from the image.
  • Remove any aria-labelledby attribute from the image.
  • Remove any <title> tag from the <svg>.

One might be tempted to give this image a text alternative such as "smiling girl with blonde hair and blue eyes," but this information is not relevant to the content. The image has only been added for visual interest.

Why It's Important Blind and visually impaired users want decorative or redundant images hidden so they do not get overwhelmed with unimportant screen reader chatter.
