Expected Result

Data tables are marked up semantically as tables.

Actual Result

Data tables are not marked up semantically as tables.

Solution

Mark up data tables as semantic tables.
  • Use native HTML tables.
  • If this is not possible, follow the recommended ARIA pattern for tables.

Example

Name Time
John 18 min
Jane 20 min 
<table>
    <tr>
        <th>Name</th>
        <th>Time</th>
    </tr>
    <tr>
        <td>John</td>
        <td>18 min</td>
    </tr>
    <tr>
        <td>Jane</td>
        <td>20 min</td>
    </tr>
</table>

Why It's Important

Table relationships are visible at a glance for sighted users. Assistive technology users need to get this relationship information from the semantic structure and to be able to easily navigate the data.
