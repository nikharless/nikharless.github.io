Expected Result

Only actual lists are semantically marked up as lists.

Actual Result

Content which does not function as a list is marked up as a list.

Solution

Remove list markup from the content. Use CSS to style as desired.

Example

Why It's Important

Assistive technology users depend on semantic information to understand the relationships between pieces of content. If semantic markup is used for presentation effect only, this can be confusing.

References

  • Understanding SC 1.3.1: Info and Relationships
  • F43: Failure of Success Criterion 1.3.1 due to using structural markup in a way that does not represent relationships in the content
  • C22: Using CSS to control visual presentation of text
