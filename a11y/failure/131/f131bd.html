Expected Result

Header cells are marked up with the correct scope attribute.

Actual Result

Header cells are marked up with the scope attribute, but the value is incorrect.

Solution

  • Use scope="col" for column headers.
  • Use scope="colgroup" for colgroup headers.
  • Use scope="row" for row headers.
  • Use scope="rowgroup" for rowgroup headers.

Example

Table with Column and Row Headers

Name Time Attempts
John 18 min 3
Jane 20 min 1 
<table>
    <tr>
        <th scope="col">Name</th>
        <th scope="col">Time</th>
        <th scope="col">Attempts</th>
    </tr>
    <tr>
        <th scope="row">John</th>
        <td>18 min</td>
        <td>3</td>
    </tr>
    <tr>
        <th scope="row">Jane</th>
        <td>20 min</td>
        <td>1</td>
    </tr>
</table>

Table with Column Group and Column Headers

Name Time
Attempt 1 Attempt 2 Attempt 3
John 22 minutes 21 minutes 18 minutes
Jane 20 minutes 18 minutes 20 minutes 
<table>
    <tr>
        <th rowspan="2" scope="col">Name</th>
        <th colspan="3" scope="colgroup">Time</th>
    </tr>
    <tr>
        <th scope="col">Attempt 1</th>
        <th scope="col">Attempt 2</th>
        <th scope="col">Attempt 3</th>
    </tr>
    <tr>
        <td>John</td>
        <td>22 minutes</td>
        <td>21 minutes</td>
        <td>18 minutes</td>
    </tr>
    <tr>
        <td>Jane</td>
        <td>20 minutes</td>
        <td>18 minutes</td>
        <td>20 minutes</td>
    </tr>
</table>

Why It's Important

Table relationships are visible at a glance for sighted users. Assistive technology users need to get this relationship information from the semantic structure and to be able to easily navigate the data.
