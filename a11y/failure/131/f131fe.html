Expected Result

Lists contain only <li>, <script>, and <template> elements.

Actual Result

List contains one or more invalid child elements.

Solution

Enclose all content inside <ul> or <ol> elements within <li> elements.

Example

<ul>
  <li><p>Coffee<br>
    <i>with cream</i></p></li>
  <li><a href="#">Tea</a>
    <ul>
      <li>Black tea</li>
      <li>Green tea</li>
    </ul>
  </li>
  <li><img src="milk.png"> Milk</li>
</ul>

Why It's Important

Screen readers do not correctly read lists when content elements other than <li> are directly below the <ul> or <ol> elements.
