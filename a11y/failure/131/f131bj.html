Expected Result

Table captions are programmatically associated with the table.

Actual Result

The table caption is not programmatically associated with the table.

Solution

  • Use the <caption> tag immediately inside the <table> tag to mark up the caption, OR
  • Use aria-labelledby to associate a caption provided outside of the table, OR
  • Use aria-label to provide a caption for screen reader users only

Example

Table with <caption>

Monthly Savings
Month Savings
January $100 
<table>
    <caption>Monthly Savings</caption>
    <tr>
        <th>Month</th>
        <th>Savings</th>
    </tr>
    <tr>
        <td>January</td>
        <td>$100<&/td>
    </tr>
    ...
</table>

Table with Associated Label

Month Savings
January $100
Monthly Savings
<table aria-labelledby="tableLabel">
        <th>Month</th>
        <th>Savings</th>
    </tr>
    <tr>
        <td>January</td>
        <td>$100<&/td>
    </tr>
    ...
</table>
<div id="tableLabel">Monthly Savings</div>

Why It's Important

Table relationships are visible at a glance for sighted users. Assistive technology users need to get this relationship information from the semantic structure and to be able to easily navigate the data.
