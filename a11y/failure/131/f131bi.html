Expected Result

Table captions are marked up as captions.

Actual Result

The table caption is marked up as a header cell.

Solution

  • Use the <caption> tag immediately inside the <table> tag to mark up the caption.
  • Use CSS to style the caption.

Example

Table with <caption>

Monthly Savings
Month Savings
January $100 
<table>
    <caption>Monthly Savings</caption>
    <tr>
        <th>Month</th>
        <th>Savings</th>
    </tr>
    <tr>
        <td>January</td>
        <td>$100<&/td>
    </tr>
    ...
</table>

Why It's Important

Table relationships are visible at a glance for sighted users. Assistive technology users need to get this relationship information from the semantic structure and to be able to easily navigate the data.
