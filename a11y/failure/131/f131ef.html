Expected Result

Only headings are tagged as headings.

Actual Result

Text that is not a heading is tagged as a heading.

Solution

Adobe Acrobat Pro

  1. In the Tags panel, right-click the incorrectly tagged text or open the Options menu and select Properties....
  2. In the Object Properties dialog, select the correct semantic tag from the Type drop-down.

Example

Adobe Acrobat Pro

Why It's Important

Visual users can see at a glance how information on the page is structured. Assistive technology users rely on semantic markup to understand how content is arranged.
