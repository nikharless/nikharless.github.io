Expected Result

There is a visible indicator showing which element is in focus.

Actual Result

The focus is removed from the element using a script.

Solution

Eliminate any script which removes keyboard focus from a focused element.

Examples

Tab through the links to see the script in action.

Link Focus Removed

Link

<a onFocus="this.blur()" href="#">Link</a>

Why It's Important

Without a focus indicator, sighted keyboard users cannot operate the page. In addition, this script renders the control unusable by a keyboard.
