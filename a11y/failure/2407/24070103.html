Expected Result

The focus indicator is not removed with a script.

Actual Result

The focus is removed from the element using a script.

Solution

Eliminate any script which removes keyboard focus from a focused element.

Examples

Tab through the links to see the script in action.

Bad Example: Link Focus Removed

Link

<a onFocus="this.blur()" href="#">Link</a>

Why It's Important

  • Keyboard users: Without a focus indicator, sighted keyboard users cannot operate the page.
  • Cognitive disorders: Users with attention, short term memory, or executive function limitations benefit from being able to easily find the focus.

References

  • Understanding Success Criterion {{ sc.name }}
  • F55: Failure of Success Criteria 2.1.1, 2.4.7, and 3.2.1 due to using script to remove focus when focus is received
