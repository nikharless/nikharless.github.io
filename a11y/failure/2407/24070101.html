Expected Result

There is a visible indicator showing which element is in focus.

Actual Result

There is no visible focus indicator.

Solution

  • Use CSS to create a visible focus indicator, OR
  • Use the unmodified user-agent focus indicator.

Examples

Tab through the links and controls to see the focus indicators.

Why It's Important

Without a focus indicator, sighted keyboard users cannot operate the page.
