Expected Result

All meaningful images must have a text alternative that conveys the same information as the image.

Actual Result

The meaningful <img> does not have a text alternative.

Solution

Add a text alternative which describes the purpose of the meaningful image using one of the following methods:
  • Add an alt attribute to the <img> tag (this is the preferred method), OR
  • Add an aria-label attribute to the <img> tag, OR
  • Add a title attribute to the <img> tag

Example

PNG Image

<img 
    src="mars-rover.png" 
    alt="An artist's concept of a NASA Mars Exploration Rover on the surface of Mars.">

Disabilities Affected

  • Blindness/Visual Impairment: Users need to have a text alternative for meaningful images so that the images can be interpreted by assistive technology such as screen readers and braille devices.
  • Cognitive Impairment: Users who have difficulty understanding images may benefit from having a text description.
