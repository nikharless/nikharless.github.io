Expected Result

All meaningful images must have a text alternative that conveys the same information as the image.

Actual Result

The meaningful <img> does not have a text alternative.

Solution

Add a text alternative which describes the purpose of the meaningful image using one of the following methods:
  • Add an alt attribute to the <img> tag, OR
  • Add an aria-label attribute to the <img> tag, OR
  • Add a title attribute to the <img> tag

Example

JPG Image

<img 
    src="mars-rover.png" 
    alt="An artist's concept of a NASA Mars Exploration Rover on the surface of Mars.">

Why It's Important

Blind and visually impaired users need to have a text alternative to meaningful images so that the images can be interpreted by assistive technology such as screen readers and braille devices. Without a text alternative, these users would not get the information conveyed by the image.
