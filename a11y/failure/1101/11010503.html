Expected Result

Each CAPTCHA must have at least one alternate method of validation which does not rely on vision.

Actual Result

The CAPTCHA does not have an alternate method of validation.

Solution

  • Consider using an alternative to the CAPTCHA.
  • Add an alternate method of validation, such as audio.

Examples

ReCAPTCHA

The current verson of Google's ReCAPTCHA is a simple checkbox. If it is unable to validate, an image test is presented.

At this point the user can Tab to a button to access an audio version.

ReCAPTCHA demo

Embedded Text

A button for an audio alternative is available for this CAPTCHA.

Why It's Important

Blind and visually impaired users cannot complete image recognition tests. They require an alternate version that doesn't rely on vision.
