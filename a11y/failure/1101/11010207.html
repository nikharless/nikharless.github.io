Expected Result

The text alternative for the meaningful image includes all meaningful text.

Actual Result

The meaningful image has a text alternative, but it does not include all meaningful text in the image.

Solution

Change the text alternative to include all the meaningful text in the image.

Examples

Describing this image as "Scrabble tiles on a table" leaves out the meaningful text that they spell out.

Why It's Important

  • Blind/visually impaired: Users need to have a text alternative for meaningful images so that the images can be interpreted by assistive technology such as screen readers and braille devices.
  • Cognitively impaired: Users who have difficulty understanding images may benefit from having a text description.
