Expected Result

Server-side image maps must have an alternative list of accessible links.

Actual Result

A server-side image map is used, but there is no text alternative for the links.

Solution

  1. Add an <alt> attribute to the <img> tag summarizing the purpose of the image map.
  2. Add an alternate list of text links containing all the same links in the image map.

Disabilities Affected

Server-side image maps send the coordinates of mouse clicks to be calculated server-side, after which a link is returned. There is nothing for keyboard users to focus on, and no way to add a text alternative to the areas of the image map.

  • Blindness/Visual Impairment: Users need to have a text alternative for functional images so that the images can be interpreted by assistive technology such as screen readers and braille devices.
  • Cognitive Impairment: Users who have difficulty understanding images may benefit from having a text description.
