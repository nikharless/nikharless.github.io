Expected Result

If a link or button contains only an image AND the link or button itself doesn't have alternative text, the image must have alternative text which describes the destination or function.

Actual Result

The link or button contains only an image, but the image does not have alternative text and the link or button does not have an accessible name.

Solution

Add a text alternative which describes the purpose of the functional image using one of the following methods:
  • Add an aria-label attribute to the image element's tag, OR
  • Add an aria-labelledby attribute to the image element's tag, set to the id value of the text to use as a label.

Why It's Important

  • Blind/visually impaired: Users need to have a text alternative for functional images so that the images can be interpreted by assistive technology such as screen readers and braille devices.
  • Cognitively impaired: Users who have difficulty understanding images may benefit from having a text description.
