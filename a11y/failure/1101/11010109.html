Expected Result

The text alternative for the functional image includes the informational text on the image.

Actual Result

The functional image has a text alternative, but it does not include all the informational text on the image.

Solution

Change the text alternative to include the informational text on the image. The text alternative should begin with the visible text.

Example

Logo

All the text in this logo should be included in the text alternative.

<a href="https://ourparks.org/">
      <img 
        src="our-parks-logo-dark.svg" 
        alt="Our Parks">
    </a>

Disabilities Affected

  • Blind/visually impaired: Users need to have a text alternative for functional images so that the images can be interpreted by assistive technology such as screen readers and braille devices.
  • Cognitively impaired: Users who have difficulty understanding images may benefit from having a text description.
  • Motion impaired: Users of speech recognition technology need accessible names to match the visual presentation of text so that it's clear which speech command will activate the control.
