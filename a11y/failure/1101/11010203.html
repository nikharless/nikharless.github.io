Expected Result

If an <input> link or button contains only an image, AND the link or button itself doesn't have alternative text, the image must have an alt attribute with alternative text that describes the control's purpose.

Actual Result

An <input> control is labeled with only an image, but the image does not have an alt tag with alternative text, and the control does not have an accessible name.

Solution

Add an alt attribute to the <input> tag.

Failing Example

Submit Button

<input type="image" name="submit" src="submit.png">

Passing Example

Submit Button

<input type="image" name="submit" src="submit.png"
    alt="Submit">

Disabilities Affected

  • Blindness/Visual Impairment: Users need to have a text alternative for functional images so that the images can be interpreted by assistive technology such as screen readers and braille devices.
  • Cognitive Impairment: Users who have difficulty understanding images may benefit from having a text description.
