Expected Result

Redundant images must be hidden from screen readers.

Actual Result

The redundant image is read by screen readers.

Solution

Hide the image using one of the following methods:
  • Add alt="" to the <img> tag.
  • Add aria-hidden="true" to the image element.
  • Implement the image as a CSS background image.

Examples

Image with Caption Text

This image has adjacent text describing its content. Repeating this in the alt text would be redundant, so alt="" is appropriate here.

Alan Bates and Joanna Pettet on Broadway in "Poor Richard"

<img src="bates-pettet.jpg" alt="">

Combined Image and Text Link

Flag images are used in a list of country links simply to illustrate each one. Since the name of the country is already provided in text, the image is redundant. The link and image should be combined within a single link and the image hidden from screen readers.


Norway 
<a href="norway.html">
    <img src="norway-flag.png" alt="">
    Norway
</a>

Button with Image

Because the image is within a <button>, it's best to attach the alternative text to the button rather than the image. This prevents the screen reader from unnecessarily saying "graphic". In this case, the <img> should have a null alt attribute.

<button aria-label="Add to Cart">
    <img 
    src="shopping-cart.png" 
    alt="">
</button>

Why It's Important

  • Blind/visually impaired: Users want decorative images hidden so that the images can be ignored by assistive technology such as screen readers and braille devices.
