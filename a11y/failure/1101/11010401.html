Expected Result

Decorative images using the <img> tag must have a null alt value.

Actual Result

The decorative <img> does not have an alt attribute.

Solution

  • Add alt="" to the <img> element.
  • If the <img> element points to a .svg file and the file has a <title> tag inside it, VoiceOver will read the <title> tag regardless of having a null alt attribute. In this case, add aria-hidden="true" to the <img> element.

Why It's Important

Blind and visually impaired users want decorative or redundant images hidden so they do not get overwhelmed with unimportant screen reader chatter.
