Expected Result

Each video element has a text alternative that summarizes its purpose.

Actual Result

A video element has a text alternative, but it does not adequately summarize the content of the video.

Solution

Change the text alternative to a summary of the content of the video.

Failing Example

HTML <video>

A text alternative like "video player" does not tell the user what the video content is.

<video width="320" height="240" controls aria-label="video player">
        <source src="blue-tit.mp4" type="video/mp4">
        Your browser does not support the video tag.
    </video>

Passing Example

HTML <video>

<video width="320" height="240" controls aria-label="blue tit chirping">
        <source src="blue-tit.mp4" type="video/mp4">
        Your browser does not support the video tag.
    </video>

Disabilities Affected

Blind and visually impaired users need to have a text alternative to embedded video which summarizes the content so they know what it is without having to play it.
