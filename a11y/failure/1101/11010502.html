Expected Result

Each CAPTCHA image must have a text alternative that describes its purpose (but not the visual elements).

Actual Result

The CAPTCHA image has a text alternative, but it doesn't describe its purpose.

Solution

  • Consider using an alternative to the CAPTCHA.
  • Add a text alternative to each challenge image which identifies its purpose but does not give the answer.

Examples

ReCAPTCHA

The current verson of Google's ReCAPTCHA is a simple checkbox. If it is unable to validate, an image test is presented.

At this point the user can Tab directly to a button to access an audio version. The image buttons are not visible to screen readers.

ReCAPTCHA demo

Image Test

This CAPTCHA has a series of images which cannot be described to blind users without invalidating the purpose. In this case, the button containing each image is described to screen readers as “Challenge Image 1”, “Challenge Image 2”, etc.
