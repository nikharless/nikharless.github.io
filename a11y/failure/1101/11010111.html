Expected Result

If a link or button contains both an image and text AND the image adds information not conveyed by the text, the image must have alternate text which conveys the same information as the image.

Actual Result

The link or button contains both an image and text, and the image adds information not conveyed by the text, but the image does not have alternative text.

Solution

Add a text alternative which describes the purpose of the functional image using one of the following methods:
  • Add an alt attribute to the <img> tag, OR
  • Add an aria-label attribute to the image element's tag, OR
  • Add an aria-labelledby attribute to the image element's tag, set to the id value of the text to use as a label, OR
  • Add a title attribute to the <img> tag

Example

Informational <img> within Link

This link's function is to open a PDF file. The image's function within the link is to convey the file type. Since the link already contains text describing its target, the image's text alternative only needs to identify the target as a PDF.

Download the User Guide 

<a href="user-guide.pdf">
    Download the User Guide 
    <img 
    src="pdf-icon.png" 
    alt="PDF" />
</a>

Why It's Important

  • Blind/visually impaired: Users need to have a text alternative for functional images so that the images can be interpreted by assistive technology such as screen readers and braille devices.
  • Cognitively impaired: Users who have difficulty understanding images may benefit from having a text description.
