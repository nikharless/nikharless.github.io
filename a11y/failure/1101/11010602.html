Expected Result

Each video element must have a text alternative which summarizes its purpose.

Actual Result

The video element does not have a text alternative.

Solution

Add a text alternative which summarizes the content of the video:
  • Add an aria-label attribute to the element, OR
  • Add an aria-labelledby attribute to the element, set to the id value of the text to use as a label

Example

HTML <video>

<video width="320" height="240" controls aria-label="blue tit chirping">
    <source src="blue-tit.mp4" type="video/mp4">
    Your browser does not support the video tag.
</video>

Why It's Important

Blind and visually impaired users need to have a text alternative to embedded video so they know what it is without having to play it.

References

  • Understanding SC 1.1.1: Non-text Content
  • G100: Providing a short text alternative which is the accepted name or a descriptive name of the non-text content
  • ARIA6: Using aria-label to provide labels for objects
