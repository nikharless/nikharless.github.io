Expected Result

All meaningful images must have a text alternative that conveys the same information as the image.

Actual Result

The meaningful image does not have a text alternative.

Solution

Add a text alternative which describes the purpose of the meaningful image.

Disabilities Affected

  • Blindness/Visual Impairment: Users need to have a text alternative for meaningful images so that the images can be interpreted by assistive technology such as screen readers and braille devices.
  • Cognitive Impairment: Users who have difficulty understanding images may benefit from having a text description.
