Expected Result

Client-side image map <area> elements must have alternative text describing the area of the image it represents.

Actual Result

The client-side image map <area> element does not have alternative text.

Solution

Add a text alternative which describes the purpose of the image <area> using the following method:
  • Add an alt attribute to each <area> tag in the image map.
  • Set it to a concise text alternative describing the purpose of the link.

Example

Client-Side Image Map

<img src="infographic.png"
    alt="Steps of the Process"
    usemap="#csimap">
<map id="csimap" name="Sample Image Map"> 
    <area shape="rect"
        coords="0,0,168,334"
        href="step1.html"
        alt="Step 1 Details">
        ...
</map>

Disabilities Affected

  • Blind/visually impaired: Users need to have a text alternative for functional images so that the images can be interpreted by assistive technology such as screen readers and braille devices.
  • Cognitively impaired: Users who have difficulty understanding images may benefit from having a text description.
