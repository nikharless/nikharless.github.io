Expected Result

The text alternative for the functional image includes the informational text on the image.

Actual Result

The functional image has a text alternative, but it does not include all the informational text on the image.

Solution

Change the text alternative to include the informational text on the image.

Example

Logo with Motto

All the text in this logo should be included in the text alternative.

<a href="https://www.beavercountypa.gov/">
  <img src="BeaverCountyLogo.png" alt="Beaver County, Pennsylvania. Divided by rivers, united by people">
</a>

Why It's Important

Blind and visually impaired users need to hear the same words that the visual user sees.

References

  • Understanding SC 1.1.1: Non-text Content
  • G94: Providing short text alternative for non-text content that serves the same purpose and presents the same information as the non-text content
