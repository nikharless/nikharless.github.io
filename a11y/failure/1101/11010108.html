Expected Result

The text alternative for the functional image describes the purpose of the control.

Actual Result

The functional image has a text alternative, but it does not describe the purpose of the control.

Solution

Change the text alternative to describe the purpose of the control. The literal description of the image is usually irrelevant.

Failing Example

Passing Example

Search Button

A text alternative such as "magnifying glass" would be inappropriate for this control, because it doesn't describe the function.

<input type="image" src="submit.png" alt="Search">

Disabilities Affected

  • Blindness/Visual Impairment: Users need to have a text alternative for functional images so that the images can be interpreted by assistive technology such as screen readers and braille devices.
  • Cognitive Impairment: Users who have difficulty understanding images may benefit from having a text description.
