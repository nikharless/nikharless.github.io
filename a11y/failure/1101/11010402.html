Expected Result

Decorative images must be hidden from screen readers.

Actual Result

The decorative image is read by screen readers.

Solution

Hide the image using one of the following methods:
  • Add aria-hidden="true" to the image element.
  • Implement the image as a CSS background image.

Examples

ARIA and <svg>

<svg aria-hidden="true" viewBox="0 0 344.20377 256" xmlns="http://www.w3.org/2000/svg">

ARIA and <img> SVG

When a .svg file is the src for an <img> element and it has a <title> element inside it, even if it has an alt="" attribute, VoiceOver will still read it. In this case, you must use aria-hidden="true" to truly hide it.

<img src="handicap.svg" alt="" aria-hidden="true">

CSS background

Lorem ipsum dolor sit amet, consectetur adipiscing elit, sed do eiusmod tempor incididunt ut labore et dolore magna aliqua. Rhoncus aenean vel elit scelerisque mauris.
<style>
    #bg {
        background-image: url("flower.jpg");
        width: 640px;
        height: 424px;
    }
</style>
<div id="bg">
    <div id="divText">Lorem ipsum dolor sit amet, consectetur adipiscing elit, sed do eiusmod tempor incididunt ut labore et dolore magna aliqua. Rhoncus aenean vel elit scelerisque mauris.</div>
</div>

Why It's Important

Blind and visually impaired users want decorative or redundant images hidden so they do not get overwhelmed with unimportant screen reader chatter.
