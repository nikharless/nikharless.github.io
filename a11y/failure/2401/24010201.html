Expected Result

<iframe>s have title attributes to allow screen reader users to use them to navigate the page.

Actual Result

Content is segmented using <iframe>s, but the <iframe>s do not have title attributes.

Solution

Add a title attribute to each <iframe> and set it to a unique value describing the <iframe>'s contents.

Examples

<iframe src="header.html" name="header-iframe" title="Header"></iframe>
<iframe src="footer.html" name="footer-iframe" title="Footer"></iframe>

Why It's Important

Screen reader and keyboard-only users are not able to scroll past blocks of content or go directly to the main content.
