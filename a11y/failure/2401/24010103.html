Expected Result

A skip link is visible to sighted keyboard users.

Actual Result

A skip link is provided, but it is not visible when focused with the keyboard.

Solution

Ensure that the skip link is visible when focused. It may be hidden when not in focus.

Failing Example

The skip link in this example does not become visible when it gains keyboard focus.

Skip navigation
siteLogo
  • Home
  • About
  • Services
  • Contact
  • Log In
This is the main content for the failing example.

There is a skip link above this header. You just can't see it, so unless you're using assistive technology, how can you know it's there?

This is a pretend link, which is where the focus should go when you press Tab after the page scrolls.

Passing Example

The skip link in this example becomes visible when it gains keyboard focus.

Skip navigation
siteLogo
  • Home
  • About
  • Services
  • Contact
  • Log In
This is the main content for the passing example.

There is a skip link above this example header. It becomes visible when it gains keyboard focus.

This is a pretend link, which is where the focus should go when you press Tab after the page scrolls.

Not Applicable

  • Non-skip-links:
    File under {{ site.data.2403.2403[2403].name }}.

Disabilities Affected

  • Blindness: Screen reader users may have to listen to all header content and dozens of links on every page before the main content is reached.
  • Visual Impairment: Screen magnifier users may have to search through a large block of content before the main content is reached.
  • Motor Impairment: Keyboard users and users of assistive technology that uses the keyboard interface may have to navigate through dozens of links before reaching the main content.
  • Cognitive Impairment: Users may benefit when links are grouped into lists.
