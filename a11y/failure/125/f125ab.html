Expected Result

Prerecorded synchronized media must have an audio description of all important visual content.

Actual Result

The prerecorded synchronized media content has an audio description, but it is not accurate.

Solution

Provide an audio description that includes all key visual elements necessary to understand the video content.

Examples

Audio Description

Why It's Important

Visually impaired users, or any users who have difficulty understanding moving content, need an alternate means of receiving the information.
