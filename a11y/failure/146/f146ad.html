Expected Result

Regular size placeholder text has a contrast ratio of at least 7:1 with its background.

Actual Result

Regular size placeholder text does not have a contrast ratio of at least 7:1 with its background.

Solution

Change the text or background so that the contrast ratio between the placeholder text and the background is at least 7:1.

Example

This placeholder text is styled to use the color #595959, which has a contrast ratio of 7:1 with the white background.

input::placeholder {
  color: #595959;
}

Why It's Important

Vision impaired and color blind users need sufficient contrast in order to be able to read text.
