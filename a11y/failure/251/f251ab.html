Expected Result

All functionality that can be operated with a pointer can be operated with single-pointer actions.

Actual Result

Functionality requires a path-based gesture.

Solution

Provide single-point controls that achieve the same result as the path-based gesture.

Examples

  • A carousel uses swipe gestures to change slides, but it also has buttons which perform the same function.
  • A slider uses path-based dragging to change the value, but it also has buttons which perform the same function.

Why It's Important

Not everyone can perform complex gestures.
