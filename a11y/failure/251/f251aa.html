Expected Result

All functionality that can be operated with a pointer can be operated with single-pointer actions.

Actual Result

Functionality requires a multi-point gesture.

Solution

Provide single-point controls that achieve the same result as the multi-point gesture.

Examples

A map uses pinch gestures to zoom in and out, but it also has buttons which perform the same function.

Why It's Important

Not everyone can perform multi-touch gestures.
