Expected Result

The <head> element of the web page has a <title> element which contains a descriptive title for the web page.

Actual Result

The web page has a title, but it does not describe the content of the page.

Solution

Change the title so that it is descriptive of the contents or purpose of the page.

Examples

<html>
  <head>
    <title>
      Understanding Success Criterion 2.4.2: Page Titled
    </title>
  </head>
  ...
&lgt;/html>

Why It's Important

Screen reader users, as well as people with cognitive, memory, or reading disabilities benefit from the ability to identify content by its title.
