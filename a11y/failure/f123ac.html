WCAG | 1.2.3 Audio Description or Media Alternative (Prerecorded) | 1.2.3.a Text or Audio Description (Prerecorded Video - with Dialogue)

Expected Result Prerecorded synchronized media must have a text or audio description of the visual content.
Actual Result The prerecorded synchronized media content has a text transcript for the video portion, but it does not include all important visual elements.
Solution Provide a text transcript that includes all key visual elements necessary to understand the video content.
Example

Text Transcript

This transcript does include the text on the cartoon images, which is positive. However, it should also include a brief description of the cartoons themselves to give context to the text.

TED: Liana Finck: Life is Hard. Art Helps.
Why It's Important Visually impaired users, or any users who have difficulty understanding moving content, need an alternate means of receiving the information.
