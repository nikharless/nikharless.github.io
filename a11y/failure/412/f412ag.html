Expected Result

If a control functions as a button, it has the role of "button".

Actual Result

A button does not have the role of "button".

Solution

Assign the role of "button" to the control, using one of the following methods:

  • Use the <button> element to create the button, OR
  • Use the input type="button" element to create the button, OR
  • Use role="button" on the element.

Examples

Button

<button>Click Here</button>

Input type "button"

<input type="button" value="Click Here">

Role of "button"

Click Here
<div role="button" tabindex="0" onclick="doThis();">Click Here</div>

Not Applicable

The following items are not applicable to this failure:

  • Links:
    <a href="#" style="border: 1px solid black; padding: 10px;">
    Elements which function as links but are styled to look like buttons
  • Assigned role
    <button role="link">Click Here</button>
    Elements which use <button> or <input type="button"> and have a different role assigned using the role attribute

Why It's Important

Assistive technology only works well when code is done properly.
