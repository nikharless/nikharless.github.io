Expected Result

Custom user interface component is compatible with assistive technology.

Actual Result

Custom user interface component is not compatible with assistive technology.

Solution

Ensure that custom user interface works with screen readers and other assistive technology.

Examples

Not Applicable

The following items are not applicable to this failure:

  • Native HTML components

Why It's Important

Assistive technology only works well when code is done properly.
