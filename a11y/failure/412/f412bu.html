Expected Result

Elements contain any required child roles.

Actual Result

An element requires a particular child role, but the child role is missing.

Solution

Ensure that any required child roles are present.
  • role="listbox" must contain one or more role="option" elements
  • role="tablist" must contain one or more role="tab" elements

Examples

Listbox

{% include examples/listbox-multi-select.html %} 
<ul role="listbox" aria-multiselectable="true">
    <li role="option" aria-selected="false">Leather seats</li>
    <li role="option" aria-selected="false">Front seat warmers></li>
</ul>
