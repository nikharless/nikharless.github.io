Expected Result

If a control functions as a checkbox, it has a checked state.

Actual Result

A checkbox does not have a checked state.

Solution

  • Use the native <input type="checkbox"> mark-up to create the checkbox, OR
  • Use the aria-checked attribute on the custom checkbox element.

Examples

Native HTML

<input type="checkbox"> Soup</label>
<input type="checkbox" checked> Salad</label>

Custom Checkbox

Subscribe to the mailing list 
<div role="checkbox" tabindex="0" aria-checked="false"> Subscribe to the mailing list
