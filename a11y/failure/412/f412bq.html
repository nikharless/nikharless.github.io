Expected Result

Tooltips have all appropriate roles and attributes.

Actual Result

A tooltip does not have all the appropriate roles or attributes.

Solution

Tooltip

  • Use the "tooltip" role on the popup element.
  • Use the aria-describedby attribute on the tooltip button and point it to the id of the tooltip.
