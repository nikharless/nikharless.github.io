Expected Result

If a control functions as a combobox, it has the role of "combobox". A combobox is an input which has an associated pop-up from which the user can select a value (i.e., listbox, grid, tree, or dialog).

Actual Result

A control functions as a combobox, but it does not have the role of "combobox".

Solution

Assign the correct role to the control, using one of the following methods:

  • Use the <select> element to create the combobox, OR
  • Use the role="combobox" attribute on the element.

Examples

HTML <select> tag

<select>
    <option value="red">Red</option>
    ...
</select>

Custom combobox

{% include examples/select-only.html %} 
<div role="combobox" aria-controls="listbox1" aria-haspopup="listbox" aria-expanded="false" tabindex="0">

Not Applicable

Why It's Important

Assistive technology only works well when code is done properly.
