Expected Result

Tab panels have all appropriate roles and attributes.

Actual Result

A tab panel does not have all the appropriate roles or attributes.

Solution

Tab List

  • Use the "tablist" role on the element that contains a set of tabs.
  • Use the "tab" role on each tab control element.
  • Use
  • aria-selected="true"
    • on the currently selected tab control.
  • Use
  • aria-selected="false"
    • on all non-selected tab controls.
  • Set
  • tab-index="-1"
    • on all non-selected tab controls.
  • Use the
  • aria-controls
    • attribute on each tab control and point it to the
  • id
    • of the associated tab panel element.
  • Use the "tabpanel" role on each container for tab content.
  • Use the
  • aria-labelledby
    • attribute on each tab panel and point it to the
  • id
    • of the associated tab control element.
