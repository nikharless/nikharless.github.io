Expected Result

Disabled controls have a disabled state.

Actual Result

Disabled control does not have a disabled state.

Solution

  • Use the disabled attribute on the control to identify the control as disabled, remove the functionality, and make it visually appear disabled, OR
  • Use the aria-disabled="true" attribute on the control to identify the control as disabled without changing the functionality or appearance.

Examples

HTML

<button disabled>Save</button>

ARIA

<button aria-disabled="true">Save</button>

Not Applicable

The following items are not applicable to this failure:

  • Hidden with CSS:
    <button style="display:none;">
    Does not appear in accessibility tree
  • Hidden with ARIA:
    <button aria-hidden="true">
    Does not appear in accessibility tree

Why It's Important

Assistive technology only works well when code is done properly.
