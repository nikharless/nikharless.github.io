Expected Result

Elements do not contain any invalid child elements or roles.

Actual Result

An element contains an invalid child element or role.

Solution

HTML

Ensure that any child elements are valid.

  • <address> must not contain heading or sectioning content (<h1>, <nav>, <header>, etc.).
  • <audio> must only contain <track> or <source> elements.
  • <colgroup> (without a span attribute) must only contain <col> elements.
  • <dl> must only contain <dt>, <dd>, <script>, or <template> elements.
  • <dt> must not contain heading or sectioning content (<h1>, <nav>, <header>, etc.).
  • <form> must not contain nested <form> elements.
  • <header> must not contain nested <header> or <footer> elements.
  • <html> must only contain one <head> and one <body>.
  • <label> must not contain nested <label> elements.
  • <menu> must only contain <li>, <script>, or <template> elements.
  • <meter> must not contain nested <meter> elements.
  • <ol> must only contain <li>, <script>, or <template> elements.
  • <optgroup> must only contain <option> or a single <legend> elements.
  • <progress> must not contain nested <progress> elements.
  • <table> must only contain the following, in order: <caption>, <colgroup>, <thead>, <tbody> or <tr> (required), <tfoot>.
  • <tbody> must only contain <tr> elements.
  • <tfoot> must only contain <tr> elements.
  • <th> must not contain heading or sectioning content (<h1>, <nav>, <header>, etc.).
  • <thead> must only contain <tr> elements.
  • <tr> must only contain <td>, <th>, <script>, or <template> elements.
  • <ul> must only contain <li>, <script>, or <template> elements.
  • <video> must only contain <track> or <source> elements.

The following elements must not contain any interactive content. Links must not contain buttons, etc.

  • <a>
  • <button>

The following elements are void elements (they do not have end tags) and must not contain any content.

  • <area>
  • <br>
  • <col>
  • <colgroup span="[value]">
  • <embed>
  • <hr>
  • <iframe> (has end tag)
  • <img>
  • <input>
  • <link>
  • <meta>
  • <source>
  • <track>

ARIA

Ensure that any child roles are valid.

  • feed must contain an article.
  • grid must contain a row or rowgroup.
  • list must contain a listitem.
  • listbox must contain a group or option.
  • menu must contain a group, menuitem, menuitemcheckbox, or menuitemradio.
  • menubar must contain a group, menuitem, menuitemcheckbox, or menuitemradio.
  • radiogroup must contain a radio.
  • row must contain a cell, columnheader, gridcell, or rowheader.
  • rowgroup must contain a row.
  • table must contain a row or rowgroup.
  • tablist must contain a tab.
  • tree must contain a group or treeitem.
  • treegrid must contain a row or rowgroup.
