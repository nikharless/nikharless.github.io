Expected Result

Grids have all appropriate roles and attributes.

Actual Result

A grid does not have all the appropriate roles or attributes.

Solution

  • Use the "grid" role on the table element.
  • If rows or columns can be hidden or if all are not displayed at the same time:
    • Use aria-rowcount on the table to set the total number of rows in the table.
    • Use aria-colcount on the table to set the total number of columns in the table.
    • Use aria-rowindex on each row (<tr>) to set the row number.
    • Use aria-colindex on each column (<th> or <td>) to set the columnn number.
  • Use aria-sort to set the sort order of a column.
    • Set to "ascending" if the column is sorted in ascending order.
    • Set to "descending" if the column is sorted in descending order.
    • Set to "none" if the column is not sorted.
