Expected Result

If a control functions as a link, it has the role of "link". The function of a link is to navigate to another location (i.e., a web page, document, or external app).

Actual Result

A link does not have the role of "link".

Solution

Assign the correct role to the control, using one of the following methods:

  • Use the <a href=""> element to create the link, OR
  • Use the role="link" attribute on the element.

Failing Examples

Emulated link

NVDA: "This is an emulated link without a role."

This is an emulated link without a role.

<p>This is an <span onclick="location.href='webpage.html'">emulated link</span> without a role.</p>

Passing Examples

HTML anchor tag

NVDA: "HTML <a> Tag link"

HTML <a> Tag 
<a href="https://www.w3schools.com/tags/tag_a.asp"><HTML <a> Tag</a>

Role of "link"

NVDA: "Web Accessibility Initiative (WAI) link"

<button role="link" onclick="openLink(event)">Web Accessibility Initiative (WAI)</button>

Disabilities Affected

  • Blindness/Vision Impairment: Programmatic roles enable compatibility with assistive technology such as screen readers and screen magnifiers.
  • Motor Impairment: Programmatic roles enable compatibility with assistive technology such as speech recognition software.
