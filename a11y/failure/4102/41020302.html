Expected Result

If a control functions as a disclosure button, it communicates the expanded/collapsed state to assistive technology.

Actual Result

A disclosure button does not communicate the expanded/collapsed state to assistive technology.

Solution

  • Use the native HTML <details> and <summary> elements to create the disclosure button, OR
  • Use the aria-expanded attribute on a custom disclosure button. Set it to "true" when expanded and to "false" when collapsed.

Failing Example

Custom Disclosure Button

This example does not communicate the expanded/collapsed state to assistive technology.

NVDA: ""

  • Shave his belly with a rusty razor
  • Put him in a long boat until he's sober
  • Stick him in a scupper with a hosepipe bottom
  • Put him in the bed with the captain's daughter

...early in the morning.

See Drunken Sailor Guidelines for more information.

Passing Examples

Native HTML

This example uses native HTML <details> and <summary> elements, which automatically announce the expanded/collapsed state.

NVDA: "What will we do with a drunken sailor? button, collapsed"

What will we do with a drunken sailor?
  • Shave his belly with a rusty razor
  • Put him in a long boat until he's sober
  • Stick him in a scupper with a hosepipe bottom
  • Put him in the bed with the captain's daughter

...early in the morning.

ARIA Button

This FAQ-style disclosure button announces the expanded/collapsed state on focus and on activation.

NVDA: "What will we do with a drunken sailor? button, collapsed"

{% include examples/disclosure.html %} 
<button type="button" aria-expanded="false" aria-controls="faq1_desc">

ARIA Menu

This menu-style disclosure button announces the expanded/collapsed state on focus and on activation.

NVDA: "About, button, collapsed"

{% include examples/disclosure-nav.html %} 
<button type="button" aria-expanded="false" aria-controls="id_about_menu">

Not Applicable

Disabilities Affected

  • Blindness/Vision Impairment: Properties enable compatibility with assistive technology such as screen readers and screen magnifiers.
  • Motor Impairment: Properties enable compatibility with assistive technology such as speech recognition software.
