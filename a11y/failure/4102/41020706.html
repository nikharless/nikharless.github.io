Expected Result

If a status message or other ARIA Live Region is used to alert the user of a content change, it provides sufficient description of the content change.

Actual Result

A status message or other ARIA Live Region is used to alert the user of a content change, but it does not provide sufficient description of the content change.

Solution

Ensure that the status message or other ARIA Live Region provides sufficient description of the content change.
