Expected Result

State changes are communicated to assistive technology.

Actual Result

A state change is not communicated to assistive technology.

Solution

  • Use native HTML controls when possible, to automatically communicate state changes, OR
  • Update the value of the attribute to match the current state of the control, AND
  • Ensure state changes are spoken by the screen reader.

Failing Example

Passing Example

Not Applicable

The following items are not applicable to this failure:

  • Hidden with CSS:
    <input type="textbox" style="display:none;">
    Does not appear in accessibility tree
  • Hidden with ARIA:
    <input type="textbox" aria-hidden="true">
    Does not appear in accessibility tree

Disabilities Affected

  • Blindness/Vision Impairment: Properties enable compatibility with assistive technology such as screen readers and screen magnifiers.
  • Motor Impairment: Properties enable compatibility with assistive technology such as speech recognition software.
