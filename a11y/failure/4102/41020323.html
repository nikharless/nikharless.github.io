Expected Result

Tree items communicate the expanded state to assistive technology.

Actual Result

A tree item does not communicate the expanded state to assistive technology.

Solution

  • Use the aria-expanded attribute on each expandable treeitem.
  • Set it to "true" if expanded or "false" if collapsed.
