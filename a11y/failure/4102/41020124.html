Expected Result

Read-only form fields have a read-only property.

Actual Result

A form control that is meant to be read-only does not have a read-only property.

Solution

  • For native HTML controls, use the readonly attribute on the control.
  • For custom controls, use the aria-readonly="true" attribute on the control.

Examples

HTML

<input type="text" value="12345" readonly>

ARIA

ID:
12345
<div role="textbox" contenteditable aria-readonly="true">12345</div>

Not Applicable

The following items are not applicable to this failure:

  • Hidden with CSS:
    <input type="textbox" style="display:none;">
    Does not appear in accessibility tree
  • Hidden with ARIA:
    <input type="textbox" aria-hidden="true">
    Does not appear in accessibility tree

Why It's Important

Assistive technology only works well when code is done properly.
