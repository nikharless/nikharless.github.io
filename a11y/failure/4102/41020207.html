Expected Result

Alert dialogs have the role of "alertdialog".

Actual Result

An alert dialog is missing the alertdialog role.

Solution

Assign role="alertdialog" to the alert dialog.

Failing Example

  • NVDA: ""

Bad Modal

This is an alert dialog!

<div aria-modal="true">

Passing Examples

Alert Dialog

This example uses role="alertdialog" to communicate the role to assistive technology.

  • NVDA (when dialog is opened): "Good Modal, dialog, This is an alert dialog!"

Good Modal

This is an alert dialog!

<div role="alertdialog" aria-modal="true">

Not Applicable

The following items are not applicable to this failure:

  • Hidden:
    <div style="display:none;">...</div>
    Does not appear in accessibility tree

Disabilities Affected

  • Blindness/Vision Impairment: Programmatic roles enable compatibility with assistive technology such as screen readers and screen magnifiers.
  • Motor Impairment: Programmatic roles enable compatibility with assistive technology such as speech recognition software.
