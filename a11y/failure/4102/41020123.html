Expected Result

Required form fields have a required property.

Actual Result

Required control does not have a required property.

Solution

  • For native HTML controls, use the required attribute on the control.
  • For custom controls, use the aria-required="true" attribute on the control.

Examples

HTML

<input type="email" required>

ARIA

Email* 
<div role="textbox" contenteditable aria-required="true">

Not Applicable

The following items are not applicable to this failure:

  • Hidden with CSS:
    <input type="textbox" style="display:none;">
    Does not appear in accessibility tree
  • Hidden with ARIA:
    <input type="textbox" aria-hidden="true">
    Does not appear in accessibility tree

Why It's Important

Assistive technology only works well when code is done properly.
