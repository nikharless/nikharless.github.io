Expected Result

If a control functions as a checkbox, it communicates the checked state to assistive technology.

Actual Result

A checkbox does not communicate the checked state to assistive technology.

Solution

  • Use the native HTML <input type="checkbox"> mark-up to create the checkbox, OR
  • Use the aria-checked attribute on the custom checkbox element.

Failing Example

Custom Checkbox

This custom checkbox does not communicate the checked state to assistive technology.

  • NVDA (on focus, regardless of state): "Subscribe to the mailing list, checkbox, not checked"
  • NVDA (on state change): Nothing spoken
Subscribe to the mailing list 
<div role="checkbox" tabindex="0"> Subscribe to the mailing list
    ...
</div>

Passing Examples

Native HTML

This example uses native HTML, which automatically communicates the checked state to assistive technology.

  • NVDA (on focus when not checked): "Soup, checkbox, not checked"
  • NVDA (on focus when checked): "Salad, checkbox, checked"
  • NVDA (on state change): "checked" or "not checked"
<input type="checkbox"> Soup</label>
<input type="checkbox" checked> Salad</label>

Custom Checkbox

This custom checkbox uses aria-checked to communicate the checked state to assistive technology.

  • NVDA (on focus when not checked): "Subscribe to the mailing list, checkbox, not checked"
  • NVDA (on focus when checked): "Subscribe to the mailing list, checkbox, checked"
  • NVDA (on state change): "checked" or "not checked"
Subscribe to the mailing list 
<div role="checkbox" tabindex="0" aria-checked="false"> Subscribe to the mailing list
    ...
</div>

Not Applicable

The following items are not applicable to this failure:

  • Hidden:
    <div style="display:none;">...</div>
    Does not appear in accessibility tree

Disabilities Affected

  • Blindness/Vision Impairment: Properties enable compatibility with assistive technology such as screen readers and screen magnifiers.
  • Motor Impairment: Properties enable compatibility with assistive technology such as speech recognition software.
