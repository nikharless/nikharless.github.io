Expected Result

If a focus change is used to alert the user of a content change, it provides sufficient description of the content change.

Actual Result

A focus change is used to alert the user of a content change, but it does not provide sufficient description of the content change.

Solution

Ensure that the focus change provides sufficient description of the content change.
