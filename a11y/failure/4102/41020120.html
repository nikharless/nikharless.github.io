Expected Result

If an option in a combobox (select) that uses aria-activedescendent is selected, it has the selected state.

Actual Result

A combobox option does not have a selected state.

Solution

  • Use aria-selected="true" on the selected option.

Examples

{% include examples/select-only.html %} 
<div role="option" id="combo1-1" class="combo-option option-current" aria-selected="true">English</div>

Not Applicable

The following items are not applicable to this failure:

  • Hidden:
    <div style="display:none;">...</div>
    Does not appear in accessibility tree

Why It's Important

Assistive technology only works well when code is done properly.
