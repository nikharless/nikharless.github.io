Expected Result

Frames have a title attribute which describes the content of the frame.

Actual Result

A frame does not have a title.

Solution

Add a title attribute to the <frame> and set it to text which describes the content of the frame.

Failing Examples

No title attribute

The frame in this example has no title attribute to provide an accessible name.

<frameset>
    <frame src="resources.html"></frame>
    ...
</frameset>

Insufficient title

The frame in this example has a title, but it does not describe the content of the frame.

<frameset>
    <frame title="one" src="resources.html"></frame>
    ...
</frameset>

Passing Example

title attribute

This example has a title attribute that sufficiently describes the content of the frame.

<frameset>
    <frame title="Resources" src="resources.html"></frame>
    ...
</frameset>

Not Applicable

The following items are not applicable to this failure:

  • No frame elements:
    No frames on the page

Disabilities Affected

  • Blindness/Vision Impairment: Programmatic names enable compatibility with assistive technology such as screen readers and screen magnifiers.
  • Motor Impairment: Programmatic names enable compatibility with assistive technology such as speech recognition software.
