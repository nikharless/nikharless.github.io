Expected Result

If a control functions as a link, it has the role of "link". The function of a link is to navigate to another location (i.e., a web page, document, or external app).

Actual Result

A link does not have an href attribute, and therefore does not have the role of "link".

Solution

Add an href attribute to the <a> tag, even if the link is scripted. This also provides a fallback for users who do not have JavaScript enabled.

Examples

Redundant href value

HTML <a> Tag 
<a href="https://www.w3schools.com/tags/tag_a.asp" onclick="goHere(event);"><HTML <a> Tag</a>

Not Applicable

The following items are not applicable to this failure:

  • Links with href attribute:
    <a href="https://www.w3schools.com/tags/tag_a.asp" role="button"><HTML <a> Tag</a>
    File under 4.1.2.1.9 Link has an incorrect role
  • Hidden:
    <a style="display:none;">Click here</a>
    Does not appear in accessibility tree

Why It's Important

Assistive technology only works well when code is done properly.
