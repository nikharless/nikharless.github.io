Expected Result

If a control functions as a switch, it has a checked state.

Actual Result

A switch does not have a checked state.

Solution

  • Use the aria-checked attribute on the switch element.

Examples

Not Applicable

The following items are not applicable to this failure:

  • Hidden:
    <div style="display:none;">...</div>
    Does not appear in accessibility tree

Why It's Important

Assistive technology only works well when code is done properly.
