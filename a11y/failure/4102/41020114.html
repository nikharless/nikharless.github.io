Expected Result

If a control functions as a disclosure button, it has an "expanded" state.

Actual Result

A disclosure button does not have an "expanded" state.

Solution

Use the aria-expanded attribute on the disclosure button. Set it to "true" when expanded and to "false" when collapsed.

Examples

Button

{% include examples/disclosure.html %} 
<button type="button" aria-expanded="false" aria-controls="faq1_desc">

Menu

{% include examples/disclosure-nav.html %} 
<button type="button" aria-expanded="false" aria-controls="id_about_menu">

Not Applicable

The following items are not applicable to this failure:

  • Toggle button
    <a tabindex="0" role="button" id="toggle" aria-pressed="false">Mute</a>
    File under 4.1.2.1.13 Toggle button is missing the pressed state.
  • Hidden:
    <div style="display:none;">...</div>
    Does not appear in accessibility tree

Why It's Important

Assistive technology only works well when code is done properly.
