Expected Result

Meters have the role of "meter".

Actual Result

A meter is missing the meter role.

Solution

Assign role="meter" to the meter element.

Failing Example

Gauge

This example does not assign a role to the meter.

  • NVDA: "Central Processing Unit (CPU) Usage"

This meter changes every 5 seconds. Use the pause button to stop changes.

Central Processing Unit (CPU) Usage

<div class="gauge" aria-labelledby="gauge-label">

Passing Examples

at 90/100

Custom Meter
