Expected Result

Comboboxes have all appropriate roles and attributes.

Actual Result

A dialog does not have all the appropriate roles or attributes.

Solution

  • Use the "dialog" role on the dialog element.
  • Use aria-modal="true" on the dialog element to inform assistive technologies that the background content is unavailable.
  • Use aria-labelledby to provide an accessible name by pointing to the visible title of the dialog.
  • If there is additional content that describes the purpose of the dialog, use aria-describedby to point to it.
