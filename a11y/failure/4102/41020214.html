Expected Result

Sliders have all appropriate roles and attributes.

Actual Result

A slider does not have all the appropriate roles or attributes.

Solution

Meter

  • Use the "slider" role on the slider element.
  • Use aria-label or another method to give the slider an accessible name.
  • Use aria-valuenow on the meter element to represent the current value of the slider.
  • Use aria-valuemin on the meter element to represent the minimum value of the slider.
  • Use aria-valuemax on the meter element to represent the maximum value of the slider.
  • Optionally, use aria-valuetext on the slider element if the default number format does not make sense in the slider's context.
