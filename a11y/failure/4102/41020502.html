Expected Result

Changes to form instructions, controls, or content are announced to assistive technology users. Note: Content added after the current focus does not need to be announced.

Actual Result

Changes to form instructions, controls, or content are not announced to assistive technology users.

Solution

Ensure that changes to form instructions, controls, or content that occur before the current focus are announced to assistive technology users.

Examples

  • The form element that triggers the change has accessible instructions notifying the user of the change ahead of time.
  • The updated content is announced with a modal dialog.
  • The focus is moved to the updated content.
  • The updated content is contained within an ARIA Live Region.

Not Applicable

The following items are not applicable to this failure:

  • Automatic updates to content
    File under 4.1.2.3.1 Automatically updated content is not announced.
  • Non-form user-initiated updates to content
    File under 4.1.2.3.3 User initiated change of content is not announced.

Disabilities Affected

  • Blind/vision impaired: Screen reader users need to be alerted of changes in page content.
