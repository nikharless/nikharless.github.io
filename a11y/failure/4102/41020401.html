Expected Result

Any values associated with a component are correct.

Actual Result

An attribute value is incorrect.

Solution

  • Ensure the correct value is set on the attribute.

Failing Example

aria-haspopup

This example opens a dialog but aria-haspopup is incorrectly set to "menu".

  • NVDA: ""
<button aria-haspopup="menu">Settings</button>

Passing Examples

aria-haspopup

This example opens a dialog and aria-haspopup is correctly set to "dialog".

  • NVDA: ""
<button aria-haspopup="dialog">Settings</button>
