Expected Result

Links have accessible names.

Actual Result

A link does not have an accessible name.

Solution

HTML

Provide an accessible name for the link, using one of the following methods:

  • Use text between the start and end tags of an <a> element as the accessible name, OR
  • Use text between the start and end tags of a custom link as the accessible name, OR
  • Use text between the start and end tags of a <button> element with role="link" as the accessible name, OR
  • Use the aria-label attribute on the link and set it to the accessible name, OR
  • Use the title attribute on the link and set it to the accessible name, OR
  • Use the aria-labelledby attribute on the link and point it to the id of a text element with the accessible name, OR
  • Use the alt attribute on an <area> tag and set it to the accessible name.

Examples

Link content text

Web Accessibility Initiative (WAI) 
<a href="https://www.w3.org/WAI/">Web Accessibility Initiative (WAI)</a>

Custom link text

Web Accessibility Initiative (WAI)
<div role="link" tabindex="0" onclick="openLink(event)" onkeyup="openLink(event)">Web Accessibility Initiative (WAI)</div>

Button as link

<button role="link" onclick="openLink(event)">Web Accessibility Initiative (WAI)</button>

aria-label

<a href="https://github.com/" aria-label="github"><img src="github.png" alt=""></a>

Title attribute

<a href="https://github.com/" title="github"><img src="{{ baseurl }}/assets/images/github.png" alt=""></a>

Image map area element

<map name="imagemap">
    <area shape="rect" coords="0,0,30,100" href="https://www.w3.org/WAI/" alt="Web Accessibility Initiative (WAI)">
</map>

Not Applicable

The following items are not applicable to this failure:

  • Link as button:
    <a href="https://www.w3.org/WAI/" role="button">Web Accessibility Initiative (WAI)</a>
    File under 4.1.2.1.1 Button does not have an accessible name.
  • Hidden with display property:
    <a href="https://www.w3.org/WAI/" style="display: none;">Web Accessibility Initiative (WAI)</a>
    Does not appear in accessibility tree
  • Hidden with visibility property:
    <a href="https://www.w3.org/WAI/" style="visibility: hidden;">Web Accessibility Initiative (WAI)</a>
    Does not appear in accessibility tree
  • Hidden with ARIA:
    <a href="https://www.w3.org/WAI/" aria-hidden="true">Web Accessibility Initiative (WAI)</a>
    Does not appear in accessibility tree
  • Anchor element without href attribute:
    <a>
    Not a link
  • Area element that is not a link:
    <area shape="rect" coords="0,0,30,100>
    Not a link

Why It's Important

Assistive technology only works well when code is done properly.
