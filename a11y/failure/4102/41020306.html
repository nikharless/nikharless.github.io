Expected Result

If a control functions as a switch, it communicates the checked state to assistive technology.

Actual Result

A switch does not communicate the checked state to assistive technology.

Solution

  • Use the aria-checked attribute on the switch element.

Failing Example

Custom Switch

Disabilities Affected

  • Blindness/Vision Impairment: Properties enable compatibility with assistive technology such as screen readers and screen magnifiers.
  • Motor Impairment: Properties enable compatibility with assistive technology such as speech recognition software.
