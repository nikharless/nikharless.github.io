Expected Result

If a control opens a menu popup, it communicates this property to assistive technology.

Actual Result

A menu button does not communicate to assistive technology that it opens a menu.

Solution

Use the aria-haspopup attribute on the menu button.

Failing Example

Custom Menu Button

This example does not communicate to assistive technology that it opens a menu.

NVDA: "WAI-ARIA Quick Links, button, collapsed"

  • W3C Home Page
  • W3C Web Accessibility Initiative
  • Accessible Rich Internet Application Specification
  • WAI-ARIA Authoring Practices
  • WAI-ARIA Implementation Guide
  • Accessible Name and Description
<button type="button" id="menubutton" aria-controls="menu2" aria-expanded="false">

Passing Example

Custom Menu Button

This example uses aria-haspopup="menu" to communicate to assistive technology that it opens a menu.

NVDA: "WAI-ARIA Quick Links, menu button, collapsed, subMenu"

{% include examples/menu-button-links.html %} 
<button type="button" id="menubutton" aria-haspopup="menu" aria-controls="menu2" aria-expanded="false">
