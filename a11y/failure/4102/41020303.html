Expected Result

Changes of content initiated by links are announced to assistive technology users.

Actual Result

Changes of content initiated by links are not announced to assistive technology users.

Solution

Ensure that changes to content initiated by clicking links are announced to assistive technology users.

Examples

  • The link that triggers the change has accessible instructions notifying the user of the change ahead of time.
  • The updated content is announced with a modal dialog.
  • The focus is moved to the updated content, and the target provides sufficient information.
  • The updated content is contained within an ARIA Live Region.

Not Applicable

The following items are not applicable to this failure:

  • Links that reload the current page.
  • Links that navigate to a different page.
  • Automatic updates to content

  • Form elements that change in response to value changes
    File under {{ site.data.4102.4102[41020302].name }}.

Disabilities Affected

  • Blind/vision impaired: Screen reader users need to be alerted of changes in page content.
