Expected Result

If a control functions as a menu button, it has an aria-haspopup attribute.

Actual Result

A menu button does not have the aria-haspopup attribute.

Solution

Use the aria-haspopup attribute on the menu button.

Examples

Menu

{% include examples/menu-button-links.html %} 
<button type="button" id="menubutton" aria-haspopup="true" aria-controls="menu2" aria-expanded="false">

Not Applicable

The following items are not applicable to this failure:

  • Toggle button
    <a tabindex="0" role="button" id="toggle" aria-pressed="false">Mute</a>
    File under 4.1.2.1.13 Toggle button is missing the pressed state.
  • Disclosure button
    <<button type="button" aria-expanded="false" aria-controls="faq1_desc">>
    File under 4.1.2.1.14 Disclosure button is missing the expanded state.
  • Hidden:
    <div style="display:none;">...</div>
    Does not appear in accessibility tree

Disabilities Affected

Assistive technology only works well when code is done properly.
