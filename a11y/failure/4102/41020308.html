Expected Result

If an option in a combobox (select) that uses aria-activedescendent is selected, it has the selected state.

Actual Result

A combobox option does not have a selected state.

Solution

  • Use aria-selected="true" on the selected option.

Failing Example

Custom Select-Only Combobox

This example does not communicate the selected state of the currently selected option to assistive technology.

  • NVDA: "Language, list, Select a Language, 1 of 7"
Language

Passing Example

Custom Select-Only Combobox

  • NVDA: "Language, list, Select a Language, 1 of 7"
{% include examples/select-only.html %}

  • Motor Impairment: Properties enable compatibility with assistive technology such as speech recognition software.
