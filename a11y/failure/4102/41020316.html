Expected Result

Sortable table headers communicate the sort state to assistive technology.

Actual Result

A sortable table header does not communicate the sort state to assistive technology

Solution

  • Use aria-sort on the sortable table header.

Failing Example

This example does not communicate the sort state to assistive technology.
