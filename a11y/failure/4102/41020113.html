Expected Result

If a control functions as a toggle button, it has a "pressed" state.

Actual Result

A toggle button does not have a "pressed" state.

Solution

Use the aria-pressed attribute on the toggle button. Set it to "true" when pressed and to "false" when not pressed.

Examples

Button

Mute 
<a tabindex="0" role="button" id="toggle" aria-pressed="false">Mute</a>

Not Applicable

The following items are not applicable to this failure:

  • Hidden:
    <div style="display:none;">Mute</div>
    Does not appear in accessibility tree

Why It's Important

Assistive technology only works well when code is done properly.
