Expected Result

Image input buttons have accessible names.

Actual Result

An image input button does not have an accessible name.

Solution

HTML

Provide an accessible name for the image button, using one of the following methods:

  • Place an alt attribute on the <input type="image"> tag and set it to the accessible name, OR
  • Place an aria-label attribute on the <input type="image"> tag and set it to the accessible name, OR
  • Place a title attribute on the <input type="image"> tag and set it to the accessible name, OR
  • Use the aria-labelledby attribute on the <input type="image"> tag and point it to the id of a text element to use as the accessible name

Failing Examples

name attribute

This example has a name attribute, but this does not provide an accessible name. Therefore, the button's accessible name is the default "Submit Query".

<input type="image" name="Search" src="search.png">

Null alt attribute

This example has a null alt attribute. Since this is an interactive element, it cannot be a decorative image and has the default accessible name "Submit Query".

<input type="image" alt="" src="search.png">

Invalid Target

This example uses aria-labelledby to point to an element with no text.

<div id="divOne"></div>
<input type="image" aria-labelledby="divOne" src="search.png">

This example uses aria-labelledby to point to a non-existent id.

Search
<div id="divOne">Search</div>
<input type="image" aria-labelledby="oneDiv" src="search.png">

Passing Examples

alt attribute

This example uses the alt attribute on the <input type="image"> element.

<input type="image" src="search.png" alt="Search">

aria-label attribute

This example uses the aria-label attribute on the <input type="image"> element.

<input type="image" src="search.png" aria-label="Search">

title attribute

This example uses the title attribute on the <input type="image"> element.

<input type="image" src="search.png" title="Search">

aria-labelledby attribute

This example uses the aria-labelledby attribute on the <input type="image"> element to point to the search field's <label>.

<label id="srch" aria-label="Search">
        <input type="search">
    </label>
    <input type="image" src="search.png" aria-labelledby="srch">

Not Applicable

The following items are not applicable to this failure:

  • Text button:
    <button>Text Button</button>
    File under {{ site.data.4102.4102[41020101].name }}
  • Input type "button":
    <input type="button" value="input type button">
    File under {{ site.data.4102.4102[41020101].name }}
  • Image inside button:
    <button><img src="search.png" alt="Search"></button>
    File under {{ site.data.1101.1101[11010101].name }}
  • Non-active image:
    <img src="search.png" alt="Search">
    File under {{ site.data.1101.1101[11010201].name }}
  • Hidden:
    <input type="image" src="search.png" style="display:none;">
    Does not appear in accessibility tree

Disabilities Affected

  • Blindness/Vision Impairment: Programmatic names enable compatibility with assistive technology such as screen readers and screen magnifiers.
  • Motor Impairment: Programmatic names enable compatibility with assistive technology such as speech recognition software.
