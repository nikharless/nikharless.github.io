Expected Result

Selectable grids communicate the selected state to assistive technology.

Actual Result

A selectable grid does not communicate the selected state to assistive technology.

Solution

Use the aria-selected attribute on the gridcell element. It may also be used on row elements when entire rows are selectable.

Failing Example

NVDA: ""

Passing Example

Calendar Grid

  • NVDA/Chrome (on focus): "1, not selected, Week 1, row 2, Sunday, column 2"
  • NVDA/Chrome (on change): "selected" or "not selected"
