Expected Result

If a control functions as a combobox (select), it has the aria-controls property and an expanded state.

Actual Result

A combobox does not have the required attributes.

Solution

  • Use the aria-controls attribute to identify the element that serves as the popup.
  • Use the aria-expanded attribute and set it to "true" when expanded and "false" when collapsed.

Examples

{% include examples/select-only.html %} 
<div role="combobox" 
    aria-controls="listbox1" 
    aria-expanded="false" 
    aria-haspopup="listbox" 
    aria-labelledby="combo1-label" 
    id="combo1" 
    tabindex="0" 
    aria-activedescendent>
    Select a Language
</div>

Not Applicable

The following items are not applicable to this failure:

  • Hidden:
    <div style="display:none;">...</div>
    Does not appear in accessibility tree

Why It's Important

Assistive technology only works well when code is done properly.
