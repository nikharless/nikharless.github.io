Expected Result

The currently selected tab communicates the selected state to assistive technology.

Actual Result

The current tab does not communicate the selected state to assistive technology.

Solution

  • Use aria-selected="true" on the currently selected tab control.
  • Use aria-selected="false" on all non-selected tab controls.
