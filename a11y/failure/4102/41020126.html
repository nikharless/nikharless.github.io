Expected Result

State changes are spoken by the screen reader.

Actual Result

A state change is not spoken by the screen reader.

Solution

  • Update the value of the attribute to match the current state of the control.
  • Ensure state changes are spoken by the screen reader.

Examples

Not Applicable

The following items are not applicable to this failure:

  • Hidden with CSS:
    <input type="textbox" style="display:none;">
    Does not appear in accessibility tree
  • Hidden with ARIA:
    <input type="textbox" aria-hidden="true">
    Does not appear in accessibility tree

Why It's Important

Assistive technology only works well when code is done properly.
