Expected Result

Meters have all appropriate roles and attributes.

Actual Result

A meter does not have all the appropriate roles or attributes.

Solution

Meter

  • Use the "meter" role on the meter element.
  • Use aria-label or another method to give the meter an accessible name.
  • Use aria-valuenow on the meter element to represent the current value of the meter.
  • Use aria-valuemin on the meter element to represent the minimum value of the meter.
  • Use aria-valuemax on the meter element to represent the maximum value of the meter.
  • Optionally, use aria-valuetext on the meter element if the default number format does not make sense in the meter's context.
