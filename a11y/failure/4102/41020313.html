Expected Result

If a control functions as a switch, it communicates the checked state to assistive technology.

Actual Result

A switch does not communicate the checked state to assistive technology.

Solution

  • Use the aria-checked attribute on the switch element.

Failing Example

Custom Switch

This switch does not communicate the checked state to assistive technology.

  • NVDA (on focus, regardless of state): "Toggle this switch, switch, off"
  • NVDA (on change): Nothing is spoken

Passing Example

Custom Switch

This example uses aria-checked to communicate the checked state to assistive technology.

  • NVDA (on focus, when off): "Toggle this switch, switch, off"
  • NVDA (on focus, when on): "Toggle this switch, switch, on"
  • NVDA (on change): "on" and "off"
