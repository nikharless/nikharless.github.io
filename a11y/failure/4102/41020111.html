Expected Result

Dialogs have an accessible name.

Actual Result

A dialog does not have an accessible name.

Solution

Provide an accessible name for the dialog using one of the following methods:

  • Use the aria-label attribute and set the value to the name, OR
  • Use the aria-labelledby attribute and point it to a text element which can function as the accessible name, such as the dialog's heading.

Failing Examples

No Accessible Name

This modal does not have an accessible name.

Modal with No Accessible Name

This modal does not have an accessible name.

<div role="alertdialog" aria-modal="true">

Passing Example

aria-labelledby

This modal uses aria-labelledby pointing to the visible modal heading to provide the modal with an accessible name.

Modal with Accessible Name

This modal uses aria-labelledby pointing to the visible modal heading to provide the modal with an accessible name.

 
<dialog aria-labelledby="dialog-heading">
    <h2 id="dialog-heading" tabindex="-1">Modal with Accessible Name</h2>

Not Applicable

The following items are not applicable to this failure:

  • No dialogs:
    No dialogs on the page

Disabilities Affected

  • Blindness/Vision Impairment: Programmatic names enable compatibility with assistive technology such as screen readers and screen magnifiers.
  • Motor Impairment: Programmatic names enable compatibility with assistive technology such as speech recognition software.
