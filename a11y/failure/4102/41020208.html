Expected Result

Dialogs have the role of "dialog".

Actual Result

A dialog is missing the dialog role.

Solution

Assign role="dialog" to the dialog element.

Failing Example

Dialog

This example does not assign a role to the dialog.

  • NVDA (when dialog is opened): ""

Passing Examples

Dialog

This example uses role="dialog".

  • NVDA (when dialog is opened): ""

Example Pass

This is a modal dialog.

<div id="modal" role="dialog" aria-modal="true" aria-labelledby="labelId">

Not Applicable

The following items are not applicable to this failure:

  • Hidden:
    <div style="display:none;">...</div>
    Does not appear in accessibility tree

Disabilities Affected

  • Blindness/Vision Impairment: Programmatic roles enable compatibility with assistive technology such as screen readers and screen magnifiers.
  • Motor Impairment: Programmatic roles enable compatibility with assistive technology such as speech recognition software.
