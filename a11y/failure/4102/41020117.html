Expected Result

If a control functions as a radio button, it has a checked state.

Actual Result

A radio button does not have a checked state.

Solution

  • Use the native <input type="radio"> mark-up to create the radio button, OR
  • Use the aria-checked attribute on the custom radio button element.

Examples

Native HTML

Side 
<label><input type="radio"> Soup</label>
<label><input type="radio" checked> Salad</label>

Custom radio button

Check this radio button 
<div role="radio" tabindex="0" aria-checked="false"> Check this radio button
