Expected Result

Elements contain any required child elements.

Actual Result

An element requires a particular child element, but the child element is missing.

Solution

HTML

Ensure that any required child elements are present.

  • <head> must contain a <title> element
  • <table> must contain one or more <tr> elements

Examples

Table

Types of paint

Watercolor water-based
Acrylic water-based
Oil oil-based 
<p>Types of paint</p>
<table>
    <tr>
        <th>Watercolor</th>
        <td>water-based</td>
    </tr>
    <tr>
        <th>Acrylic</th>
        <td>water-based</td>
    </tr>
    <tr>
        <th>Oil</th>
        <td>oil-based</td>
    </tr>
</table>
