WCAG | 1.2.8 Media Alternative (Prerecorded) | 1.2.8.a Text Transcript (Prerecorded)

Expected Result Prerecorded synchronized media and video-only media has a text transcript which includes the visual content.
Actual Result The video has a text transcript, but it is not adequate.
Solution Provide a text transcript that includes all key visual elements necessary to understand the video content.
Example
Why It's Important Deafblind users cannot see or hear the video content, so they require both the audio and visual information in text format so that it can be interpreted by assistive technology such as a braille device.
