Expected Result

When a component gains focus, a new window, tab, or dialog is not automatically launched.

Actual Result

When a component gains focus, a new window, tab, or dialog is automatically launched.

Solution

  • Launch new windows, tabs, or dialogs only on activation.

Failing Examples

Link Opens in New Window on Focus

When this link receives focus for the first time, the link automatically opens in a new window.

Top Ten Songs of 1972

Dialog Opens on Focus

When the user focuses on the checkbox for the first time, an alert dialog automatically opens.

Passing Examples

Link Only Opens on Activation

Top Ten Songs of 1972

Dialog Only Opens on Request

The user can click the info button to open a dialog.

Not Applicable

  • Context changes on input:
    File under {{ site.data.3202.3202[3202].name }}.
  • Links open in new window on activation:
    File under {{ site.data.2404.2404[2404].name }}.
  • Automatic tabs:
    Tab content changes when focusing on a tab in a tab list is not a change of context, because the behavior is logical within the tab list context.

Disabilities Affected

  • Blind/Visual Impairment: Unexpected changes of context can cause significant confusion if the user can't see what has happened.
  • Cognitive Impairment: Unexpected changes of context can cause the user to become disoriented.
  • Motor Impairment: Unexpected changes of context can make content significantly more difficult to navigate.
