Expected Result

Functionality that uses motion actuation has alternative user interface controls.

Actual Result

Motion actuation has no alternative user interface controls.

Solution

Add user interface controls which perform the same functionality.

Examples

  • Shaking a device performs an undo function. A button is also available to do the same thing.
  • A setting is provided to disable the shake-to-undo functionality.

Why It's Important

Some users cannot hold a device or keep it steady.

References

  • Understanding Success Criterion {{ sc.name }}
  • G213: Provide conventional controls and an application setting for motion activated input
