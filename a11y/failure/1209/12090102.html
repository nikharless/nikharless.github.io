Expected Result

Live audio broadcasts must have a text equivalent.

Actual Result

The live audio broadcast has a text equivalent, but it is inaccurate.

Solution

If you are using automatic captions, switch to manually written ones.

Example

Why It's Important

People who are deaf, hard of hearing, or have difficulty understanding audio information for any reason, need to be able to access the same information provided in the live audio broadcast.
