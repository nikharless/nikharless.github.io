Expected Result

Live audio broadcasts must have a text equivalent.

Actual Result

The live video has captions, but they are difficult to read and can't be adjusted.

Solution

  • Ensure caption text has at least 3:1 contrast (assuming a minimum font size of 18 points) with the background at all times, OR
  • Use a player which allows the user to adjust the appearance of the captions.

Example

Able Player

Able Player allows the user to customize caption appearance.

Why It's Important

Users with low vision need to be able to easily read the captions.
