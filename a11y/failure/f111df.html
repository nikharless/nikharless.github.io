WCAG | 1.1.1 Non-text Content | 1.1.1.d Decorative Images (Text Alternative)

Expected Result

 Decorative or redundant images must be hidden from screen readers.

Actual Result

 The decorative or redundant image is not hidden from screen readers.

Solution

 Set the image as decorative either in the source document or directly in the PDF.

Adobe PDF

In PDF files, decorative elements are tagged as <Artifact>.

To find decorative images tagged as <Figure> without alternate text:

  1. Select Tools > Accessibility > Accessibility Check.
  2. Select Alternate Text and Headings from the Category drop-down, make sure Figures require alternate text is checked, and activate the Start Checking button.
  3. In the results, check for failures under Alternate Text > Figures alternate text.
  4. Select a figure from the failure list.
  5. Right-click on the figure (or go to the Options menu) and select Fix from the menu.
  6. Check the Decorative figure checkbox, then activate Save & Close.

To manually mark decorative images incorrectly tagged as <Figure> as <Artifact>:

  1. Locate the decorative image in the Tags panel.
  2. Right-click on the image (or go to the Options menu) and select Change Tag to Artifact....

To mark an untagged image as <Artifact>:

  1. Locate the decorative image in the PDF document.
  2. Select Tools > Accessibility > Reading Order.
  3. Draw a rectangle around the decorative image to select it.
  4. Activate the Background/Artifact button in the Reading Order dialog.
