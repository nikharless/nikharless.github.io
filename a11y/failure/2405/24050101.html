Expected Result

There are at least two ways to locate a web page within a set of web pages.

Actual Result

There are fewer than two ways to locate a web page within the set of web pages.

Solution

Provide at least two of the following, or some other method:

  • Menu
  • Table of contents
  • Site map
  • Search mechanism
  • List of links
  • Links within content

Failing Example

Passing Example

Not Applicable

  • Pages that are a step in a process

Why It's Important

  • Blind/visually impaired: Users may find it easier to use a search function rather than navigate through a large number of links.
  • Motion impaired: Users may find it easier to use a search function rather than navigate through a large number of links.
  • Cognitively impaired: Users may prefer a table of contents or site map rather than navigating through complex navigation menus. Others may prefer to explore the site in a sequential manner with next and back links.
