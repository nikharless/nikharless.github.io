Expected Result

All content can be accessed when the screen is resized.

Actual Result

Elements are removed entirely when the content is resized.

Solution

Use a responsive layout which restructures the screen when the screen size changes. Some options for simplifying the layout without removing content include the following:
  • Reflow sidebars to appear in the vertical flow
  • Consolidate horizontal menus into pop-up menus
  • Move content blocks inside disclosure widgets or pop-ups
  • Move content to separate pages and provide links
  • Resize meaningful images instead of removing them

Examples

Why It's Important

Users with visual disabilities need to be able to view and use the content when zoomed.
