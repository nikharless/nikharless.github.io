Expected Result

The content is viewable and usable without scrolling in two dimensions when the viewport is set to 320 CSS pixels wide.

Actual Result

When the viewport is set to 320 CSS pixels wide, two-dimensional scrolling is required to access all the content.

Solution

  • Use a layout which reflows when the screen size changes, AND
  • Use relative measurements for fonts, controls, images, and containers

Examples

Responsive Design

The following working examples resize and reflow elements to fit them within the viewport.

  • Working example: Using media queries and grid CSS to reflow columns
  • Working example: Using CSS Flexbox for Reflow
  • Working example: Using Adjustable Labels and Inputs for Reflow

Why It's Important

Users with visual or cognitive disabilities can become disoriented if they have to scroll back and forth to read and use content.
