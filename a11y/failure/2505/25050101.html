Expected Result

Control target size is at least 44x44 CSS pixels.

Actual Result

Control target size is less than 44x44 CSS pixels.

Solution

Increase the target size to at least 44x44 CSS pixels.

Examples

Buttons

<button style="min-height:44px; min-width:44px;">Click Here</button>
.triangle-right-img {
    background-image: url("triangle.png");
    width: 50px;
    height: 50px;
    background-size: 50px;
}

These icons are 40px, but there is 10px of space between them.

font-size: 40px; 
margin: 0 10px 0 0;

Why It's Important

Some users cannot click or tap small controls.
