Expected Result

The web page has a lang attribute that sets the default page language.

Actual Result

The web page has a lang attribute, but the language code does not match the actual language of the page.

Solution

Set the lang value to the language code for the actual language of the page.

Examples

<html lang="fr">

Why It's Important

Assistive technology users get information in the correct language.
