Expected Result

When an error is automatically detected, the error is described in text.

Actual Result

An error is automatically detected, but it is not described in text.

Solution

Provide an error message describing the error in text.

Examples

Why It's Important

Flagging errors helps people with reduced sight and cognitive disabilities resolve them.
