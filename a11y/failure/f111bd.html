WCAG | 1.1.1 Non-text Content | 1.1.1.b Meaningful Images (Text Alternative)

Expected Result All meaningful images must have a text alternative that conveys the same information as the image.
Actual Result The meaningful emoji does not have a text alternative.
Solution Add a text alternative which describes the purpose of the meaningful emoji using the following method:
  1. Add a <span> tag around the emoji.
  2. Add role="img" to the <span>.
  3. Add an aria-label attribute to the <span>.
  4. Set the value to a text alternative that makes sense in context.

Why It's Important Blind and visually impaired users need to have a text alternative to meaningful images so that they can be interpreted by assistive technology such as screen readers and braille devices. Without a text alternative, these users might not get the information intended by the emojis.
