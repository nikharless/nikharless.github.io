Expected Result

Form input labels and instructions have enough information for the user to know what input is expected.

Actual Result

Form input label does not accurately describe the input's purpose.

Solution

Provide labels for each form input which succinctly and accurately describe the input's purpose.

Examples

Individual Labels

Why It's Important

Everyone, especially those with cognitive disabilities, will know how to respond.
