Expected Result

Form input labels and instructions have enough information for the user to know what input is expected.

Actual Result

Form input label does not include the expected data format.

Solution

Include the expected data format in the label or instructions.

Examples

Why It's Important

Everyone, especially those with cognitive disabilities, will know how to respond.
