Expected Result

Form input labels and instructions have enough information for the user to know what input is expected.

Actual Result

Form input instructions are provided, but they are not accurate.

Solution

Ensure the instructions for completing the form inputs are accurate.

Examples

The format indicated in the label matches the format actually accepted by the form validation.

Why It's Important

Everyone, especially those with cognitive disabilities, will know how to respond.
