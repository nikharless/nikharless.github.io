Expected Result

Regular size placeholder text has a contrast ratio of at least 4.5:1 with its background.

Actual Result

Regular size placeholder text does not have a contrast ratio of at least 4.5:1 with its background.

Solution

Change the text or background so that the contrast ratio between the placeholder text and the background is at least 4.5:1.

Passing Example

This placeholder text is styled to use the color #767676, which has a contrast ratio of 4.5:1 with the white background.

input::placeholder {
    color: #767676;
}

Not Applicable

  • Large text:
    File under {{ site.data.1403.1403[14030204].name }}.

Disabilities Affected

  • Vision impaired: Users often have difficulty reading low contrast text.
  • Colorblind: Some color deficiencies lower the contrast in the user's vision.
