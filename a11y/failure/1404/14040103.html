Expected Result

Text content is readable and functional when resized up to 200%

Actual Result

Text content is clipped, truncated or obscured when resized to 200% and cannot be accessed.

Solution

  • Ensure that text containers resize when the text resizes.
  • Use measurements that are relative to other measurements in the content.

Examples

Bad Example: Truncated Text

The chief obstacle to the progress of the human race is the human race.
.quote {
    font-size:100%; 
    width:120px; 
    height:100px; 
    overflow: hidden; 
    border: thin solid gray;
}

Good Example: Flexible Height

The chief obstacle to the progress of the human race is the human race.
.quote {
    font-size:100%; 
    width:120px; 
    min-height:6.25em; 
    border: thin solid gray;
}

Why It's Important

Vision impaired users need to be able to read and use the content when zoomed.
