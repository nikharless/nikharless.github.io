Expected Result

All content and functionality must be available within one click or tap when resized to 200%.

Actual Result

Content or functionality is removed entirely when resized to 200%.

Solution

Ensure content is not removed entirely when resized to 200%. It may be moved or hidden, but should remain accessible with a single click or tap.

Failing Examples

A web page has a menu with a visible list of several options. When the user resizes the page to 200%, the menu disappears and the options are no longer available on the page.

Passing Examples

A web page has a menu with a visible list of several options. When the user resizes the page to 200%, the menu is collapsed into a hamburger icon, which the user may click or tap to view the same options.

Not Applicable

Disabilities Affected

  • Visual Impairment: Users must be able to increase the text size of content so that they can read it.

References

  • Understanding Success Criterion {{ sc.name }}
