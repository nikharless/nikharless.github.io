Expected Result

Text content is readable and functional when resized up to 200%.

Actual Result

Text content is clipped, truncated or obscured when resized to 200%. The content can still be accessed, but it is not obvious that content is missing and can be accessed.

Solution

Provide an indicator that more content is available.

Examples

Single Line

In this example, a tooltip shows the full text, and an ellipsis alerts the user that the text is truncated.

Ellipsis for Single Line Text
.title {
    font-size:150%; 
    font-weight: bold;
    width:200px; 
    white-space: nowrap;
    overflow: hidden; 
    text-overflow: ellipsis;
    border: thin solid gray;
}
<h2 class="title" title="Ellipsis for Single Line Text">Ellipsis for Single Line Text</h2>

Multiline

This example works with multiline text.

The chief obstacle to the progress of the human race is the human race.
.quote {
    width: 150px;
    border: thin solid gray;
    overflow: hidden;
    display: -webkit-box;
    -webkit-line-clamp: 3;
    -webkit-box-orient: vertical;
}
<div class="quote" title="The chief obstacle to the progress of the human race is the human race.">The chief obstacle to the progress of the human race is the human race.</div>

Why It's Important

Vision impaired users need to be able to read and use the content when zoomed.
