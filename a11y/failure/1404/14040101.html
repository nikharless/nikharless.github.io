Expected Result

Text content is resizable to 200%.

Actual Result

Text does not resize due to the use of viewport units.

Solution

Use another unit to resize text, such as percent, named sizes, or em units.

Failing Example

Text sized using viewport units

Zoom the page to view how each of the examples resizes.
