Expected Result

Text content is resizable to 200%.

Actual Result

Text does not resize due to the use of viewport meta tags which disable zooming.

Solution

  • In the "viewport" <meta> tag, remove the user-scalable=0 or user-scalable=no option, or set it to "yes".
  • Either remove the maximum-scale option, or set it to at least "2".

Failing Examples

user-scalable is set to "no"

<meta name="viewport" content="user-scalable=no">

maximum-scale is set to "1"

<meta name="viewport" content="maximum-scale=1.0">

maximum-scale is set to "yes" which translates to 1

<meta name="viewport" content="maximum-scale=yes">

Passing Examples

No user-scalable is set to "yes"

<meta name="viewport" content="user-scalable=yes">

maximum-scale is set to "2"

<meta name="viewport" content="maximum-scale=2.0">

Initial scale

This fits the initial view of the content to the screen size without disabling zoom capability.

<meta name="viewport" content="width=device-width, initial-scale=1.0">

Not Applicable

  • No <meta name="viewport"> element
  • No content attribute in the <meta name="viewport"> element
  • <meta name="viewport"> element does not specify maximum-scale or user-scalable

Why It's Important

  • Visually impaired: Users must be able to increase the text size of content so that they can read it.
