Expected Result

Header cells are marked up with the correct scope attribute.

Actual Result

Header cells are marked up with the scope attribute, but the value is invalid.

Solution

Use one of the 4 valid values for scope:
  • Use scope="col" for column headers.
  • Use scope="colgroup" for colgroup headers.
  • Use scope="row" for row headers.
  • Use scope="rowgroup" for rowgroup headers.

Disabilities Affected

  • Blind/visually impaired: Screen reader users rely on programmatic markup to understand how content is arranged. Table relationships are visible at a glance for sighted users. Assistive technology users need to get this relationship information from the programmatic structure and to be able to easily navigate the data.
