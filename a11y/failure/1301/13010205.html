Expected Result

Header cells are marked up with the correct scope attribute.

Actual Result

Header cells are marked up with the scope attribute, but the value is invalid.

Solution

Use one of the 4 valid values for scope:
  • Use scope="col" for column headers.
  • Use scope="colgroup" for colgroup headers.
  • Use scope="row" for row headers.
  • Use scope="rowgroup" for rowgroup headers.

Example

Table with Column and Row Headers

Name Time Attempts
John 18 min 3
Jane 20 min 1 
<table>
    <tr>
        <th scope="col">Name</th>
        <th scope="col">Time</th>
        <th scope="col">Attempts</th>
    </tr>
    <tr>
        <th scope="row">John</th>
        <td>18 min</td>
        <td>3</td>
    </tr>
    <tr>
        <th scope="row">Jane</th>
        <td>20 min</td>
        <td>1</td>
    </tr>
</table>

Why It's Important

Table relationships are visible at a glance for sighted users. Assistive technology users need to get this relationship information from the semantic structure and to be able to easily navigate the data.
