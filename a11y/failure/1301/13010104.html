Expected Result

Visible sections of a web page are marked up semantically as landmarks.

Actual Result

Visible sections of a web page are marked up as landmarks, but a landmark is invalid.

Solution

Ensure regions are nested correctly:
  • The <header> and <footer> elements must reside at the top level. If they are nested, they are not recognized as header or footer regions.

Passing Example

<html>
        <header>
            <img src="logo.png" alt="Yoyodyne Home">
            <nav aria-label="main>
                ...
            </nav>
        </header>
        <main>
            ...
            <aside aria-labelledby="recent">
                <h2 id="recent">Recent Articles</h2>
            </aside>
        </main>
        <footer>
            © 2024 Yoyodyne Propulsion Systems
            <nav aria-label="footer">
                ...
            </nav>
        </footer>
    </html>

Disabilities Affected

  • Blindness/vision impairment: Page regions are visible at a glance for sighted users. Landmarks allow assistive technology users to have this same information and to easily navigate to different areas of the page.
