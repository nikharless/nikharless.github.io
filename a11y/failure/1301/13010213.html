Expected Result

Layout tables do not have ARIA table roles.

Actual Result

A layout table has ARIA table roles.

Solution

  • Change the role to role="presentation"
  • Remove any of the following from the markup:
    • role="row"
    • role="rowgroup"
    • role="columnheader"
    • role="rowheader"
    • role="cell"

Failing Example

Layout Table with role="table"

Using ARIA table roles causes assistive technology to read the layout as a table.

Not Applicable

  • Data tables

Disabilities Affected

  • Blind/visually impaired: Assistive technology conveys data table information differently from regular content, which can cause confusion if layout tables are read as tables.
