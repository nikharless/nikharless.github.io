Expected Result

Ordered lists are marked up as ordered lists.

Actual Result

Ordered list is not marked up as an ordered list.

Solution

Native HTML (Preferred Method)

Use the <ol> and <li> tags to mark up ordered (numbered) lists.

OR

ARIA

Use ARIA to mark up the list.

Note: ARIA does not distinguish between ordered and unordered lists. Therefore, it's best practice to use native HTML list markup.

  • Add the role="list" attribute to the list element.
  • Add the role="listitem" attribute to each list item element.

Example

Native HTML (Preferred Method)

  1. Item 1
  2. Item 2
  3. Item 3
<ol>
    <li>Item 1</li>
    <li>Item 2</li>
    <li>Item 3</li>
<ol>

ARIA

Item 1
Item 2
Item 3
<div role="list">
    <div role="listitem" style="display: list-item; margin-left: 1.3em; list-style-type: decimal;">Item 1</div>
    <div role="listitem" style="display: list-item; margin-left: 1.3em; list-style-type: decimal;">Item 2</div>
    <div role="listitem" style="display: list-item; margin-left: 1.3em; list-style-type: decimal;">Item 3</div>
</div>

Not Applicable

  • Comma-separated lists
  • Non-list items marked up as lists

Disabilities Affected

  • Blindness/Visual Impairment: Screen reader users rely on programmatic markup to understand how content is arranged. When lists are properly marked up, screen readers inform users of the structure and how many items are in the list.
