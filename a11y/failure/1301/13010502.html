Expected Result

Ordered lists are marked up as ordered lists.

Actual Result

Ordered list is not marked up as an ordered list.

Solution

Native HTML (Preferred Method)

Use the <ol> and <li> tags to mark up ordered (numbered) lists.

OR

ARIA

Use ARIA to mark up the list.

Note: ARIA does not distinguish between ordered and unordered lists. Therefore, it's best practice to use native HTML list markup.

  • Add the role="list" attribute to the list element.
  • Add the role="listitem" attribute to each list item element.

Example

Native HTML (Preferred Method)

  1. Item 1
  2. Item 2
  3. Item 3
<ol>
    <li>Item 1</li>
    <li>Item 2</li>
    <li>Item 3</li>
<ol>

ARIA

Item 1
Item 2
Item 3
<div role="list">
    <div role="listitem">Item 1</div>
    <div role="listitem">Item 2</div>
    <div role="listitem">Item 3</div>
</div>

Why It's Important

When a list is properly marked up, screen readers notify users that the list exists and how many items are in it. This is information that visual users can see at a glance, so it must be conveyed to assistive technology users as well.
