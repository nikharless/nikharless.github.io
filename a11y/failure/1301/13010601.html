Expected Result

Form element is programmatically associated with its visible label or instruction.

Actual Result

Form element is not programmatically associated with its visible label or instruction

Solution

  • Use explicit labels (<label for="[form element id]">), OR
  • Use implicit labels (enclose form element within <label> tags), OR
  • Use aria-labelledby (for labels) or aria-describedby (for additional instructions).

Native HTML is the preferred solution and should be used whenever possible.

Examples

Explicit Labels


<label for="fname">First Name</label>
<input type="text" id="fname">

Implicit Labels

<label>First Name
    <input type="text">
</label>

ARIA

First Name
Last Name
I agree to the terms and conditions. 
<span id="fname">First Name>/span>
<input type="text" aria-labelledby="fname">

Disabilities Affected

  • Blind/visual impairment: Screen reader users hear the label when they navigate to the form element without having to search around to find it.
