Expected Result

Data table header cells are identified programmatically

Actual Result

Data table header cells are not identified programmatically

Solution

Mark up table header cells as semantic header cells.
  • Use native HTML, preferred (e.g., <th>), OR
  • Use ARIA to assign table header roles (e.g., role="columnheader", role="rowheader").

Failing Example

Data Table without Header Markup

The headers in this table are marked up as data cells. This prevents assistive technology from associating the headers with the data.

Park Top Feature
Denali National Park Denali - North America's highest peak
Great Smoky Mountains National Park Clingmans Dome

Not Applicable

  • Layout tables

Why It's Important

  • Blind/visually impaired: Screen reader users rely on programmatic markup to understand how content is arranged. Table relationships are visible at a glance for sighted users. Assistive technology users need to get this relationship information from the programmatic structure and to be able to easily navigate the data.
