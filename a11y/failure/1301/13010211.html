Expected Result

Non-tabular data is not marked up as a table.

Actual Result

Table markup is used for layout purposes.

Solution

  • Add the role="presentation" attribute to the <table> element, OR
  • Remove any of the following from the markup:
    • <th> elements
    • <caption> elements
    • summary attributes
    • headers attributes
    • scope attributes
    • CSS display property values of table, table-caption, table-cell, table-column, table-column-group, table-footer-group, table-header-group, or table-row

Passing Example

This is an outdated method for page layout. Do not code pages this way. If updating an old page, add role="presentation".

Not Applicable

  • Data tables

Disabilities Affected

  • Blind/visually impaired: Assistive technology conveys data table information differently from regular content, which can cause confusion if layout tables are read as tables.
