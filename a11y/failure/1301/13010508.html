Expected Result

Only content that functions as lists are tagged as lists in a PDF document.

Actual Result

Content which does not function as a list is tagged as a list.

Solution

Adobe Acrobat Pro

  • Select the Tags button in the navigation pane.
  • In the Accessibility tags panel, find and select the incorrectly tagged element.
  • Right-click the element or open the Options (...) menu and select Properties....
  • From the Type drop-down, select the correct tag for the element.
  • Close the Object Properties dialog.

Example

Why It's Important

Screen readers rely on accessibility tags in PDF documents to interpret the content for users.
