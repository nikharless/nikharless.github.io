Expected Result

Lists in PDF documents are tagged as lists.

Actual Result

The list is not tagged as a list.

Solution

Adobe Acrobat Pro

  • Tag the list with the <List> tag.
  • Tag the bullet or number character with the <Lbl> tag.
  • Tag the bullet text with the <LBody> tag.
  • Enclose the <Lbl> and <LBody> elements within the <LI> tag.
  • Nest the <LI> tags within the <List> tag.

Example

<L>
    <LI>
        <Lbl>
        <LBody>
    <LI>
    ...

Disabilities Affected

  • Blind/visually impaired: Screen reader users rely on programmatic markup to understand how content is arranged. When lists are properly marked up, screen readers inform users of the structure and how many items are in the list.
