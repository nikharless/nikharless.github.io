Expected Result

Form element is programmatically associated with its corresponding table header.

Actual Result

Form element is not programmatically associated with its corresponding table header.

Solution

  • Use native HTML table header elements (<tr">) along with scope="col" or scope="row" to define the header/cell relationship, OR
  • Use ARIA role="rowheader" or row="colheader" on the table header elements.

Native HTML is the preferred solution.
