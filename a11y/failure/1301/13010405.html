Expected Result

Headings in PDF are tagged as headings.

Actual Result

Headings in PDF are not tagged as headings.

Solution

Adobe Acrobat Pro

No Tag

  1. Select Tools > Accessibility > Reading Order to open the Reading Order dialog.
  2. Draw a rectangle around the heading. It should highlight with a pink outline.
  3. Activate the appropriate Heading button in the Reading Order dialog.

Incorrect Tag

  1. In the Tags panel, right-click the incorrectly tagged heading or open the Options menu and select Properties....
  2. In the Object Properties dialog, select the correct heading tag from the Type drop-down.

Examples

Adobe Acrobat Pro

No Tag

Incorrect Tag

Disabilities Affected

  • Blind/visually impaired: Screen reader users rely on programmatic markup to understand how content is arranged. Screen readers also allow users to navigate through the page with headings.
