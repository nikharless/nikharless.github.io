Expected Result

Headings are tagged hierarchically.

Actual Result

Headings are tagged with the wrong heading level.

Solution

Adobe Acrobat Pro

  1. In the Tags panel, right-click the incorrectly tagged heading or open the Options menu and select Properties....
  2. In the Object Properties dialog, select the correct heading level tag from the Type drop-down.

Example

Adobe Acrobat Pro

Why It's Important

Visual users can see at a glance how information on the page is structured. Assistive technology users rely on semantic markup to understand how content is arranged.
