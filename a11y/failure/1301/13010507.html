Expected Result

Lists in PDF documents are nested correctly.

Actual Result

List elements are not nested correctly.

Solution

Adobe Acrobat Pro

  • Nest the <LI> tags within the <List> tag.
  • Nest the <Lbl> and <LBody> elements within the <LI> tag.

Example

<L>
    <LI>
        <Lbl>
        <LBody>
    <LI>
    ...

Why It's Important

List tags must be properly nested in order for screen readers to interpret them correctly.
