Expected Result

When both column and row headers are present, the header cells are marked up with the scope attribute.

Actual Result

Both column and row headers are present, but the header cells do not have a scope attribute.

Solution

  • Add scope="col" to column headers.
  • Add scope="colgroup" to column group headers.
  • Add scope="row" to row headers.
  • Add scope="rowgroup" to row group headers.

Not Applicable

Why It's Important

  • Blind/visually impaired: Screen reader users rely on programmatic markup to understand how content is arranged. Table relationships are visible at a glance for sighted users. Assistive technology users need to get this relationship information from the programmatic structure and to be able to easily navigate the data.
