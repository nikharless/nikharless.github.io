Expected Result

Form elements are programmatically associated with their visible associations.

Actual Result

A form element is not programmatically associated with a visible association.

Solution

Error message

  1. Add an id attribute to the error message container and set it to a unique name.
  2. Add the aria-describedby attribute to the <input> field and point it to the id value of the error message.

Failing Example

Error Next to Field

All fields are required.

<label>First Name:
    <input type="text" required aria-invalid="true">
    <span id="fnameError">First Name is required.</span>

Passing Examples

Error Next to Field

All fields are required.

<label>First Name:
    <input type="text" required aria-invalid="true" aria-describedby="fnameError">
    <span id="fnameError">First Name is required.</span>
