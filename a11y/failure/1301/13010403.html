Expected Result

Only text that functions as a heading is marked up semantically as a heading.

Actual Result

Text that does not function as a heading is marked up as a heading.

Solution

  • Remove heading tags (<h1>-<h6>) from non-heading text.
  • Use CSS to style the text as desired.

Example

100%
coverage!
<span style="font-size: 48px; font-weight: bolder;">100%</span><br>
coverage!</p>

Why It's Important

Visual users can see at a glance how information on the page is structured. Assistive technology users rely on semantic markup to understand how content is arranged.

References

  • Understanding SC 1.3.1: Info and Relationships
  • F43: Failure of Success Criterion 1.3.1 due to using structural markup in a way that does not represent relationships in the content
