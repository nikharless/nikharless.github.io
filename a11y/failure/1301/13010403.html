Expected Result

Only text that functions as a heading is marked up programmatically as a heading.

Actual Result

Text that does not function as a heading is marked up as a heading.

Solution

  • Remove heading tags (<h1>-<h6>) from non-heading text.
  • Use CSS to style the text as desired.

Example

100%
coverage!
<span style="font-size: 48px; font-weight: bolder;">100%</span><br>
coverage!</p>

Why It's Important

  • Blind/visually impaired: Screen reader users rely on programmatic markup to understand how content is arranged. Screen readers also allow users to navigate through the page with headings.
