Expected Result

Layout tables do not have header elements or attributes.

Actual Result

A layout table has header elements or attributes.

Solution

  • Add the role="presentation" attribute to the <table> element, OR
  • Remove any of the following from the markup:
    • <th> elements
    • headers attributes
    • scope attributes
    • CSS display property values of table-header-group

Not Applicable

  • Data tables

Why It's Important

  • Blind/visually impaired: Assistive technology conveys data table information differently from regular content, which can cause confusion if layout tables are read as tables.
