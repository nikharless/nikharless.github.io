Expected Result

If individual labels for controls in a group do not provide enough context to understand the purpose of the control, the controls are programmatically grouped.

Actual Result

The individual labels for controls in a group do not provide enough context to understand the purpose of the control, and the controls are not programmatically grouped.

Solution

Native HTML (Preferred Method)

Use the <fieldset> and <legend> tags to group the controls.

  • Enclose the entire group of controls in a <fieldset> element.
  • Place the group label inside a <legend> element as a child of the <fieldset> element.
OR

ARIA

Use ARIA to group the controls.

  • Create a parent element for the control group.
  • Add the role="group" attribute to the parent element.
  • Add an id attribute to the tag containing the group label text, and set it to a unique value.
  • Add an aria-labelledby attribute to the parent (group) element and point it to the id of the group label tag.

Passing Examples

Native HTML (Preferred Method)

Pancake Toppings 

<fieldset>
        <legend>Pancake Toppings</legend>
        <input type="checkbox" name="toppings" id="syrup">
        <label for="syrup">Maple syrup</label><br>
        <input type="checkbox" name="toppings" id="berries">
        <label for="berries">Blueberries</label><br>
        <input type="checkbox" name="toppings" id="cream">
        <label for="cream">Whipped cream</label>
    </fieldset>

ARIA

Pancake Toppings



<p id="toppingsLabel">Pancake Toppings</p>
    <div role="group" aria-labelledby="toppingsLabel">
        <input type="checkbox" name="toppings" id="syrup">
        <label for="syrup">Maple syrup</label><br>
        <input type="checkbox" name="toppings" id="berries">
        <label for="berries">Blueberries</label><br>
        <input type="checkbox" name="toppings" id="cream">
        <label for="cream">Whipped cream</label>
    </div>

Option group

<label for="pancakeToppings">Pancake Toppings</label>
    <select id="pancakeToppings" name="pancakeToppings">

Disabilities Affected

  • Blindness/Visual Impairment: Assistive technology users browsing through a form will not hear group labels unless they are programmatically associated. Screen readers in forms mode will ignore any text that is not programmatically associated with form controls.
