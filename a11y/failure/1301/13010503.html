Expected Result

Description lists are marked up as description lists.

Actual Result

A description list is not marked up as a description list.

Solution

Native HTML (Preferred Method)

Use the <dl>, <dt>, and <dd> tags to mark up description lists.

OR

ARIA

Use ARIA to mark up the list.

  • Add the role="term" attribute to the term element.
  • Add the role="definition" attribute to the description element.

Examples

Native HTML (Preferred Method)

Accessibility
The practice of making your website usable by as many people as possible.
<dl>
    <dt>Accessibility</dt>
    <dd>The practice of making your website usable by as many people as possible.</dd>
<dl>

ARIA

Accessibility
The practice of making your website usable by as many people as possible.
<div>
    <div role="term">Accessibility</div>
    <div role="definition">The practice of making your website usable by as many people as possible.</div>
<div>

Not Applicable

  • Comma-separated lists
  • Non-list items marked up as lists
  • Unordered lists:
    File under {{ site.data.1301.1301[13010501].name }}.
  • Ordered lists:
    File under {{ site.data.1301.1301[13010502].name }}.

Why It's Important

  • Blind/visually impaired: Screen reader users rely on programmatic markup to understand how content is arranged. When lists are properly marked up, screen readers inform users of the structure and how many items are in the list.
