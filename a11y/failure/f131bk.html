Expected Result

Non-tabular data is not marked up as a table.

Actual Result

Table markup is used for layout purposes.

Solution

  • Remove any of the following from the markup:
    • <th> elements
    • <caption> elements
    • summary attributes
    • headers attributes
    • scope attributes
    • CSS display property values of table, table-caption, table-cell, table-column, table-column-group, table-footer-group, table-header-group, or table-row
  • Add the role="presentation" attribute to the <table> element.

Why It's Important

Screen readers read tables in a special way in order to communicate the data relationships within. Layout tables should not be read this way, as it can cause confusion.
