WCAG | 1.2.2 Captions (Prerecorded) | 1.2.2.a Captions (Prerecorded)

Expected Result Prerecorded video with audio must have synchronized captions which convey the same information provided in the audio.
Actual Result The video has captions, but they are difficult to read and can't be adjusted.
Solution
  • Ensure caption text has at least 3:1 contrast (assuming a minimum font size of 18 points) with the background at all times.
  • Use a video player which allows the user to adjust the appearance of the captions.
Example

Caption Settings

Some video players allow the user to adjust the appearance of captions.
Why It's Important Users with low vision need to be able to easily read the captions.
