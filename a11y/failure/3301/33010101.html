Expected Result

Error messages identify the items with the error.

Actual Result

Error message does not identify the item with the error.

Solution

Multiple methods are encouraged.

  • Identify the item with the error in the error message text, OR
  • Modify the label of the item in error to indicate it has an error, OR
  • Set the aria-invalid attribute to "true" on the item with the error, OR
  • Use an icon with a text alternative next to the item in error, OR
  • Use color to highlight the item in error. Note: See 1.4.1 Use of Color for requirement.

Examples

Why It's Important

Flagging errors helps people with reduced sight and cognitive disabilities resolve them.
