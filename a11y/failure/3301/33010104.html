Expected Result

When an error is automatically corrected, an error message must be provided in visible text.

Actual Result

An error is auto-corrected, but the error is not identified in visible error text.

Solution

Provide error text identifying the error and the auto-correction in visible text.

Example

Why It's Important

  • Blind/vision impaired: Users are able to perceive that an error has occured, where it has occurred, and why.
  • Cognitive impairments: Users are able to understand that an error has occured, where it has occurred, and why.
