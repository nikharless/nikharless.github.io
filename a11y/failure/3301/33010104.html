Expected Result

When an error is automatically detected, the error message is readable by assistive technology.

Actual Result

An error message is visible, but it is not read by assistive technology.

Solution

  • Use an alert dialog to provide the error message, OR
  • Display the error message in an aria-live region, OR
  • Move focus to the error message.

Examples

Why It's Important

Flagging errors helps people with reduced sight and cognitive disabilities resolve them.
