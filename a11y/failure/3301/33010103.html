Expected Result

When an error is automatically detected, the error is described in visible text.

Actual Result

An error is automatically detected, but it is not described in visible text.

Solution

Provide a visible error message describing the error in text.

Examples

Why It's Important

  • Blind/vision impaired: Users are able to perceive that an error has occured, where it has occurred, and why.
  • Cognitive impairments: Users are able to understand that an error has occured, where it has occurred, and why.

Not Applicable

  • No error detection:
    The form does not check for errors.
  • Suggestions for fixing errors:
    File under {{ site.data.3303.3303[3303].name }}.
  • Programmatic association:
    File under {{ site.data.1301.1301[1301].name }}.
  • Assistive technology notification:
    File under {{ site.data.4103.4103[4103].name }}.
