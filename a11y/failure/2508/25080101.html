Expected Result

Control target size is at least 24 x 24 CSS pixels.

Actual Result

Control target size is less than 24 x 24 CSS pixels.

Solution

Increase the target size to at least 24 x 24 CSS pixels.

Examples

Buttons

<button style="min-height:24px; min-width:24px;">Click Here</button>
.triangle-right-img {
    background-image: url("triangle.png");
    width: 24px;
    height: 24px;
    background-size: 24px;
}

These icons are 20px, but there is 10px of space between them.

font-size: 20px; 
margin: 0 10px 0 0;

Select

Menu options are also subject to this criterion.

{% include examples/select-only.html %} 
/* menu options */
.combo-option {
    padding: 10px 12px 12px;
    min-height: 24px;
}

Why It's Important

Some users cannot click or tap small controls.
