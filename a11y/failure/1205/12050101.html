Expected Result

Prerecorded synchronized media must have an audio alternative (audio descriptions) that includes all of the information presented in the visual content.

Actual Result

The prerecorded synchronized media content does not have an audio alternative for the video portion.

Solution

Add an audio description that includes all of the information presented in the video using one of the following methods:

  • Provide an audio file of the audio descriptions that can be played alongside the video, OR
  • Provide a separate video with an audio description embedded.

Descriptions should be neutral, clear, and present tense.

Why It's Important

  • Blind/vision impaired: People who are blind or vision impaired need to be able to access the same information provided in the video track.
  • Information processing: People who have the ability to see use alternatives to more easily process the visual information.
