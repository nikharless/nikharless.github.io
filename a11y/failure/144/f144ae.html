Expected Result

Text content is resizable to 200%.

Actual Result

Text-based form controls do not resize.

Solution

Provide an indicator that more content is available.

Examples

Single Line

.title {
    font-size:150%; 
    font-weight: bold;
    width:200px; 
    white-space: nowrap;
    overflow: hidden; 
    text-overflow: ellipsis;
    border: thin solid gray;
}
<h2 class="title" title="Ellipsis for Single Line Text">Ellipsis for Single Line Text</h2>

Why It's Important

Vision impaired users need to be able to zoom the content so they can read it.
