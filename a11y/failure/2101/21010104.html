Expected Result

Modals are dismissible, without submitting data, using only the keyboard.

Actual Result

A modal cannot be dismissed using only the keyboard without submitting data.

Solution

  • Ensure any close buttons in the modal can be activated with the keyboard.
  • Enable the use of the Esc key to close the modal.
