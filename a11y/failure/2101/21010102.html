Expected Result

All functionality is available through the keyboard interface.

Actual Result

A control cannot be accessed using only a keyboard, because it's coded with only mouse event handlers.

Solution

  • Provide keyboard event handlers alongside all mouse event handlers, OR
  • Provide alternate, keyboard accessible controls that accomplish the same function.

Examples

Custom Control

Using onclick and onkeyup allows activation with both the mouse and the keyboard.

Do Stuff
<div id="customButton" role="button" tabindex="0">Do Stuff</div>
<script>
    customButton.onclick = function (e) { doStuff(); };
    customButton.onkeyup = function (e) { if(e.keyCode == 13) { doStuff(); } };
</script>

Why It's Important

Some users with vision disabilities or hand tremors are unable to use a mouse well or at all.
