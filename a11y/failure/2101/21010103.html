Expected Result

Links (<a>) have href attributes.

Actual Result

A link is not in the tab order, because it's coded without an href attribute.

Solution

  • Provide an href attribute pointing to the target url, AND
  • Provide keyboard event handlers for any mouse event handlers.

Failing Example

Scripted Link with no href

Our Parks

<a onclick="return doStuff();">Our Parks</a>

Passing Example

Scripted Link with Inactive href

Our Parks 
<a href="javascript:void(0);" onclick="return doStuff();">Our Parks</a>

Not Applicable

Disabilities Affected

Link anchor tags (<a>) are not placed in the tab order unless they have an href attribute.

  • Blindness/Vision Impairment: Blind users cannot use devices such as mice which require eye-hand coordination.
  • Motor Impairment: Users have difficulty using a mouse and rely on a keyboard.
