Expected Result

All functionality is available through the keyboard interface.

Actual Result

A link cannot be activated using only a keyboard, because it's coded without an href attribute.

Solution

  • Provide an href attribute pointing to the target url, AND
  • Provide keyboard event handlers.

Examples

Scripted Link

Using onclick allows activation with both the mouse and the keyboard.

Click this link 
<a href="#" onclick="return doStuff();">Click this link</a>

Why It's Important

Some users with vision disabilities or hand tremors are unable to use a mouse well or at all.
