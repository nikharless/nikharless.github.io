Expected Result

All functionality is available through the keyboard interface.

Actual Result

Multiple accesskey attributes have the same value, resulting in a conflict which renders some functionality unusable with the keyboard.

Solution

Ensure each accesskey has a unique value.

Examples

The search field uses the accesskey value "s", so when this access key is pressed, the search field gains focus.

Access key implementation varies across browsers. Press Shift + Alt + S to use the access key in most Windows browsers and Control + Option + s to use the access key in most Mac browsers.

<input type="search" accesskey="s"> 
<button>Search</button>

Why It's Important

Duplicate accesskey values cause conflicts which render some functionality unusable with the keyboard.
