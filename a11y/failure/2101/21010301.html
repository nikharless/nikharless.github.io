Expected Result

All functionalities based on custom gestures are also available when a screen reader is turned on.

Actual Result

An action cannot be performed using a screen reader, because the custom gesture conflicts with an existing assistive technology shortcut, and there is no equivalent means of achieving the action.

Solution

Change the custom gesture to one that does not conflict with existing assistive technology gestures.

Examples

Why It's Important

Screen readers provide users with gestures for browsing the web. These gestures will override any conflicting custom gestures on a web page.
