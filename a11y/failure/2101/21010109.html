Expected Result

All functionality is available through the keyboard interface.

Actual Result

Content that displays on mouse hover does not display on keyboard focus.

Solution

Add keyboard focus/blur event handlers equivalent to the mouseover/mouseout event handlers.

Examples

Informational tooltip

Why It's Important

Some users with vision disabilities or hand tremors are unable to use a mouse well or at all.
