Expected Result

Drag and drop feature has keyboard functionality.

Actual Result

Drag and drop feature does not have keyboard functionality.

Solution

  • Ensure that the drag and drop functionality can be controlled with the keyboard, OR
  • Provide an alternate mechanism that can be controlled with the keyboard.

Example

Drag and Drop

Drag and drop the artists under the heading they belong. See keyboard instructions.

Favorite Artists

  • Frida Kahlo
  • Vincent Van Gogh
  • Francis Bacon
  • Pablo Picasso
  • Gustav Klimt
  • Lucian Freud

Tolerable Artists

  • None

Rejected Artists

  • None

Keyboard Instructions

  • Use the Tab to move to the artist you want to move.
  • Select the item you want to move with the Spacebar.
  • Use the Up and Down arrow keys to select a target in the context menu.
  • Press Enter to move the artist to the target list.

Disabilities Affected

  • Blind/vision impaired: Blind users cannot use devices such as mice which require eye-hand coordination.
  • Hand tremors: Users have difficulty using a mouse and rely on a keyboard.
