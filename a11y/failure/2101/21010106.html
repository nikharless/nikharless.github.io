Expected Result

All functionality is available through the keyboard interface.

Actual Result

A server-side image map is used, but there are no text links for keyboard users.

Solution

Add an alternate list of text links containing all the same links in the image map.

Examples

Server-Side Image Map

  • North America
  • South America
  • etc.

Why It's Important

Some users with vision disabilities or hand tremors are unable to use a mouse well or at all.
