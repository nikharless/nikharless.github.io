Expected Result

Server-side image map has redundant text links for keyboard users.

Actual Result

A server-side image map is used, but there are no text links for keyboard users.

Solution

Add an alternate list of text links containing all the same links in the image map.

Failing Example

Server-Side Image Map

Click a continent for more information.

<img ismap src="world-map.png" alt="World Map">

Passing Example

Server-Side Image Map with Redundant Links

Click a continent or one of the links below for more information.

  • North America
  • South America
  • etc.

Not Applicable

  • Client-side image maps

Disabilities Affected

  • Blindness/Vision Impairment: Blind users cannot use devices such as mice which require eye-hand coordination.
  • Motor Impairment: Users have difficulty using a mouse and rely on a keyboard.

References

  • Understanding Success Criterion {{ sc.name }}
