Expected Result

Hidden elements do not gain focus.

Actual Result

Focus moves to a hidden element.

Solution

Ensure hidden elements do not get focus.

Examples

When the link inside this expandable area is hidden, focus doesn't move to it.

{% include examples/disclosure.html %}

Disabilities Affected

  • Blind/visually impaired: Screen reader and screen magnifier users can become disoriented if tabbing moves the focus in an illogical order.
  • Cognitive impairments: Users can become disoriented if tab focus is in an illogical order.
  • Mobility impairments: Users may have a more difficult time navigating a page if tab focus is in an illogical order.
