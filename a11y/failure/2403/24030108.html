Expected Result

Hidden elements do not gain focus.

Actual Result

Focus moves to a hidden element.

Solution

Ensure hidden elements do not get focus.

Failing Example

When the link inside this expandable area is visually hidden, keyboard focus can still move to it.

  • Shave his belly with a rusty razor
  • Put him in a long boat until he's sober
  • Stick him in a scupper with a hosepipe bottom
  • Put him in the bed with the captain's daughter

...early in the morning.

See Drunken Sailor Guidelines for more information.

Passing Example

When the link inside this expandable area is hidden, focus doesn't move to it.

Disabilities Affected

  • Blindness/Vision Impairment: Screen reader and screen magnifier users can become disoriented if tabbing moves the focus in an illogical order.
  • Cognitive Impairment: Users can become disoriented if tab focus is in an illogical order.
  • Motor Impairment: Users may have a more difficult time navigating a page if tab focus is in an illogical order.
