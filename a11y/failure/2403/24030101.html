Expected Result

The navigation order of focusable elements is logical and intuitive.

Actual Result

The navigation order of focusable elements is not logical and intuitive.

Solution

Ensure the focus order is logical and preserves the meaning and operability of the content.

Examples

Why It's Important

Navigating a site or application with only the keyboard will make sense.

References

  • Understanding Success Criterion {{ sc.name }}
  • G59: Placing the interactive elements in an order that follows sequences and relationships within the content
