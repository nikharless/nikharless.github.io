Disabilities Affected

  • Blindness/Vision Impairment: Screen reader and screen magnifier users can become disoriented if tabbing moves the focus in an illogical order.
  • Cognitive Impairment: Users can become disoriented if tab focus is in an illogical order.
  • Motor Impairment: Users may have a more difficult time navigating a page if tab focus is in an illogical order.
