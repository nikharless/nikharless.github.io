Expected Result

Focus is restricted to the modal while it is open.

Actual Result

Focus is not restricted to the modal while it is open, and the user can focus on background content.

Solution

Restrict the focus to stay within the modal while it is open. Focus order should move from the last focusable item in the modal back to the first focusable item in the modal.

Examples

Focus is maintained within this modal until it is closed.

{% include examples/modal.html %}

Why It's Important

  • Blind/visually impaired: Screen reader and screen magnifier users can become disoriented if tabbing moves the focus in an illogical order.
  • Cognitive impairments: Users can become disoriented if tab focus is in an illogical order.
  • Mobility impairments: Users may have a more difficult time navigating a page if tab focus is in an illogical order.
