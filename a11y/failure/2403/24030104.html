Expected Result

Focus moves to modal dialog when it is opened.

Actual Result

Focus does not move to modal dialog when it is opened.

Solution

Place the focus inside the modal when it is opened. Generally, focus should be placed on the first control inside the modal.

Failing Example

When this modal opens, focus does not move to it.

Example Failing Modal

Sign up for our mailing list.

Note: This form does not actually collect any information.

Passing Example

When this modal opens, focus goes to the "Email" field.

Example Passing Modal

Sign up for our mailing list.

Note: This form does not actually collect any information.

Disabilities Affected

  • Blindness/Vision Impairment: Screen reader and screen magnifier users can become disoriented if tabbing moves the focus in an illogical order.
  • Cognitive Impairment: Users can become disoriented if tab focus is in an illogical order.
  • Motor Impairment: Users may have a more difficult time navigating a page if tab focus is in an illogical order.
