Expected Result

Focus moves to modeless dialog immediately after the trigger element.

Actual Result

Focus does not move to modeless dialog immediately after the trigger element.

Solution

  • Place the modeless dialog in the DOM immediately after the trigger element so the focus moves to it in logical order, OR
  • Deliberately move the focus to the modeless dialog after the trigger element.

Examples

The added content in this example is located immediately after the trigger element in the DOM. When the user tabs through the page, the focus goes from the trigger element to the link within the additional content before moving to the next control.

Why It's Important

Navigating a site or application with only the keyboard will make sense.
