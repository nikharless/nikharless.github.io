Expected Result

When modal is closed, focus returns to the trigger element or to the next logical location.

Actual Result

When modal is closed, focus is not returned to the trigger element or the next logical location.

Solution

When the modal is closed, place the focus back on the trigger element. If this is not the most logical location, place the focus on the element that is.

Disabilities Affected

  • Blindness/Vision Impairment: Screen reader and screen magnifier users can become disoriented if tabbing moves the focus in an illogical order.
  • Cognitive Impairment: Users can become disoriented if tab focus is in an illogical order.
  • Motor Impairment: Users may have a more difficult time navigating a page if tab focus is in an illogical order.
