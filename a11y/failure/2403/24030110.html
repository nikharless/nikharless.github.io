Expected Result

PDF tab order follows the document structure.

Actual Result

PDF tab order does not follow the document structure.

Solution

Set the PDF preferences to use the document structure for the Tab order.

Adobe Acrobat Pro

  1. Select All tools > Prepare for accessibility > Check for accessibility.
  2. In the Accessibility Checker Options, under Category: Document, make sure Document structure provides a logical reading order is checked and activate the Start Checking button.
  3. In the Accessibility Checker results, expand the Page Content section.
  4. Select Tab order - Failed.
  5. Right-click or open the Options menu and select Fix.

OR

  1. Select Pages (thumbnails) from the toolbar.
  2. In the Pages pane, press Ctrl+a (Windows) to select all pages.
  3. Right-click a page thumbnail or open the Options menu and select Page Properties.
  4. In the Page Properties dialog, select the Tab Order tab.
  5. Select the Use Document Structure radio button and activate the OK button.

Examples

Why It's Important

Navigating a PDF file with only the keyboard will make sense.
