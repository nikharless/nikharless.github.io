Expected Result

When a page visually scrolls as a result of a scripted action, such as a "go to top" link, the focus also moves to the target location.

Actual Result

Focus does not follow a scripted visual scroll.

Solution

Deliberately move the focus to the target location in addition to visually scrolling the page.

Passing Example

These links visually scroll to the target area and also move the keyboard focus.

Disabilities Affected

  • Blindness/Vision Impairment: Screen reader and screen magnifier users can become disoriented if tabbing moves the focus in an illogical order.
  • Cognitive Impairment: Users can become disoriented if tab focus is in an illogical order.
  • Motor Impairment: Users may have a more difficult time navigating a page if tab focus is in an illogical order.
