WCAG | 1.2.7 Extended Audio Description (Prerecorded) | 1.2.7.a Extended Audio Description (Prerecorded)

Expected Result Prerecorded synchronized media must have an audio description of all important visual content.
Actual Result The prerecorded synchronized media content has an audio description, but it is not adequate.
Solution Provide an extended audio description that includes all key visual elements necessary to understand the video content. Pause the video where necessary to fit all the information.
Examples

Extended Audio Description
Why It's Important Visually impaired users, or any users who have difficulty understanding moving content, need an alternate means of receiving the visual information.
