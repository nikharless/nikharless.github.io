Expected Result

When text is resized up to 200%, it does not require horizontal scrolling.

Actual Result

When text is resized up to 200%, it requires horizontal scrolling.

Solution

  • Use a layout which reflows when the screen size changes, AND
  • Use relative measurements for fonts and containers

Examples

Why It's Important

Users with visual or cognitive disabilities can become disoriented if they have to scroll back and forth to read a block of text.
