Expected Result

There is a visible indicator showing which element is in focus.

Actual Result

The focus indicator is disabled with CSS.

Solution

Eliminate any CSS which removes or obscures the default focus indicator, such as:

  • :focus { outline: none }
    This removes the default focus indicator, so focused links look exactly like non-focused links.
  • a { outline: thin dotted black }
    This makes the non-focused link have the same outline as the default focus indicator, so again focused and non-focused links look exactly the same.
  • a { border: medium solid black }
    This styling obscures the default focus indicator, making it impossible to see.

Examples

Tab through the links and controls to see the focus indicators.
