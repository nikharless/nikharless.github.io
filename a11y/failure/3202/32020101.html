Expected Result

When a control state or value changes, focus is not removed or redirected.

Actual Result

When a control state or value changes, focus is removed or redirected.

Solution

  • Do not remove focus when the state or value changes, AND
  • Intentionally place focus if necessary so that it isn't lost. OR
  • Provide instructions warning that the change will occur.

Examples

Why It's Important

Content that behaves predictably is especially important to people with disabilities.
