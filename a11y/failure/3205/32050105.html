Expected Result

Page content is not refreshed without user request.

Actual Result

Page content is refreshed without user request.

Solution

  • Refresh content only on user request.
  • Do not use time limits on redirects, so that they are not noticeable.

Examples

Why It's Important

Content that behaves predictably is especially important to people with disabilities.
