Expected Result

Real text is used to convey information rather than images of text whenever possible.

Actual Result

An image of text is used where real text is possible.

Solution

Replace the image of text with real text, using CSS styling to retain the desired visual appearance.

Examples

Accessibility Matters
.neon {
    font-family: Roboto, Oxygen-Sans, Ubuntu, Cantarell, sans-serif; 
    font-size: 2rem;
    font-weight: 400; 
    letter-spacing: 5px;
    text-align: center; 
    text-transform: uppercase; 
    color: #fff; 
    text-shadow: 0 0 5px #fff, 0 0 10px #fff, 0 0 20px #fff, 0 0 40px #0ff, 0 0 80px #0ff, 0 0 90px #0ff, 0 0 100px #0ff, 0 0 150px #0ff;
}

Why It's Important

Users with vision or reading disabilities often need to adjust the text in order to read it.
