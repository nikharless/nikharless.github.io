Expected Result

The focused state has at least a 3:1 contrast with the unfocused state.

Actual Result

The focused state does not have at least a 3:1 contrast with the unfocused state.

Solution

Adjust the colors so that the focus state has at least a 3:1 contrast with the same pixels in the unfocused state.

Examples

Tab to each element to see the focus indicator.

Background Color

The green focus state background has a 4.2:1 contrast with the unfocused state background.

  • Home
  • News
  • Contact
  • About

Bottom Border

The green focus state bottom border has a 4.2:1 contrast with the same pixels in the unfocused state background.

  • Home
  • News
  • Contact
  • About

Outline

The focus state outline has a 4.5:1 contrast with the same pixels (white) in the unfocused state.

Inset Border

The focus state border has a 3.48:1 contrast with the same pixels in the unfocused state background.

Double Border

Using a two-color border can be useful for ensuring adequate contrast on light and dark backgrounds. Since each border is at least 2px thick, only one of them needs to contrast with adjacent colors.

Why It's Important

Users with visual impairments, including older people, can't see small changes in visual appearance.
