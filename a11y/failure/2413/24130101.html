Expected Result

The area of the focus indicator is at least as large as the area of a 2 CSS pixel thick perimeter around the element.

Actual Result

The area of the focus indicator is smaller than the area of a 2 CSS pixel thick perimeter around the element.

Solution

Make the focus indicator larger so that it's at least as large as the area of a 2 CSS pixel thick perimeter around the element. The easiest way to do this is to make it a 2 CSS pixel thick line around the element.

Examples

Tab to each element to see the focus indicator.

Solid Line

The outline, border, and outset buttons have 2px lines. Because the focus indicator is smaller than the size of the button, the inset button has a 3px line and the bottom border button has a 6px line to make up for the difference in area.

Why It's Important

Users with visual impairments, including older people, can't see small changes in visual appearance.
