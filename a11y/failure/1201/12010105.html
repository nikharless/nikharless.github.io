Expected Result

Text transcript must be an actual text alternative.

Actual Result

The prerecorded audio-only content has a text transcript, but it is not a true equivalent.

Solution

Update the text transcript to be an equivalent alternative to the information contained in the audio.

The following are examples of text that would not be equivalent alternatives to audio:

  • File names
  • Placeholder text
  • Programming references

Example

Why It's Important

  • Deaf/hard of hearing: People who are deaf or hard of hearing need to be able to access the same information provided in the audio.
  • Information processing: People who have the ability to hear use transcripts to more easily process the auditory information.
  • Non-native speakers: Transcripts help non-native speakers of the audio language understand the content.
  • Environment: Transcripts allow all users to access the auditory information in environments where sound can't be used or cannot be heard.
  • SEO: Transcripts allow search engines to catalog the information contained in audio.
