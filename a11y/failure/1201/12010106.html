Expected Result

If audio-only media is used as a media alternative to text, it must be clearly labeled as such.

Actual Result

The audio-only media is an alternative to text on the page, but it is not clearly labeled as such.

Solution

Clearly label the audio-only media as an alternative to text on the page.

Failing Example

Passing Example

Not Applicable

  • Prerecorded Synchronized Media:
    File under {{ site.data.1202.1202[12020107].name }}.
  • Notification Sounds

Disabilities Affected

Media which is intended as an alternative to text on the page needs to be clearly labeled as such, so users can find it and so users who expect text alternatives know not to look for them.
