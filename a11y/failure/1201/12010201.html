Expected Result

Prerecorded video-only content must have a text (transcript) or audio (audio descriptions) alternative that includes all of the information presented in the video.

Actual Result

The prerecorded video-only content does not have an audio description or text transcript.

Solution

Add an audio description or text transcript that includes all of the information presented in the video using one of the following methods:

  • Provide an audio file of a description of the video, OR
  • Provide a separate video with an audio description included as the audio track, OR
  • Provide a text transcript describing the video content adjacent to the video, OR
  • Provide a text transcript via a link adjacent to the video.

Passing Example

Silent Video with Audio Description

Not Applicable

Disabilities Affected

  • Blind/vision impaired: People who are blind or vision impaired need to be able to access the same information provided in the video.
  • Information processing: People who have the ability to see use alternatives to more easily process the visual information.
  • Deafblind: Transcripts allow deafblind users access to information presented in video content.
  • SEO: Transcripts allow search engines to catalog the information contained in video content.
