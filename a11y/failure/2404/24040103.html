Expected Result

The purpose of the PDF link can be determined from the link text along with its immediate context without having to leave the link.

Actual Result

The purpose of the PDF link cannot be determined without leaving the link.

Solution

Adobe Acrobat Pro

  1. Locate the link in the Tags panel.
  2. Right-click the <Link> tag or open the Options menu and select Properties....
  3. In the Object Properties dialog, select the Tag tab.
  4. Enter descriptive link text in the Alternate Description for Links field.

Disabilities Affected

  • Blind/vision impaired: Users can determine the purpose of the link without having to explore around the page.
  • Motion impaired: Users can skip links they are not interested in without having to visit the links to see what the content is.
  • Cognitively impaired: Users can avoid confusion caused by having to navigate to and from links to see what the content is.
