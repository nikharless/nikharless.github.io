Expected Result

The purpose of the link can be determined from the link text along with its immediate context without having to leave the link.

Actual Result

The purpose of the link cannot be determined without leaving the link.

Solution

Adobe Acrobat Pro

  1. Locate the link in the Tags panel.
  2. Right-click the <Link> tag or open the Options menu and select Properties....
  3. In the Object Properties dialog, select the Tag tab.
  4. Enter descriptive link text in the Alternate Description for Links field.

Examples

Why It's Important

Users with visual and cognitive disabilities can navigate more easily.
