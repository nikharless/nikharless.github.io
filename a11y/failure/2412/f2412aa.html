Expected Result

Focus is not obscured in any way by author-created content.

Actual Result

Focus is partially obscured by author-created content.

Solution

Ensure focusable content is not obscured in any way by author-created content. Some ways of avoiding overlap include:

  • Use CSS scroll-padding to make room for sticky content.
  • Put the overlapping content into a modal which can be dismissed.
  • Remove overlapping content when focus is removed.

Examples

  • Working example: Using CSS scroll-padding to un-obscure content.

Why It's Important

People who can't use a mouse need to see what has keyboard focus.
