Expected Result

The mechanism for pausing, stopping, or hiding the moving, blinking, or scrolling content passes all applicable Success Criteria.

Actual Result

The mechanism for pausing, stopping, or hiding the moving, blinking, or scrolling content does not pass all applicable Success Criteria.

Solution

Provide a mechanism which itself passes all applicable Success Criteria.

For example, if a button is used, the button must meet all requirements for buttons.

Examples

Disabilities Affected

If the mechanism itself does not pass all applicable Success Criteria, it does not sufficiently perform its intended function.
