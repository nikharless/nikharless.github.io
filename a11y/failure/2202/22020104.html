Expected Result

The user is able to interact with other content while moving, blinking, or scrolling content is paused.

Actual Result

The user is not able to interact with other content while the moving, blinking, or scrolling content is paused.

Solution

Provide a mechanism to pause, stop, or hide the moving, blinking, or scrolling element which allows the user to interact with other content while it is paused.

Examples

Scrolling Text

This scrolling text element can be paused when the mouse is hovered over it. However, if this were the only means of pausing it, the user could not interact with the other content, because as soon as they removed the mouse hover, it would begin scrolling again.


Scrollin', scrollin', scrollin'
Keep scrollin', scrollin', scrollin'
Until you are controllin'

Why It's Important

Some people with cognitive disabilities and attention deficits cannot concentrate with continual movement.
