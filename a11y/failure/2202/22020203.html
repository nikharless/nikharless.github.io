Expected Result

The user is able to interact with other content while auto-updating content is paused.

Actual Result

The user is not able to interact with other content while the auto-updating content is paused.

Solution

Provide a mechanism to pause, stop, or hide the auto-updating content which allows the user to interact with other content while it is paused.

Examples

This carousel pauses when the mouse is hovered over it. However, if this were the only means of pausing it, the user could not interact with the other content, because as soon as they removed the mouse hover, it would begin auto-advancing again.

{% include examples/carousel.html %}

Why It's Important

Some people with cognitive disabilities and attention deficits cannot concentrate with continual movement.

References

  • Understanding Success Criterion {{ sc.name }}
  • G4: Allowing the content to be paused and restarted from where it was paused
  • G186: Using a control in the Web page that stops moving, blinking, or auto-updating content
