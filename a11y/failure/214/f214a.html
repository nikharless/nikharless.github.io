Expected Result

If a single character key shortcut exists, then it can be turned off, remapped, or are only active when the relevant user interface component is in focus.

Actual Result

A single character key shortcut is always active and cannot be turned off or remapped.

Solution

  • Only allow the shortcut to be active when the appropriate element is in focus, OR
  • Provide a mechanism to turn off the shortcut, OR
  • Provide a mechanism to remap the shortcut

Examples

Bad Examples

Videos of speech user trouble with single character key shortcuts:

  • Single character key shortcuts affecting speech input – example 1
  • Single character key shortcuts affecting speech input – example 2

Why It's Important

Character-key shortcuts are easy to accidentally trigger, especially with speech input.
