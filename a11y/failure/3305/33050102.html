Expected Result

When labels are not sufficient to fully describe the functionality, context-sensitive help is available for text inputs.

Actual Result

Text input labels do not include expected data format, but no context-sensitive help is available.

Solution

Provide context-sensitive help to describe the expected data format.

Examples

Why It's Important

People with cognitive or other disabilities can complete their tasks more easily.

References

  • Understanding Success Criterion {{ sc.name }}
  • G89: Providing expected data format and example
  • G184: Providing text instructions at the beginning of a form or set of fields that describes the necessary input
