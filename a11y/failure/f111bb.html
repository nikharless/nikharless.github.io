Expected Result

All meaningful images must have a text alternative that conveys the same information as the image.

Actual Result

The meaningful <svg> does not have a text alternative.

Solution

Add a text alternative which describes the purpose of the meaningful image using the following method:
  1. Add a <title></title> element as the first child of the the <svg> element, and enter the text alternative within, AND
  2. Add a unique id to the <title> element, AND
  3. Add the aria-describedby attribute to the <svg> element, and point it to the <title> element's id value, AND
  4. Add the role="img" attribute to the <svg> element to ensure that screen readers recognize it as an image and read the text alternative.

Example

SVG Image

<svg role="img" aria-describedby="a11yPolicy">
    <title id="a11yPolicy">Accessibility Policy</title>
    <g id="SVGRepo_tracerCarrier" stroke-width="0">
    ...
</svg>

Why It's Important

Blind and visually impaired users need to have a text alternative to meaningful images so that the images can be interpreted by assistive technology such as screen readers and braille devices. Without a text alternative, these users would not get the information conveyed by the image.
