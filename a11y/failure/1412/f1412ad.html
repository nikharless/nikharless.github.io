Expected Result

Text content is readable and functional when the spacing between letters, words, lines of text, and paragraphs is adjusted.

Actual Result

Text content is clipped, truncated or obscured when the spacing between letters, words, lines of text, and paragraphs is adjusted. The content can still be accessed, but it is not obvious that content is missing and can be accessed.

Solution

Provide an indicator that more content is available.

Examples

Use Dylan Barrellâ€™s 1.4.12 Text Spacing Bookmarklet to change the spacing.

Single Line

In this example, a tooltip shows the full text, and an ellipsis alerts the user that the text is truncated when the text spacing is adjusted.

Lorem ipsum
.title {
    font-size:150%; 
    font-weight: bold;
    width:200px; 
    white-space: nowrap;
    overflow: hidden; 
    text-overflow: ellipsis;
    border: thin solid #888;
}
<h2 class="title" title="Lorem ipsum">Lorem ipsum</h2>

Multiline

This example works with multiline text.

Lorem ipsum dolor sit amet, consectetur.
.quote {
    width: 150px;
    height: 85px;
    border: thin solid #888;
    overflow: hidden;
    display: -webkit-box;
    -webkit-line-clamp: 3;
    -webkit-box-orient: vertical;
}
<div class="quote" title="Lorem ipsum dolor sit amet, consectetur.">Lorem ipsum dolor sit amet, consectetur.</div>

Why It's Important

People with low vision or dyslexia may need to adjust text spacing in order to read the text.
