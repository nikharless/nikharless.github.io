Expected Result

The spacing between letters, words, lines of text, and paragraphs can be adjusted without being clipped.

Actual Result

Text is cut off when the spacing between letters, words, lines of text, and paragraphs is adjusted.

Solution

Code the text so that the spacing can be adjusted without clipping content.

Why It's Important

People with low vision or dyslexia may need to adjust text spacing in order to read the text.
