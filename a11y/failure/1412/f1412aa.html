Expected Result

The spacing between letters, words, lines of text, and paragraphs can be adjusted.

Actual Result

The spacing between letters, words, lines of text, and paragraphs cannot be adjusted.

Solution

Code the text so that it can be adjusted by the user with external tools.

Examples

Lorem ipsum dolor sit amet, consectetur adipiscing elit. Suspendisse et ultrices enim. Pellentesque pulvinar massa orci, sed ultrices tellus hendrerit ac. Ut turpis dui, dapibus id augue id, fermentum consectetur velit. Pellentesque pharetra hendrerit erat, ac convallis est lacinia quis.

Why It's Important

People with low vision or dyslexia may need to adjust text spacing in order to read the text.
