Expected Result

The different language text has a lang attribute set to the correct language.

Actual Result

The different language text has a lang attribute on its container tag, but the language code does not match the actual language of the text.

Solution

Change the value of the lang attribute to the appropriate language code.

Examples

You're not coming? ¡Qué chafa!

You’re not coming? <span lang="es">¡Qué chafa!</span>
