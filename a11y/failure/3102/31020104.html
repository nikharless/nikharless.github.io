Expected Result

The foreign text container tag has the language set to the appropriate language.

Actual Result

The foreign text container tag does not have the language set to the appropriate language.

Solution

  1. For each instance, locate and select the corresponding tag in the Tags tree.
  2. Right-click the tag or activate the Options menu, and select Propertiesâ€¦.
  3. In the Object Properties dialog, select the Tag tab.
  4. Select the correct language from the Language drop-down.

Examples

Why It's Important

Assistive technology users get information in the correct language.
