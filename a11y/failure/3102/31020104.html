Expected Result

The different language text container tag has the language set to the appropriate language.

Actual Result

The different language text container tag does not have the language set to the appropriate language.

Solution

  1. For each instance, locate and select the corresponding tag in the Tags tree.
  2. Right-click the tag or activate the Options menu, and select Propertiesâ€¦.
  3. In the Object Properties dialog, select the Tag tab.
  4. Select the correct language from the Language drop-down.

Examples

Disabilities Affected

  • Blindness/Visual Impairment: Screen reader users need text read to them with the correct pronunciation.
  • Cognitive Impairment: Users of text-to-speech software need content to be read with the correct pronunciation.
