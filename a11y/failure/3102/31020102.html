Expected Result

The foreign text has a lang attribute set to the correct language.

Actual Result

The foreign text has a lang attribute on its container tag, but it is invalid.

Solution

Change the value of the lang attribute to the appropriate valid language code.

Examples

You’re not coming? <span lang="es">¡Qué chafa!</span>

Why It's Important

Assistive technology users get information in the correct language.
