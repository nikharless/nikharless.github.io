Expected Result

The foreign text has a lang attribute set to the correct language.

Actual Result

The foreign text does not have a lang attribute on its container tag.

Solution

Add a lang attribute to the foreign text's container tag and set it to the appropriate language code.

Examples

You’re not coming? <span lang="es">¡Qué chafa!</span>

Why It's Important

Assistive technology users get information in the correct language.
