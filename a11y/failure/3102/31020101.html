Expected Result

The different language text has a lang attribute set to the correct language.

Actual Result

The different language text does not have a lang attribute on its container tag.

Solution

Add a lang attribute to the different language text's container tag and set it to the appropriate language code.

Examples

You're not coming? ¡Qué chafa!

You’re not coming? <span lang="es">¡Qué chafa!</span>

Not Applicable

The following are not applicable to this failure:

  • Default page language:
    File under {{ site.data.3101.3101[3101].name }}.

Why It's Important

  • Blind/visually impaired: Screen reader users need text read to them with the correct pronunciation.
  • Cognitively impaired: Users of text-to-speech software need content to be read with the correct pronunciation.
