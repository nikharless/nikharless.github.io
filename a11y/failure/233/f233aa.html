Expected Result

The user can disable any animation triggered by interaction.

Actual Result

Animation triggered by interaction cannot be disabled.

Solution

  • Provide a mechanism for the user to disable any animation triggered by interaction.

Examples

  • A site which uses parallax animation has a control at the top of the page which allows the user to turn off the parallax effect.
  • A site uses the prefers-reduced-motion CSS media query to automatically disable the animation.

Why It's Important

Motion animation can cause some users to experience dizziness, headaches, or nausea.
