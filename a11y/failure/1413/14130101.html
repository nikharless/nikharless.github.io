Expected Result

When additional content is triggered on pointer hover or on keyboard focus, that additional content can be dismissed without moving the mouse hover or keyboard focus.

Actual Result

When additional content is triggered on pointer hover or on keyboard focus, that additional content cannot be dismissed without moving the mouse hover or keyboard focus.

Solution

Code the popup to be dismissible using the Esc key and/or the mobile dismiss gesture.

Examples

The tooltip popup can be dismissed using the Esc key.

Informational tooltip

Why It's Important

Visually impaired users, when using a magnifier or zoomed text, may have troubles with triggered additional content (like custom tooltips). Mouse users frequently move the pointer to pan the magnified viewport. Almost the entire portion of the page visible in the restricted viewport may trigger the additional content, making it difficult for a user to pan without re-triggering the content. Keyboard users may also have trouble with triggered additional text obstructing the userâ€™s view of other content. Both use cases need a keyboard method of dismissing something that is obscuring the current focal area.
