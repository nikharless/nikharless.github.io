Expected Result

Visual indicators of a control's boundaries must have at least 3:1 contrast with the background.

Actual Result

Visual indicators of a control's boundaries do not have at least 3:1 contrast with the background.

Solution

Change the color of the boundary to have at least 3:1 contrast with the background.

Examples

Disabilities Affected

Color blind and visually impaired users need to be able to distinguish non-text user controls.

References

  • Understanding Success Criterion {{ sc.name }}
  • G174: Providing a control with a sufficient contrast ratio that allows users to switch to a presentation that uses sufficient contrast
