Expected Result

Single-pointer functionality doesn't use the down-event or it can be aborted or undone.

Actual Result

Irreversible action is executed on the down-even with no mechanism to undo it.

Solution

  • Execute the function on the up-event, OR
  • Provide a mechanism to cancel the action before completion, OR
  • Provide a mechanism to undo the action after completion, OR
  • Reverse the action of the down-event with the up-event.

The Javascript click() event activates on the up-event.
