Expected Result

The purpose of the link can be determined from the image alternative text alone.

Actual Result

The purpose of the link cannot be determined from the image alternative text alone.

Solution

  • Add alternative text to the image, describing the link purpose (alt, title, aria-label, or aria-labelledby), OR
  • If the image has accompanying visible text, combine the image and the text into a single link and use alt="" on the image.

Examples

Image Alternative Text

<a href="http://github.com">
      <img src="github-mark.png" alt="GitHub home">
    </a>

Link Alternative Text

<a href="#" aria-label="Add to Cart">
      <img src="shopping-cart.png" alt="">
    </a>

Image and Text in Single Link

Download the PDF User Guide

<a href="user-guide.pdf">
      <img src="pdf-icon.png" alt=""> Download the PDF User Guide
    </a>

Disabilities Affected

Descriptive link text is more understandable for all users, especially when using assistive technology.
