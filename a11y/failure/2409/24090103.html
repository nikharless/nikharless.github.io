Expected Result

The purpose of the link can be determined from the link text alone.

Actual Result

The purpose of the link cannot be determined from the link text alone.

Solution

Adobe Acrobat Pro

  1. Locate the link in the Tags panel.
  2. Right-click the <Link> tag or open the Options menu and select Properties....
  3. In the Object Properties dialog, select the Tag tab.
  4. Enter descriptive link text in the Alternate Description for Links field.

Examples

Why It's Important

Descriptive link text is more understandable for all users, especially when using assistive technology.
