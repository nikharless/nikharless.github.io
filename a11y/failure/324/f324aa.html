Expected Result

Form controls that have the same functionality within a set of web pages are identified consistently.

Actual Result

Form controls that have the same functionality within a set of web pages are not identified consistently.

Solution

Ensure form controls that have the same functionality within a set of web pages have consistent labels and accessible names each time they appear.
