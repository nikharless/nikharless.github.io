Expected Result

Icons that have the same purpose within a set of web pages are identified consistently.

Actual Result

Icons that have the same purpose within a set of web pages are not identified consistently.

Solution

Ensure icons that have the same purpose within a set of web pages have consistent accessible names each time they appear.

Why It's Important

Consistently identified actions are especially important to people with disabilities.
