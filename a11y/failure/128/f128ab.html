Expected Result

Prerecorded synchronized media and video-only media has a text transcript which includes the visual content.

Actual Result

The video has a text transcript, but it is not adequate.

Solution

Provide a text transcript that includes all key visual elements necessary to understand the video content.

Example

Text Transcript

This transcript does include the text on the cartoon images, which is positive. However, it should also include a brief description of the cartoons themselves to give context to the text.

TED: Liana Finck: Life is Hard. Art Helps.

Why It's Important

Deafblind users cannot see or hear the video content, so they require both the audio and visual information in text format so that it can be interpreted by assistive technology such as a braille device.
