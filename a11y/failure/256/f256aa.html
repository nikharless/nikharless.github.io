Expected Result

User can switch between different methods of input.

Actual Result

The content restricts input methods.

Solution

Ensure the content can be operated by various input methods. For example:

  • Use input-agnostic event handlers in JavaScript such as focus, blur, and click.
  • Register JavaScript event handlers for keyboard and pointer inputs simultaneously.
  • When one input method is detected, do not disable others.

Examples

  • A mobile device user can pair a keyboard and mouse and use those to interact with the content.
  • A desktop user can attach a touch-enabled monitor and use that to interact with the content.
  • A user can alternate between using speech recognition and keyboard/mouse input.

Why It's Important

People may not be able to work using just one input method.
