Expected Result

If synchronized media is used as a media alternative to text, it must be clearly labeled as such.

Actual Result

The video is an alternative to text on the page, but it is not clearly labeled as such.

Solution

Clearly label the video as an alternative to text on the page.

Failing Example

Passing Example

Not Applicable

  • Audio-only:
    File under {{ site.data.1201.1201[12010106].name }}.
  • Video-only:
    File under {{ site.data.1201.1201[12010204].name }}.

Disabilities Affected

Media which is intended as an alternative to text on the page needs to be clearly labeled as such, so users can find it and so users who expect text alternatives know not to look for them.
