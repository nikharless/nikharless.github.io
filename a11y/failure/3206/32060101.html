Expected Result

Help mechanisms which appear on multiple web pages appear in the same relative order on each page.

Actual Result

Help mechanisms which appear on multiple web pages do not appear in the same relative order on each page.

Solution

Place help mechanisms in the same relative order to other content on each page that they appear

Examples

Why It's Important

People who need help can find it more easily if it's in the same place.
