Expected Result

A method is provided to skip blocks of content repeated across pages.

Actual Result

<iframe>s have title attributes which describe the <iframe>'s contents, but they are not unique.

Solution

Set each title attribute to a unique value describing the <iframe>'s contents.

Examples

<iframe src="header.html" name="header-iframe" title="Header"></iframe>
<iframe src="footer.html" name="footer-iframe" title="Footer"></iframe>

Why It's Important

Screen reader users need to be able to tell which sections they can navigate to.
