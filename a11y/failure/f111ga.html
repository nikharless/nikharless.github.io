WCAG | 1.1.1 Non-text Content | 1.1.1.g CAPTCHA (Text Alternative)

Expected Result Each CAPTCHA must have a text alternative that describes its purpose (but not the visual elements).
Actual Result The CAPTCHA does not have a text alternative.
Solution
  • Consider using an alternative to the CAPTCHA.
  • Add a text alternative to the image which identifies it as a CAPTCHA.
Examples

ReCAPTCHA

The current verson of Google's ReCAPTCHA is a simple checkbox. If it is unable to validate, an image test is presented.

At this point the user can Tab directly to a button to access an audio version. The image buttons are not visible to screen readers.

ReCAPTCHA demo

Distorted Text

The text in the image cannot be put into the text alternative, since that would invalidate the purpose of the CAPTCHA. A text alternative is added to identify its purpose.
