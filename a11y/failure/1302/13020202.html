Expected Result

All meaningful content is readable with a screen reader.

Actual Result

Meaningful content is not readable with the screen reader.

Solution

Don't use aria-hidden="true" on meaningful content or its container elements.

Failing Example

aria-hidden="true" on Container

The paragraph below includes a redundant image. The entire paragraph has been hidden from assistive technology with aria-hidden="true". Assistive technology skips the entire thing, because aria-hidden="true" hides all non-interactive child elements of whatever element it is placed on.

Norway

<p aria-hidden="true"><img src="norway-flag.png"> Norway</p>

Passing Example

Hidden Redundant Image Only

This is the same paragraph as above, but the redundant image alone has been hidden from assistive technology with alt="".

Norway

<p><img src="norway-flag.png" alt=""> Norway</p>
