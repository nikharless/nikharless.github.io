Expected Result

All meaningful content is readable with a screen reader.

Actual Result

Meaningful content is not readable with the screen reader.

Solution

Don't use aria-hidden="true" on meaningful content or its container elements.

Failing Example

aria-hidden="true" on Link

This link includes a redundant image inside a link. The entire link has been hidden from assistive technology with aria-hidden="true".

PDF User Guide

<p aria-hidden="true"><a href="#"><img src="pdf-icon.png"> PDF User Guide</a></p>

Passing Example

Hidden Redundant Image Only

This is the same link as above, but the redundant image alone has been hidden from assistive technology with alt="".

PDF User Guide

<a href="#"><img src="pdf-icon.png" alt=""> PDF User Guide</a>
