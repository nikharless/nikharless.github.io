Expected Result

The reading order of unstyled page content is logical.

Actual Result

The reading order of page content is not logical when viewed without CSS.

Solution

Ensure that elements are ordered logically in the source code, so that when they are viewed without CSS they still appear in the correct order.

Failing Example

CSS Positioning

This example displays two lists which have been positioned using CSS. If CSS is turned off, the lists will not display properly.

Arizona Parks Nevada Parks Grand Canyon National Park Petrified Forest National Park Saguaro National Park Great Basin National Park Death Valley National Park 
<div class="box">
    <span class="menu1">Arizona Parks</span>
    <span class="menu2">Nevada Parks</span>
    <span class="item1">Grand Canyon National Park</span>
    <span class="item2">Petrified Forest National Park</span>
    <span class="item3">Saguaro National Park</span>
    <span class="item5">Great Basin National Park</span>
    <span class="item4">Death Valley National Park</span>
</div>
