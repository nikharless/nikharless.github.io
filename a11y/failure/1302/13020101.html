Expected Result

The reading order of unstyled page content is logical.

Actual Result

The reading order of page content is not logical when viewed without CSS.

Solution

Ensure that elements are ordered logically in the source code, so that when they are viewed without CSS they still appear in the correct order.

Failing Example

CSS Positioning

This is a bad example from W3 showing 2 lists which have been positioned using CSS.

<div class="box">
    <span class="menu1">Products</span>
    <span class="menu2">Locations</span>
    <span class="item1">Telephones</span>
    <span class="item2">Computers</span>
    <span class="item3">Portable MP3 Players</span>
    <span class="item5">Wisconsin</span>
    <span class="item4">Idaho</span>
</div>
  • Example of CSS positioning failure
  • Example of CSS positioning failure with styles removed

Passing Example

HTML Lists

This example could be fixed by using semantic HTML elements and then styling those with CSS.

Products

  • Telephones
  • Computers
  • Portable MP3 Players

Locations

  • Idaho
  • Wisconsin
<h2>Products</h2>
    <ul>
        <li>Telephones</li>
        <li>Computers</li>
        <li>Portable MP3 Players</li>
    </ul>

    <h2<Locations</h2>
    <ul>
        <Idaho>
        <Wisconsin>
    </ul>
