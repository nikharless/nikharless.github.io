Expected Result

When a modal is open, the screen reader is restricted to reading only the content inside it.

Actual Result

When a modal is open, the screen reader is able to read content outside of it.

Solution

  1. Add aria-modal="true" to the element containing the modal.
  2. When the modal is open, assign aria-hidden="true" to the elements outside the modal.
  3. When the modal is closed, remove aria-hidden="true" from the elements outside the modal.

Failing Example

Bad Modal
