Expected Result

Content which is meant to be hidden from all users is not readable by the screen reader.

Actual Result

Content which is meant to be hidden from all users can be read by the screen reader.

Solution

Use one of the following to hide content from all users:

  • CSS display: none
  • CSS visibility: hidden
  • HTML hidden attribute
  • CSS width: 0px; height: 0px

Failing Example

Off-screen Menu

This menu is visually hidden by positioning it off-screen. However, it is not hidden from assistive technology.

Note: This button example does not open when activated. It's only purpose is to demonstrate that the screen reader reads the menu when closed.

  • DuckDuckGo
  • Brave
  • Ecosia
  • Gibiru
Code Example 
.menu-button-links-fail [role="menu"] {
    left: -2000px;
    position: absolute;
}

Passing Example

Menu Hidden from All Users

This menu is hidden from all users with display: none.

  • DuckDuckGo
  • Brave
  • Ecosia
  • Gibiru
Code Example 
.menu-button-links [role="menu"] {
    display: none;
    position: absolute;
}
