Expected Result

Content which is meant to be hidden from all users is not readable by the screen reader.

Actual Result

Content which is meant to be hidden from all users can be read by the screen reader.

Solution

Use one of the following to hide content from all users:

  • CSS display: none
  • CSS visibility: hidden
  • HTML hidden attribute
  • CSS width: 0px; height: 0px

Failing Example

  • DuckDuckGo
  • Brave
  • Ecosia
  • Gibiru
