Expected Result

Prerecorded synchronized media must have a text or audio description of the visual content.

Actual Result

The prerecorded synchronized media content does not have either a text or audio alternative for the video portion.

Solution

  • Provide an audio description that includes all key visual elements necessary to understand the video content, OR
  • Provide a text transcript that includes description of all key visual elements required to understand the video content.

Examples

Audio Description

Text Transcript

Why It's Important

Visually impaired users, or any users who have difficulty understanding moving content, need an alternate means of receiving the information.
