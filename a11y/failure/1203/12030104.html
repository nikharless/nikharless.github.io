Expected Result

If the prerecorded synchronized media is used as a media alternative to text, it must be clearly labeled as such.

Actual Result

The synchronized media is an alternative to text on the page, but it is not clearly labeled as such.

Solution

Clearly label the synchronized media as an alternative to text on the page.

Example

Why It's Important

Media which is intended as an alternative to text on the page needs to be clearly labeled as such, so users can find it and so users who expect text alternatives know not to look for them.
