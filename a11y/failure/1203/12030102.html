Expected Result

Prerecorded synchronized media must have a text (transcript) or audio (audio descriptions) alternative that includes all of the information presented in the visual content.

Actual Result

The prerecorded synchronized media content has an audio description for the video portion, but it does not include all important visual elements.

Solution

Update the audio description to include all of the information presented in the video. Descriptions should be neutral, clear, and present tense.

Important visual information may include:

  • Actions
  • Facial expressions
  • Indication of dreams or memories
  • Passage of time
  • Scene changes
  • Text embedded in the video

Example

Audio Description

Disabilities Affected

  • Blind/vision impaired: People who are blind or vision impaired need to be able to access the same information provided in the video track.
  • Information processing: People who have the ability to see use alternatives to more easily process the visual information.
