Expected Result

Prerecorded synchronized media must have a text or audio description of the visual content.

Actual Result

The prerecorded synchronized media content has an audio description for the video portion, but it does not include all important visual elements.

Solution

Provide an audio description that includes all key visual elements necessary to understand the video content.

Example

Audio Description

Why It's Important

Visually impaired users, or any users who have difficulty understanding moving content, need an alternate means of receiving the information.
