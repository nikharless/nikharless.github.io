Expected Result

Prerecorded synchronized media must have a text (transcript) or audio (audio descriptions) alternative that includes all of the information presented in the visual content.

Actual Result

The prerecorded synchronized media content has a text transcript for the video portion, but it does not include all important visual elements.

Solution

Provide a text transcript that includes all key visual elements necessary to understand the video content.

Example

Text Transcript

This transcript does include the text on the cartoon images, which is positive. However, it should also include a brief description of the cartoons themselves to give context to the text.

TED: Liana Finck: Life is Hard. Art Helps.

Disabilities Affected

  • Blind/vision impaired: People who are blind or vision impaired need to be able to access the same information provided in the video track.
  • Information processing: People who have the ability to see use alternatives to more easily process the visual information.
  • Deafblind: Transcripts allow deafblind users access to information presented in video content.
  • SEO: Transcripts allow search engines to catalog the information contained in video content.
