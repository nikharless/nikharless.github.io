Expected Result

If the content has a time limit and the user is given a warning when time is about to expire, the user must be given a means to extend the time limit with a simple action at least ten times.

Actual Result

The user is given a warning when time is about to expire and is provided a means to extend the time limit with a simple action, but they cannot extend it at least ten times.

Solution

Allow the user to extend the time limit at least 10 times.

Example

Why It's Important

  • Blind/visually impaired: Users need more time to locate things on screen and to read. Screen reader users need more time to understand screen layouts, find information, and operate controls.
  • Motion impaired: Users need more time to react, type, and complete activities.
  • Cognitively impaired: Users need more time to read and understand the information.
