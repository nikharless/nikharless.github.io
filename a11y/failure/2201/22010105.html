Expected Result

Reading time limits can be turned off or paused and restarted.

Actual Result

The reading time limit cannot be paused or turned off.

Solution

  • Provide a means to pause and restart the content from where it was paused, OR
  • Provide a means to turn off the time limit before time expires, OR
  • Provide a means for users to display moving, scrolling, or auto-updating text in a static area.

Examples

  • A site contains a scrolling news banner at the top of the page. Users who need more time to read it can press the Escape key to pause the scrolling. Pressing Escape again restarts it.
  • An animation includes text that appears and disappears. A pause button is provided so that users who have trouble reading the text before it disappears can read it.

Disabilities Affected

  • Blindness/Visual Impairment: Users need more time to locate things on screen and to read. Screen reader users need more time to understand screen layouts, find information, and operate controls.
  • Motion impaired: Users need more time to react, type, and complete activities.
  • Cognitive Impairment: Users need more time to read and understand the information.

References

  • Understanding Success Criterion {{ sc.name }}
  • G4: Allowing the content to be paused and restarted from where it was paused
  • G198: Providing a way for the user to turn the time limit off
  • SCR33: Using script to scroll content, and providing a mechanism to pause it
  • SCR36: Providing a mechanism to allow users to display moving, scrolling, or auto-updating text in a static window or area
