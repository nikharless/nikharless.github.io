Expected Result

If the content has a time limit and the user is given a warning when time is about to expire, the user must be given a means to extend the time limit with a simple action.

Actual Result

The user is given a warning when time is about to expire but is not provided a means to extend the time limit with a simple action.

Solution

  • For a period of at least 20 seconds, allow the user to extend the time limit with a simple action (e.g., pressing the spacebar), AND
  • Provide a means for the user to extend the time limit at least 10 times.

Examples

  • A site logs the user out after a period of inactivity to protect their data. The site provides a dialog asking if the user needs more time. The user can extend the time limit an indefinite number of times. After 30 seconds, if there is no response, the user is logged out.

Why It's Important

  • Blind/visually impaired: Users need more time to locate things on screen and to read. Screen reader users need more time to understand screen layouts, find information, and operate controls.
  • Motion impaired: Users need more time to react, type, and complete activities.
  • Cognitively impaired: Users need more time to read and understand the information.
