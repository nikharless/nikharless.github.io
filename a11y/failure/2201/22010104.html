Expected Result

Session time limits can be turned off, adjusted, or extended.

Actual Result

The session time limit cannot be extended, adjusted, or turned off.

Solution

  • Provide a checkbox on the first page of a multipart form that allows users to ask for a longer session time limit or no session time limit, OR
  • Provide a means to turn off the time limit before time expires

Examples

  • A web page contains the first part of a five-part form. Immediately following the general instructions for completing the form is a checkbox with the label, "Allow an additional 15 minutes to complete each part of this form."

Disabilities Affected

  • Blindness/Visual Impairment: Users need more time to locate things on screen and to read. Screen reader users need more time to understand screen layouts, find information, and operate controls.
  • Motion impaired: Users need more time to react, type, and complete activities.
  • Cognitive Impairment: Users need more time to read and understand the information.
