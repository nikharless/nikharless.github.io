Expected Result

The width of a block of text can be adjusted so that no more than 80 characters are displayed without the need for horizontal scrolling.

Actual Result

When a block of text is resized so that no more than 80 characters are displayed, horizontal scrolling is required.

Solution

  • Use relative width measurements such as ems or percentages, OR
  • Do not set width on text containers

Examples

Use browser zoom to resize text to 200%.
