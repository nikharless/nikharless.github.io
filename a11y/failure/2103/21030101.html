Expected Result

All functionality is available through the keyboard interface.

Actual Result

A control cannot be accessed using only a keyboard

Solution

Remove the content which requires path-based input.

Examples

  • Freehand drawing
  • Painting

Why It's Important

People who can only use the keyboard interface need to be able to accomplish everything.
