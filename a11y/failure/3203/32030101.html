Expected Result

Navigation items which appear on multiple web pages appear in the same relative order on each page.

Actual Result

Navigation items which appear on multiple web pages do not appear in the same relative order on each page.

Solution

Consistently order navigation that repeats across multiple pages.

  • Do not change the order of menu items between pages.
  • Do not change the order of navigational elements relative to other content in the DOM.
  • Removing or inserting items (such as submenus) is allowed.

Examples

Why It's Important

Content that behaves predictably is especially important to people with disabilities.
