Expected Result

Instructions provided when components require non-standard keystrokes to navigate away from them must be accurate.

Actual Result

Non-standard keystrokes are required to navigate away from an element, but the instructions explaining them are not accurate.

Solution

Ensure that the instructions and the actual functionality match.

Failing Example

Non-standard Keyboard Shortcuts with Instructions

Instructions are provided for the non-standard functionality. However, this shortcut does not always work in form fields, because the keystroke combination is used to autofill data.

Click here to disable the trap.

All fields are required. Press Ctrl + Down Arrow to leave the form.

Keyboard focus is trapped.

Passing Example

Non-standard Keyboard Shortcuts with Instructions

Instructions are provided for the non-standard functionality, and these instructions work. If at all possible, you should stick to standard functionality instead.

All fields are required. Press Alt + Enter to leave the form.

Keyboard focus is trapped.
