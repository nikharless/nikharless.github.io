Expected Result

Instructions are provided when components require non-standard keystrokes to navigate away from them.

Actual Result

Non-standard keystrokes are required to navigate away from an element, but instructions explaining them are not provided.

Solution

Provide instructions on how to navigate away from the element.

Failing Example

Non-standard Keyboard Shortcuts

To escape this form, the user must press Ctrl + Enter. No instructions are provided, so there is no way the user would know to do this.

All fields are required.

Keyboard focus is trapped.

Passing Example

Non-standard Keyboard Shortcuts with Instructions

Instructions are provided for the non-standard functionality. If at all possible, you should stick to standard functionality instead.

All fields are required. Press Alt + Enter to leave the form.

Keyboard focus is trapped.

  • Motor Impairment: Users have difficulty using a mouse and rely on a keyboard.
