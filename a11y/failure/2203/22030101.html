Expected Result

The user is not subject to time limits on the content.

Actual Result

There is a time limit on interacting with the content.

Solution

Do not use time limits, except for video and live events.

Examples

  • A test is implemented without a time limit.
  • A game is turn-based rather than real-time.
  • An auction is implemented as a blind auction rather than a live competitive one.

Why It's Important

Users with various disabilities often need more time to complete actions.

References

  • Understanding Success Criterion {{ sc.name }}
  • G5: Allowing users to complete an activity without any time limit
