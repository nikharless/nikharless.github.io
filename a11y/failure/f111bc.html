WCAG | 1.1.1 Non-text Content | 1.1.1.b Meaningful Images (Text Alternative)

Expected Result All meaningful images must have a text alternative that conveys the same information as the image.
Actual Result The meaningful CSS background image does not have a text alternative.
Solution Add a text alternative which describes the purpose of the meaningful image using the following method:
  • Add an aria-label attribute to the container tag
Example

Background Image with Text

Lorem ipsum dolor sit amet, consectetur adipiscing elit, sed do eiusmod tempor incididunt ut labore et dolore magna aliqua. Rhoncus aenean vel elit scelerisque mauris.

<style>
    #bg {
        background-image: url("time-for-change.jpg");
        width: 649px;
        height: 427px;
    }
</style>
<div id="bg" aria-label="Time for Change"><div id="divText">Lorem ipsum dolor sit amet, consectetur adipiscing elit, sed do eiusmod tempor incididunt ut labore et dolore magna aliqua. Rhoncus aenean vel elit scelerisque mauris.</div></div>
Why It's Important Blind and visually impaired users need to have a text alternative to meaningful images so that they can be interpreted by assistive technology such as screen readers and braille devices. Without a text alternative, these users would not get the information conveyed by the image.
