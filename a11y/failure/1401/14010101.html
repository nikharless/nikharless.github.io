Expected Result

Color is not used as the only visual means of conveying information.

Actual Result

Color is used as the only visual means of conveying information.

Solution

Provide a visual means of conveying the information in addition to color.

Failing Examples

Required Fields Are Red

In this example, red text is the only visual indication that a field is required. While the fields have the required attribute, programmatic indicators don't satisfy this checkpoint.



* Required fields are in red.

<label for="lastName" style="color: #ee0000;">Last Name
    <input required type="text" id="lastName">
</label>

Error is in Red

Fields with errors are highlighted in red, but no other visual indicator exists. The aria-invalid attribute does not count as an additional means because it's not visual.

Note: This example would also fail {{ site.data.3301.3301[3301].name }} due to its lack of an error message.


<label for="zip">Postal code:</label>
<input type="text" name="zip" id="zip" aria-invalid="true" style="background-color: #fffafa; border: 2px solid #ee0000; border-radius: 4px;">

Passing Examples

Required Fields Are Red

Adding text to the labels provides a means other than color for indicating required fields.



<label for="lastName" style="color: #ee0000;">Last Name (required)
    <input required type="text" id="lastName">
</label>

Error is in Red

In addition to the symbol and the red highlighting, the error is also in text.



Error: Please enter a valid postal code. 
<label for="zip">Postal code:</label>
<input type="text" name="zip" id="zip" aria-invalid="true" aria-describedby="errorMsg" style="background-color: #fffafa; border: 2px solid #ee0000; border-radius: 4px;">
<span id="errorMsg" style="color: #ee0000;">Error: Please enter a valid postal code.</span>

Disabilities Affected

  • Blindness/Visual Impairment: Users cannot see colors or have limited color vision.
  • Color blindness: Users cannot distinguish between colors.
  • Cognitive Impairment: Users may benefit from additional text cues.
