{{ currentpg.name }} (Level {{ currentpg.level | upcase }})

Success Criterion

If an input error is automatically detected and suggestions for correction are known, then the suggestions are provided to the user, unless it would jeopardize the security or purpose of the content.

Definitions

input error
information provided by the user that is not accepted

Note: This includes information that is required by the web page but omitted by the user and information that is provided by the user but that falls outside the required data format or values.
