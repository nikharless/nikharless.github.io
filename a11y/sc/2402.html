{{ currentpg.name }} (Level {{ currentpg.level | upcase }})

Success Criterion

Web pages have titles that describe topic or purpose.

Definitions

web page
a non-embedded resource obtained from a single URI using HTTP plus any other resources that are used in the rendering or intended to be rendered together with it by a user agent

Note 1: Although any "other resources" would be rendered together with the primary resource, they would not necessarily be rendered simultaneously with each other.

For the purpose of conformance with these guidelines, a resource must be "non-embedded" within the scope of conformance to be considered a web page.

Examples:

  • A web resource including all embedded images and media
  • A web mail program that lives entirely at a single url but includes an inbox, a contacts area, and a calendar. Links or buttons are provided that cause the inbox, contacts, or calendar to display but do not change the URI of the page as a whole.
  • A customizable portal site, where users can choose content to display from a set of different content modules.
  • A movie-like interactive shopping environment where you visually move around in a store dragging products off of the shelves areound you and into a visual shopping cart in front of you. This might be a single-page website or just one page within a website.
