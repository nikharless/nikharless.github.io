WCAG | 1.2.3 Audio Description or Media Alternative (Prerecorded)

Basic Requirements and Best Practices

= Requirement
= Best Practice

  • Prerecorded synchronized media must have a text or audio description of the visual content.
  • The text or audio description should be presented next to the audio and be easy to find.
Success: You can get all the information conveyed in the visual portion without being able to see it.

Testing Logic

Does the page contain synchronized media (e.g., video with dialogue)?

  • Yes; Does the video content in the synchronized media have an audio description?
    • Yes; Does the audio description include all the important actions, visuals, and text in the video?
      • Yes; PASS
      • No; Include all important actions, visuals, and text from the video in the audio description.
    • No; Is the video itself a media alternative for text?
      • Yes; Is it clearly labeled as such?
        • Yes; PASS
        • No; Add text making it clear that the video is an alternative for the text content.
      • No; Is the video accompanied by a text alternative describing all the important actions, visuals, and text in the video?
        • Yes; PASS
        • No; Add an alternative for the video content. This could be a link to audio descriptions, an alternative audio track with audio descriptions, an alternative version of the video with audio descriptions, a static text alternative, etc.
  • No; PASS

Web Test Procedure

  1. Locate each video element on the page.
  2. Determine whether the video content contains any information not provided through the audio content.
  3. If it does, verify that it has a text or audio description that provides the visual information in another form.

Mobile App Test Procedure

  • Follow the procedure for Web.

Failures

  • F1.2.3.a.a Synchronized media does not have either a text or audio alternative for the video portion. (Medium)
  • F1.2.3.a.b Existing audio description is incorrect. (Medium)
  • F1.2.3.a.c Existing text description is incorrect. (Medium)
