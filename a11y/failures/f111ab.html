WCAG | 1.1.1 Non-text Content | 1.1.1.a Alternative Text (Active Images)

Expected Result If a control is labeled with only an image, the image must have alternative text which describes the function or destination of the control.
Actual Result The control is labeled with only an image, but the image does not have a text alternative.
Solution
  1. Add a <title> element as the first child of the <svg> element, and place the text alternative within it.
  2. Ensure the <svg> has the role="img" attribute to allow screen readers to identify the text alternative.
