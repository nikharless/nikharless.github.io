WCAG | 1.1.1 Non-text Content | 1.1.1.a Alternative Text (Active Images)

Expected Result The text alternative for the functional image describes the purpose of the control.
Actual Result The functional image has a text alternative, but it does not describe the purpose of the control.
Solution

Change the text alternative to describe the purpose of the image.

Example

Search Button

<input type="image" name="submit" src="submit.png"
    alt="Search">
Why It's Important

Blind and visually impaired users need to have a text alternative to functional images so that they can be interpreted by assistive technology such as screen readers and braille devices.

Server-side image maps send the coordinates of mouse clicks to be calculated server-side, after which a link is returned. There is nothing for keyboard users to focus on, and no way to add a text alternative to the areas of the image map.
