WCAG | 1.1.1 Non-text Content | 1.1.1.a Alternative Text (Active Images)

Expected Result If a control is labeled with only an image, the image must have alternative text which describes the function or destination of the control.
Actual Result The control is labeled with only an image, but the image does not have a text alternative.
Solution
  1. Add a text alternative which describes the purpose of the interactive image using one of the following methods:
    • Add an alt attribute to the <img> tag, OR
    • Add an aria-label attribute to the <img> tag, OR
    • Add an aria-labelledby attribute to the <img> tag, set to the id value of the text to use as a label, OR
    • Add a title attribute to the <img> tag
