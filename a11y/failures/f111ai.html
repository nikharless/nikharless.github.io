WCAG | 1.1.1 Non-text Content | 1.1.1.a Alternative Text (Active Images)

Expected Result When the functional image changes dynamically, the text alternative updates to match.
Actual Result When the functional image changes dynamically, the text alternative does not update to match.
Solution

Code the page to update the text alternative whenever the image changes.
Example

Play/Pause Button

This button uses JavaScript and CSS to change the aria-labelledby value along with the displayed image.

<div class="player" role="button" tabindex="0" aria-labelledby="play">
    <img id="play" src="play.png" class="play" aria-hidden="true" alt="Play">
    <img id="pause" src="pause.png" class="pause" aria-hidden="true" alt="Pause">
</div>
Why It's Important

Blind and visually impaired users need to hear the same words that the visual user sees.
