Table (Responsive, Collapsible)

Shrink the size of your browser window to a very narrow width to experience the responsive features. The table is converted to a list at small sizes to retain readability with reflow.

Keyboard: Column headers are buttons, which are focused with the Tab key and activated with the spacebar or Enter.

Screen Reader: The sort state is appended to the caption. The sort state is also read with the button whose sort state is currently active. Buttons which are not currently toggled on are only read with their names. Activating a button triggers an aria-live announcement.
